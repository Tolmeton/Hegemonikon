## 2. 普遍性のジレンマ (The Universality Dilemma)

### 定式化

```
抽象度 α ∈ [0, 1]

α → 1: 説明力 E(α) → max, 予測力 P(α) → 0
α → 0: 説明力 E(α) → 0,   予測力 P(α) → max

E(α) と P(α) は反比例関係にある
```

### 情報理論的厳密化: Information Bottleneck (v2.1.0)

> **v1.0.0-2.0.0 の形式化 E≈log|Ω| は撤回する。**
> Newton 統一 (天体力学+落下力学→万有引力) は |Ω|↓ (統合) を伴うが、説明力は増大する。
> E≈log|Ω| ではこの統一を記述できない。

正しい形式化は **Tishby, Pereira, Bialek (1999) の Information Bottleneck (IB) 原理**:

IB は入力 X を圧縮した表現 T_α を通じて、ターゲット Y を予測する最適化問題:

```
min_{p(t|x)} [ I(X; T_α) − β · I(T_α; Y) ]

where:
  I(X; T_α)  = 圧縮情報量 — 表現が入力について保持する情報量
  I(T_α; Y)  = 予測情報量 — 表現がターゲットについて保持する情報量
  β = 1/α    = トレードオフパラメータ
```

#### E×P のIB対応

| HGK | IB | 操作 |
|:----|:----|:-----|
| E(α) 説明力 | I(X; T_α) の**情報量的代理指標** | 単一表現でどれだけ多くの構造を束ねられるか |
| P(α) 予測力 | I(T_α; Y) 予測情報量 | α↑ → 具体的予測能力が低下 |
| E↑ ⇒ P上限↓ | IB パレートフロンティア | 圧縮が進むほど予測上限が下がる最適トレードオフ曲面 |
| compile path (§9.4) | β を段階的に増大 | 抽象→具体の各段階は β シフト |

> [!IMPORTANT]
> ここでの `E(α)` は `I(X;T_α)` そのものと**同義ではない**。
> `I(X;T_α)` は、HGK が言う「単一表現でどれだけ多くの構造を束ねるか」という説明力を
> 情報理論の語彙へ写した**代理指標**である。
> この区別を落とすと、「圧縮が増えるのに説明力が増えるのはなぜか」という符号混線が生じる。

#### Levins GRP の IB 的基礎づけ (v2.2.0 — T-028)

上記の `E↑ ⇒ P上限↓` は Levins (1966) の GRP トレードオフの再記述である。IB が両者を統一する:

| Levins | IB | HGK |
|:-------|:---|:----|
| P (Precision) | I(T; Y) — T から Y の予測精度 | P(α) 予測力 |
| G (Generality) | Val(X) の広さ — T が適用される入力空間の範囲 | E(α) の適用範囲 |
| R (Realism) | p(t\|x) の構造的忠実さ — encoder がメカニズムを再現する程度 | compile path の射保存数 |

データ処理不等式 (DPI: I(T;Y) ≤ I(X;T)) から:

- **高 G (広い適用) → 低 I(X;T) → 低 I(T;Y) → 低 P**: 一般的なモデルは精密予測できない
- **高 R (T≈X) → 高 I(X;T) → 低 G**: 忠実なモデルは狭い適用範囲
- **高 G + 高 R → β→∞ が必要 → T はデータベース**: 理論ではなくなる

Orzack & Sober (1993) は GRP を「論理的必然ではなく経験的一般化」と批判した。IB は中間的回答を与える: DPI からの連続版 GRP トレードオフは論理的帰結だが、Levins の離散的3頂点版は連続的フロンティアの粗視化 (coarse-graining) である。

> [!NOTE]
> **確信度**: GRP の IB 的基礎づけ [推定: 75%]。R の IB 的定義の精密化が主要な残課題。
> **SOURCE**: Bareinboim et al. (2022) 原典精読 [62ページ, SOURCE] + T-028 成果物

#### Newton 統一の矛盾解消

旧形式化では:
- K1 (天体力学) + K2 (落下力学) → Newton 統一: |Ω|↓ → E↓ → **矛盾**

IB 形式化では:
- **K1, K2 は別々の encoder** p₁(t|x), p₂(t|x)
- **Newton は単一の encoder** p_N(t|x) で両方のデータを説明
- 同一の圧縮表現 T が K1, K2 両方の Y を予測 → **圧縮効率↑** = I(X; T_N) の情報密度↑
- よって E(Newton) > E(K1∪K2)。**統一は説明力を増大させる** ✅

#### 忘却論との接続

**Tishby の「学習の本質は忘却 (forgetting irrelevant information)」は、Paper I の「力の本質は忘却 (忘却関手 G の適用)」と同型命題:**

| 忘却論 (圏論) | IB (情報理論) | 関係 |
|:-------------|:-------------|:-----|
| G (忘却関手) | 圧縮 encoder p(t\|x) | IB 圧縮 ⊂ 忘却。圧縮は忘却の**情報量的測定** |
| F (回復関手) | 復号 decoder p(y\|t) | 対応 |
| ker(G) | I(X; T) から落ちた情報 | 対応 |
| image(G) | T に残った情報 | 対応 |
| CI 定理 (G∘F 不動点の安定性) | IB パレートフロンティア上の安定性 | 対応 |

> [!IMPORTANT]
> 忘却 ≥ 圧縮。等号ではない。
> 忘却 = 選択 （何を ker(G) に送るか = 何を選ぶか）。これのみが等価。
> 圧縮・削除・洗練・劣化は忘却の**部分集合**。
> IB の I(X;T) は忘却の「測定ツール」であって、忘却そのものではない。

#### 随伴構造による圏論的根拠 (v4.1.0 — essay §9)

上記の ker(G) / image(G) は、随伴対の単位 η の核・像として圏論的に基礎づけられる:

```
知覚  L_i: Amb → Cat_i   (左随伴 = 外部情報の内在化)
行動  R_i: Cat_i → Amb    (右随伴 = 内部モデルの外部化)
L_i ⊣ R_i

単位  η: Id_Amb → R_i ∘ L_i
  Ker(η) = 内在化されなかった構造 = ker(G)
  Im(η)  = 内在化された構造       = image(G)
```

**理解と予測の随伴的相補性** (essay「反証可能性は死んだ」§9.4):

| 概念 | 圏論 (随伴) | IB (情報理論) | 定義 |
|:--|:--|:--|:--|
| 理解 | Im(L\_i) が大きい | I(X; T\_α) が大きい | 対象の構造を自身の圏に内在化すること |
| 補完₁ | Ker(η) の自己補完 | I(X;Y) − I(T\_α;Y) のうち内在化失敗の穴埋め | 内在化できなかった情報についての内部推論 |

**帰結**: 補完₁の必要性は |Ker(η)| の単調増加関数。理解が深まれば補完₁は減る。

**原理的制約** (Paper VII §6.2): η は構造に対して同型に近づきうるが、値に対しては原理的に同型にならない。∴ Ker(η) > 0 は常に成立し、補完₁由来の内部穴埋めは原理的には消滅しない — しかし漸近的に減少する。

**`E↑ ⇒ P上限↓` の導出**: パレートフロンティア上の最適理論について、DPI (I(T;Y) ≤ I(X;T)) から E の増加は P の上限を構造的に制約する。これが ker(G) / Ker(η) の随伴的構造の情報理論的表現である。

> [!NOTE]
> この随伴的定式化は「説明ばかりで予測がない」批判への構造的応答を可能にする:
> **E が大きく P の上限が低い = Ker(η) が小さい側にある = 構造保持の広い理論候補**。
> ここで荷重しているのは予測₁一般ではなく、主として補完₁の少なさである。
> 補完₁の少なさは理論の欠陥ではなく、理解の深さの指標。
> 詳細: essay「反証可能性は死んだ」§9.1-9.8。

### 真理と予測の型分け (v2.4.0 — 2026-04-11)

上の議論では「真理」と「予測」が複数の意味で使われており、未定義のままでは
「予測を生む理論は真理ではない」が `偽` と `同一でない` のあいだで滑る。
ここでは型を分ける:

| 記号 | 名称 | 定義 | 直観 |
|:--|:--|:--|:--|
| **真理₀** | 構造真理 / 絶対真理 | 値・座標・観測手続き・実装に先立つ、不変な普遍構造 | π そのもの。有限桁展開ではない |
| **真理₁** | 経験真理 | 真理₀ を有限理論として世界に接続したとき、観測・介入・誤差訂正に耐える限りでの真 | π の近似式。完全同一ではないが、世界拘束の下で有効 |
| **予測₀** | 構造予測 | 何が可能/不可能か、何が何に従うか、どの順序・制約が保たれるかの予告 | 値ではなく「形」の予告 |
| **予測₁** | 値予測 | 初期条件・境界条件・観測写像を入れて具体値・時系列・選択を出すこと | 数値、曲線、行動列の予告 |

**Paper VII §6.2 との接続**:

- η は**構造**に対しては同型に近づきうる → 真理₀ / 予測₀ に対応
- η は**値**に対しては原理的に同型にならない → 真理₁ / 予測₁ に対応

したがって:

1. **予測₁を生む理論は真理₀そのものではない**。それは既に座標化・値化・境界条件化されている
2. **しかし予測₁を生む理論は真理₁ではありうる**。ゆえに「予測を生む理論 = 偽」ではない
3. 科学は真理₀へ近づくために真理₁を鍛える営みであり、工学は真理₁を制御可能性へ翻訳する営みである

> [!IMPORTANT]
> HGK で「予測を生む理論は真理ではない」と言うとき、通常は
> **「真理₀と同一ではない」** ことを意味する。
> これは「偽である」という意味ではない。
> 真理₀ / 真理₁ を分けない議論は、真偽論争ではなく単なる語義滑りを生む。

### ポパー適用不能の最小主張 (v2.7.0 — 2026-04-11)

HGK がここで言いたいのは、「論理が死んだ」ではない。
言いたいのは、**論理的反証を科学の主権原理として戴冠することが、科学の実際の対象型に合っていない**ということだ。

そのために、反証の単位を孤立理論から理論束へ上げる:

```
B = (T, A, M, I)
  T = 理論核
  A = 補助仮説
  M = 測定系・前処理・観測写像
  I = 解釈規則・観測文言語

Obs_B(o) = I(M(o))
  = 理論束 B を通って生成された観測文
```

このとき:

- **論理的反証** = 孤立命題に対する `modus tollens`
- **科学的反証** = 理論束 `B` に対する公開的更新要求

> [!IMPORTANT]
> `modus tollens` 自体は有効である。
> しかし科学が実際に試験しているのは、孤立命題ではなく **理論束 B** であり、
> そこで使われる観測文も `Obs_B(o)` という理論負荷的生成物である。
> **ゆえに「ポパーの論理的反証」は科学にそのまま適用できない。**

### 適用不能の帰結としての更新規則3層分解 (v2.7.0 — 2026-04-11)

反証可能性批判で滑りやすいのは、「何が壊れたのか」と「何を捨てるべきか」と
「その理論にまだ価値があるか」を1つの操作に潰してしまうことである。
HGK では、科学的反証を次の 3 層へ展開する:

| 層 | 名称 | 問い | 典型操作 |
|:--|:--|:--|:--|
| **U₀** | 公開矛盾の発生 | ある観測文 `Obs_B(o)` は、その理論束と両立するか | 禁止された事例の出現、整合性検査 |
| **U₁** | 責任配分層 | 壊れたとき、理論束のどこを改訂すべきか | fault localization, 補助仮説の切り分け |
| **U₂** | 価値層 | 改訂後もその理論束を保持する価値があるか | 統合力、説明範囲、改訂コスト、将来有用性の評価 |

この区別に従うと:

1. **科学的反証**はまず U₀ の出来事である
2. **どこを直すか**は U₁ の問題であり、Duhem-Quine / Lakatos が効く
3. **理論全体を捨てるか**は U₂ の問題であり、単一反例からは自動には出ない

> [!IMPORTANT]
> したがって、HGK の war claim は
> **「論理的反証は無効ではないが、科学には適用不能である」**
> である。
> `U₀/U₁/U₂` はその帰結として必要になる更新分解であって、
> 主定理そのものではない。

### 予測の二層分解 (v2.5.0 — 2026-04-11)

上の真理₀ / 真理₁ と更新規則 3 層を接続するため、予測の語も固定しておく:

| 名称 | 記号 | 定義 | 主な位置 |
|:--|:--|:--|:--|
| **経験的予測** | 予測₁ | 初期条件・境界条件・観測写像を入れて具体値を出すこと | 真理₁, U₀ の公開テスト |
| **内部自己補完** | 補完₁ | 随伴できなかった情報を内部モデルで穴埋めすること | U₁ の近辺。理解不足の補助操作 |
| **構造予測** | 予測₀ | どの制約・保存則・順序が成り立つかを先に言うこと | 真理₀ の展開 |

> [!NOTE]
> 長編・抽出版で `Ker(η)` と結びつくのは主に **補完₁** である。
> Mangalam 型批判の「何も予測しない」は通常 **予測₁** を指す。
> この 2 つを分けないと、「予測概念をすり替えた」という最も安い反論を招く。

### 「1」の比喩

数字の **1** を考える:

- **説明力**: 極大。あらゆるものを「1つの〇〇」として記述可能
- **予測力**: ゼロ。「1」から具体的な対象を一意に導出できない
- **有用性**: 極大。数学の全体系がこの概念なしには成立しない

**1 はなにも予測しない。だが 1 はあらゆるものを説明する。これは 1 の欠陥ではなく、普遍性の必然的構造である。**

IB を連想させる直観像として: **1 は「極限的に圧縮された記述」を表す比喩である。ここで言いたいのは情報保持量の厳密計算ではなく、具体値を一意導出しない普遍記述の位置づけである。**

## 3. メタ原理の定義

### 科学的仮説との区別

| 基準 | 科学的仮説 | メタ原理 |
|:-----|:----------|:---------|
| **評価軸** | 予測力・反証可能性 | 統合力・整合性・有用性 |
| **失敗条件** | 予測が外れる | 統合できない理論が出現する |
| **成功条件** | 予測が当たる | 異なる分野を統一的に記述できる |
| **例** | 特定の神経回路モデル | 圏論、FEP、情報理論 |

### 先行事例

**圏論** は数学の「メタ言語」:

- 圏論自体は何も予測しない
- だが圏論なしには諸分野の統合的理解は不可能
- 「圏論は反証不可能だから無価値」とは誰も言わない

### 応答: カテゴリーの混同

Mangalam の批判は FEP を**科学的仮説**として評価している。しかし:

1. **FEP は科学的仮説ではなくメタ原理**。メタ原理に反証可能性を要求するのは、数学に実験的検証を要求するのと同型のカテゴリーエラー

### 注意: この応答のリスク

> [!WARNING]
> 「FEP はメタ原理だから反証不可能でよい」という主張は、弦理論の「多元宇宙だから反証不可能でよい」と**同型のリスク**を持つ (Hossenfelder 2018)。

|:-----|:----------|:-----------------|
| **GRP** | パレートフロンティア上の移動**方向** | 忘却の**内容** (何を ker(G) に送るか) |
| **因果階梯** | β 軌道上の**位置** | 忘却の**深度** (ker(G) をどのくらい大きくするか) |

```
                  忘却の深度 (因果階梯 = β 軸)
                  ├── L3 (β大): 全構造保持。ker(G) 最小
                  ├── L2 (β中): 反事実を忘却
                  └── L1 (β小): 因果メカニズムを忘却。ker(G) 最大
                  ↕
忘却の方向 (GRP = パレート方向)
  ├── G を忘却: 適用範囲を限定し精密化
  ├── R を忘却: メカニズムを捨て汎用化
  └── P を忘却: 予測を捨て説明力を最大化
```

**直感**: GRP は「**何を**忘れるか」、因果階梯は「**どのくらい**忘れるか」。同じ忘却関手 G の異なる座標 (direction vs magnitude) を捉えている。

IB の相転移現象 (Tishby & Zaslavsky 2015) は、β を連続的に変化させたときに T の構造が不連続に変化する臨界点の存在を示す。因果階梯の離散的構造 (L1/L2/L3) は、この IB 相転移の自然な帰結であるという仮説を提案する。

> [!IMPORTANT]
> **論証の性質**: 上記は「構造的埋め込み (embedding)」の存在を示す論証であり、「形式的同型 (isomorphism)」の証明ではない。
> 主要な残課題: (1) R の情報量的定義の精密化 [推定: 65%]、(2) IB 相転移と因果階梯の離散性の対応 [仮説: 50%]

**確信度**: GRP と因果階梯の IB 統合 [推定: 72%]
