# 80_運用｜Ops — 能動推論 · 恒常性 (H)

> **FEP 演繹**: Ops は能動推論の「恒常性維持」の場。
> システムを安定した状態に保つ EFE の pragmatic value を担う。

---

## 分類基準: EFE 変数 H (Homeostasis / 恒常性)

**唯一の問い: 「このツールはどの恒常性を保つか？」**

| 帯 | H (恒常性) | 内容 |
|:---|:-----------|:-----|
| `00_スクリプト｜Scripts` | **自動化** | 定期実行・自動化スクリプト |
| `01_開発ツール｜Devtools` | **開発環境** | 開発者向けツール・IDE 設定 |
| `04_配備｜Deploy` | **配備管理** | systemd / K8s / Docker |
| `05_設定｜Config` | **設定管理** | 環境設定ファイル |
| `06_文書｜Docs` | **運用文書** | 運用手順書・リファレンス |
| `_src｜ソースコード` | **(実装)** | 各ツールのソースコード |

> ※ `02_` と `03_ログ` が欠番: 過去に整理された帯の痕跡として許容。
