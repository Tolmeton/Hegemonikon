> **FEP 演繹**: Peira は能動推論の「探索」の場。

---

## 分類基準: EFE 変数 IG (情報利得 / Epistemic Value)

**唯一の問い: 「この実験はどの不確実性を減らすか？」**

| 帯 | IG (情報利得) | 内容 |
|:---|:-------------|:-----|
| `00_汎用｜General` | **汎用実験** | ベンチマーク、PoC、単発スクリプト群 |
| `01_検証｜Verification` | **整合性確認** | 既存体系の整合性チェック |
| `02_テスト｜Tests` | **回帰テスト** | 実装の品質保証 (MCP, cortex, embedder 等) |
| `03_リモート｜Remote` | **リモート実験** | GPU 環境・LS 検証 (00 の特殊ケース) |
| `04_知覚PoC｜PerceptionPoc` | **知覚実装検証** | OmniParser / Windows Agent 等の PoC |
| `05_スペクトル解析｜SloppySpectrum` | **理論検証** | Fisher 情報量 / VFE / 自己参照の解析的導出 |
| `06_Hyphē実験｜HyphePoC` | **チャンク理論検証** | Hyphē 理論に基づくセッションログ自動チャンカー PoC |
| `50_自律研究｜Autoresearch` | **自律 R&D** | 独立 git。Creator 不在の自律研究プログラム |

## 境界ケースの判定

| ケース | 判定 |
|:-------|:-----|
| 実験が成功 → どこへ？ | 確定知識 → **30_記憶/04_知識** へ移行 |
| 実験が失敗 → どこへ？ | 90_保管庫 へアーカイブ |
| テスト vs 検証 | 自動化テスト → **02_テスト** / 手動検証 → **01_検証** |
| 理論的実験 | Fisher/VFE 数学導出 → **05_スペクトル解析** |
| 長期自律実験 | 独立リポジトリ → **50_自律研究** |
