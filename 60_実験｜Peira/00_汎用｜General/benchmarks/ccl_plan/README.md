# CCL-plan 統合ベンチマークプロジェクト

> **状態**: 実装中
> **目的**: HGK全認知機構を /ccl-plan 実行パイプラインに意図的に統合し、統合テスト (ベンチマーキング) を行うこと。

## 戦略的文脈 (2026-02-24)

通常、全機構を1つのマクロに詰め込むことは「過剰介入 (anti-pattern)」とされる。
しかし、`/ccl-plan` (v6.0) は別格であり、**「HGK システムの究極的な統合テスト（ベンチマーク）」** として機能する。

**Creator の意図**:
「意図的に全機構を詰め込む。機構が正しく機能し、意味があることを証明した上で、後から引き算してまともに使える実用版にする」

本プロジェクトは、この戦略に基づいて現在欠落している全機構を `macro_executor.py` および `/ccl-plan` の実行パスに統合し、完全な動作を証明することを目指す。
