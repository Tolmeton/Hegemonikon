## 2. Kalon 定義の要約（監査の出発点）

```
Kalon(x) ⟺ x = Fix(G ∘ F)    where F ⊣ G

F: 左随伴（自由構成、展開、Explore、Colimit 的）
G: 右随伴（忘却/蒸留、核だけ残す、Exploit、Limit 的）
Fix(G∘F): G∘F の繰り返し適用が収束する不動点
```

**三属性**: Fix(G∘F) ∧ Presheaf ∧ Self-referential

**7定理**: T1(Limit) T2(Colimit) T3(Beauty) T4(VFE) T5(Fractal) T6(F⊣G法則) T7(Self-ref)

**§11 拡張**: 出力の Kalon = Trace ∧ Negativa ∧ Iso
