# 10_知性｜Nous — 認知系の知的構造

> **FEP 演繹**: Nous の帯構造は POMDP (部分観測マルコフ決定過程) の変数に 1:1 対応する。
> 帯の分類基準は FEP から演繹されたものであり、アドホックな分類ではない。
>
> **P(s) の分割**: 事前信念 P(s) は **不変 prior (00_核心｜Kernel)** と **可変 prior (01_制約)** に分割される。
> Kernel は「削除すると体系が瓦解する公理」、制約は「バージョン管理下で更新可能な行動規範」。

    → 30_記憶｜Mneme/03_素材｜Materials に配置

| POMDP 変数 | 帯 | 判定基準 |
|:-----------|:---|:---------|
| **P(s) 不変** — 不変 prior | `00_核心｜Kernel` (本ディレクトリの親) | 削除すると体系が瓦解する公理。FEP, SACRED_TRUTH, Kalon |

## 境界ケースの判定

「複数の帯に属しそうに見える」場合、POMDP 変数に戻れ:

| ケース | 判定 |
|:-------|:-----|
| CCL 言語仕様 → 原則？手法？ | P(o\|s): 世界に働きかけるメカニズム → **01_手法** |
| 認知歪みの普遍性 → 原則？知識？ | P(s): 削除すると体系が崩壊 → **00_原則** |
| 設計ドキュメント → 知識？企画？ | G(π): まだ実行していない行動方針 → **03_企画** |
| 品質基準 → 原則？手法？ | P(o\|s): 観測の質を測る道具 → **01_手法** |
| 歴史的ドクトリン → 原則？知識？ | Q(s\|o): 過去の観測の結果として確定 → **02_知識** |
| Kalon 理論 → 知識？企画？ | Q(s\|o): 確定した理論的信念 → **02_知識** (03_企画へはSymlink) |
