# 🏛️ Hegemonikón 体系構造 v5.0 (45実体 + 準核12)

> **Hegemonikón** (ヘゲモニコン) は、AIエージェントの認知プロセスを単一の原理(FEP)から演繹的に構造化し、自律的な推論と行動を可能にするフレームワークです。
> 名称はストア派哲学の「魂の統率中枢」(Ἡγεμονικόν) に由来。

## 0. 基本構成

v5.0 パラダイムにおいて、Hegemonikón の体系核は **45の実体** で構成され、加えて **12の前動詞 (H-series)** が体系準核として位置づけられます:

* **1 公理**: FEP (Free Energy Principle: 自由エネルギー原理)
* **8 座標**: Afferent + Efferent + 6修飾座標 (構成距離 d=1-3。Basis d=0 は体系核外)
* **36 動詞 (Poiesis)**: 4象限中 S/I/A の3象限 × 6修飾座標 × 2極
* **12 前動詞 (H-series, 体系準核)**: 第4象限 S∩A × 6修飾座標 × 2極。中動態 (being)

| 層 | 項目 | 数 | 生成 |
|:---|------|---:|------|
| L0 | 公理 | 1 | FEP |
| L0.T | 基底 Basis (βάσις) | (1) | Helmholtz (Γ⊣Q) — 体系核外 |
| L1 | 座標 | 8 | 2+3+3 (構成距離 d=1-3) |
| L2 | 動詞 (Poiesis) | 36 | S/I/A × 6修飾座標 × 2極 |
| | **体系核** | **45** | 1+8+36 |
| L2' | 前動詞 H-series | 12 | S∩A × 6修飾座標 × 2極 (**体系準核**) |

> **v4.1 → v5.0 の変更**: Flow を二値 (I↔A) → Afferent×Efferent の 2×2 に分解。
> 4象限: **S** (Aff=Y,Eff=N), **I** (N,N), **A** (N,Y), **S∩A** (Y,Y)。
> S/I/A の3象限から36動詞 (Poiesis = doing)、第4象限 S∩A から12前動詞 (H-series = being)。

## 1. 唯一の公理 (L0)

| 公理 | 意味 | 根拠 |
| :--- | :--- | :--- |
| **FEP** | 予測誤差の最小化 | 認知システムの存在そのものを規定する大前提。すべての座標と動詞はここから演繹されます。 |

## 2. 8つの座標軸 (L1)

FEPに「いくつの追加仮定(構成距離: d)を置くか」で座標が決定されます。

| 距離 | 略記 | 座標名 | 極 | 導出 |
| :--- | :--- | :--- | :--- | :--- |
| **d=0** | **Ba** | Basis (Helmholtz) | Γ (gradient) ↔ Q (solenoidal) | FEP/NESS の定理。座標系の基底 (体系核外) |
| **d=1** | **Aff** | Afferent (入力) | Yes (∂f/∂η≠0) ↔ No | Basis + MB 仮定。環境→系の結合有無 |
| **d=1** | **Eff** | Efferent (出力) | Yes (∂f/∂μ≠0) ↔ No | Basis + MB 仮定。系→環境の結合有無 |
| **d=2** | **Va** | Value | E (認識) ↔ P (実用) | VFE/EFE の分解 |
| **d=2** | **Fu** | Function | Explore ↔ Exploit | EFE による行動選択 |
| **d=2** | **Pr** | Precision | C (確信) ↔ U (留保) | 予測誤差の逆分散 |
| **d=3** | **Sc** | Scale | Mi (微視) ↔ Ma (巨視) | 階層的MB入れ子構造 |
| **d=3** | **Vl** | Valence | + (肯定) ↔ - (否定) | VFE勾配方向の符号 |
| **d=2** | **Te** | Temporality | Past ↔ Future | VFE≠EFE → 時間的非対称性 |

### Afferent × Efferent の4象限

```
              Efferent (出力)
              Yes        No
Afferent  Yes │ S∩A(反射)│ S(知覚) │
(入力)    No  │ A(行為)  │ I(推論) │
```

- **S (知覚)**: 環境から受け取るのみ (受容)
- **I (推論)**: 外部と結合しない内部処理
- **A (行為)**: 環境に働きかけるのみ (出力)
- **S∩A (反射弧)**: 受容と出力が直結 (中動態 = H-series)

## 4. 族内関係 (3型 × 各族)

各族のT1-T6の6動詞間に、3方向のペアリングが存在する:

```
       極1            極2
  S   T5 ─────H───── T6       H型 (自然変換): 同一Flow・極反転
      │ ╲             │ ╲       T5↔T6, T1↔T2, T3↔T4
      D   X           D   X
      │     ╲         │     ╲   D型 (随伴 ⊣): 同一極・Flow反転
  I   T1 ─────H───── T2       T5⊣T1, T6⊣T2 (S⊣I)
      │ ╲             │ ╲       T1⊣T3, T2⊣T4 (I⊣A)
      D   X           D   X
      │     ╲         │     ╲   X型 (双対 ↔): 対角
  A   T3 ─────H───── T4       T5↔T2, T6↔T1 (S/I間)
                                T1↔T4, T2↔T3 (I/A間)
```
