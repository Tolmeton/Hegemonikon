## 🎛️ 修飾子の意味

### 深度修飾子

| 修飾子 | 意味 | 実行方法 |
|:-------|:-----|:---------|
| `/verb` | L2 標準 | Claude が直接 WF を読んで実行 |
| `/verb-` | L1 縮約 | 最小限の実行。速さ優先 |
| `/verb+` | L3 深化 | hermeneus_run 経由。自動検索 (Gnōsis/Periskopē/S2) 付き |

### Dokimasia 修飾子 — 座標の色を混ぜる

**構文**: `/verb[座標略:極]` — 1つの認知行為に別の座標のニュアンスを着色する。

| 座標略 | 座標 | 極1 | 極2 |
|:-------|:-----|:----|:----|
| `Va` | Value | `E` | `P` |
| `Fn` | Function | `Ex` | `Ep` |
| `Pr` | Precision | `C` | `U` |
| `Sc` | Scale | `Mi` | `Ma` |
| `Vl` | Valence | `+` | `-` |
| `Tm` | Temporality | `Pa` | `Fu` |

**修飾子 vs 合成の使い分け**:
- `/sap[Sc:Mi]` → 1行為: 精読しつつ微細部に注目 (**修飾子**)
- `/sap * /prs` → 2行為: 精読と注視を同時に (**並列合成**)
- `/sap >> /prs` → 2段階: まず精読、次に注視 (**順次合成**)

**実用例**:
| CCL | 意味 |
|:----|:-----|
| `/sap[Sc:Mi]+` | ミクロ部分を高精度で精読 |
| `/the[Pr:U,Sc:Ma]` | 広域を低精度で観照 (≈ /ski+/per) |
| `/exe[Sc:Ma]` | 全体的に弱点を吟味 |
| `/his[Sc:Mi]` | 特定行の変更履歴 (git blame 的) |
