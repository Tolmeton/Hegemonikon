# DX-014-S1: Step① Flow (d=0) の一意性 — 証明スケッチ

> **DX**: DX-014 (形式的導出 + 弱2-圏)
> **Step**: ① Flow (d=0) の一意性
> **目標**: MB partition → I↔A 分離が FEP の必然的帰結であることの半形式的証明
> **状態**: 🟢 概ね証明済み (文献的根拠あり) → 形式化する

---

## 主張 (Claim)

> **C1**: FEP のもとで存続する任意のシステムは、Markov blanket partition を持ち、
> その partition は **推論 (I) と行動 (A) の二値の区別** を一意に定める。
> この二値の区別が Flow 座標 (d=0) である。

---

## 証明スケッチ

### 定義

| 記号 | 定義 | SOURCE |
|:-----|:-----|:-------|
| x | 確率的動力系の状態空間 | — |
| η | 外部状態 | Spisak §3.1 |
| μ | 内部状態 | Spisak §3.1 |
| s | 感覚状態 (sensory) | Spisak §3.1 |
| a | 活動状態 (active) | Spisak §3.1 |
| b = {s, a} | blanket 状態 | Friston 2023 |
| F(s, a, μ) | 変分自由エネルギー (VFE) | Spisak eq.(3) |

### 公理

> **A0** (FEP): 存続するシステムは VFE を最小化する。

### 補題1: Markov blanket の必然性

**主張**: A0 のもとで存続するシステムは、状態空間を {η, s, a, μ} に partition する Markov blanket を**持たなければならない**。

**論証**:

1. 存続 = 定常分布 p(x) が存在し、エントロピー H[p(x)] が有界
   (SOURCE: Friston 2023 — particular partition の定義)
2. 定常分布の維持 = η と μ が条件付き独立: η ⊥ μ | b
3. 条件付き独立 = blanket 状態 b = {s, a} が η と μ の間に介在
4. ∴ MB partition {η, s, a, μ} は存続の**必要条件**

**強さ**: 🟡 (SOURCE: Friston et al. 2023, Spisak §3.1)
**弱さ**: Biehl et al. 2021 は「sparse flows + uncoupled autonomous states」の条件が生物系で常に成立するかを疑問視
**反駁**: Friston は Biehl への反論で「条件は十分条件であり、より弱い条件でも MB は成立する」と主張 (Friston/Da Costa/Parr)

### 補題2: I↔A の二値分離の一意性

**主張**: blanket 状態 b = {s, a} の分離は**一意**であり、s と a の役割は交換不能。

**論証**:

1. MB partition の定義より:
   - 外部 η は s にのみ影響、μ には直接影響しない
   - 内部 μ は a にのみ影響、η には直接影響しない
2. この非対称性は flow の方向で決まる:
   - s: η → s → μ (世界 → 知覚 → 推論) = **I 方向 (推論)**
   - a: μ → a → η (推論 → 行動 → 世界) = **A 方向 (行動)**
3. s と a を入れ替えると:
   - η → a → μ (外部が行動を通じて内部に影響) — MB の定義に矛盾
   - ∵ a は定義上「内部状態にのみ依存し、外部に影響を与える」
4. ∴ s と a は交換不能 → I/A の区別は一意

**強さ**: 🟢 (flow の方向が partition の定義に組み込まれている)
**注意**: これは s と a の**ラベリング**であり、より深いレベルでは「入力/出力の区別」。ラベリングの一意性は情報理論的に自明。

### 定理: Flow の一意性

**主張**: (C1) A0 のもとで存続する任意のシステムに対し、Flow 座標 I↔A は一意に定まる。

**証明**:

1. [補題1] A0 → MB partition {η, s, a, μ} は必然
2. [補題2] s (入力) と a (出力) の区別は一意
3. I = s 方向 (推論), A = a 方向 (行動) と定義する
4. ∴ Flow: I↔A は MB の存在とその方向から一意に決定される
5. この構成は FEP 以外の追加仮定を必要としない → d=0

Q.E.D. (半形式的)
