## 証明スケッチ (修正版)

### 補題J': sgn(−ΔF) の存在

1. FEP (A0): F(q) は実数値汎関数 → F(t) ∈ ℝ
2. Temporality (A3): F(t) と F(t+1) が共に利用可能
3. ΔF = F(t+1) − F(t) ∈ ℝ
4. sgn(−ΔF) ∈ {+1, 0, −1} — 数学的に自明
5. **+1** = 改善 (F 減少) → 接近 (+)
6. **−1** = 悪化 (F 増加) → 回避 (−)
7. **0** = 変化なし → 中立

**強さ**: 🟢 — F が実数値であることと Temporality から直接

### 補題K': Precision との独立性 (v3.0 — 決着)

**旧主張 (誤り)**: Precision ≈ |ΔF| → 極座標分解で独立でない

**修正**: π ≠ |ΔF|。これは根本的に異なる量。

| 量 | 定義 | 作用する対象 | 時間的性質 |
|:---|:-----|:---------|:---------|
| **Precision π** | 1/σ² (予測誤差の逆分散) | 各瞬間の予測の質 | 瞬時的 |
| **Valence sgn(−ΔF)** | F(t+1) − F(t) の符号 | F の時間変化 | 時間差分 |

(SOURCE: Friston 2019, arXiv:1906.10184, 293引用 — π の定義)

**4組合せテスト** (独立性の操作的検証):

- 高π + Valence(+) = 「自信を持って改善中」✅ 意味あり
- 高π + Valence(−) = 「自信を持って悪化中」✅ 意味あり
- 低π + Valence(+) = 「不確かだが改善中」✅ 意味あり
- 低π + Valence(−) = 「不確かで悪化中」✅ 意味あり

**結論**: ✅ **独立** — π と sgn(−ΔF) は異なる量に作用する。極座標分解の主張は誤った同一視に基づいていた。

**強さ**: 🟢

### 補題L': 他座標からの独立性

| 座標 | sgn(−ΔF) から独立か | 理由 |
|:-----|:------|:-----|
| Flow (I/A) | ✅ | Flow は方向 (I→Aの向き)、Valence は改善/悪化 (Fの変化) |
| Value (E/P) | ✅ | Value は「何を最適化するか」、Valence は「最適化の結果」 |
| Function (Explore/Exploit) | ✅ | Function は「どう行動するか」、Valence は「行動の帰結の評価」 |
| Precision (C/U) | ✅ | π=1/σ² (瞬時的精度) ≠ sgn(−ΔF) (時間変化の符号) |
| Scale (Mi/Ma) | ✅ | Scale は空間的レベル、Valence は時間的変化の符号 |
| Temporality (Past/Future) | ⚠️ | **依存あり**: ΔF の計算に Temporality が必要 |

### 定理 (v3.0)

1. [A0] FEP → F は実数値
2. [A3] Temporality → F(t) と F(t+1) が利用可能
