# F1 論文消化: iGRPO + The Missing Reward

> 日付: 2026-02-14 | SOURCE: arXiv HTML 直接読了

## 1. iGRPO (arXiv 2602.09000)

**核心**: GRPO の2段階拡張 — Stage 1 (探索的ドラフト生成) + Stage 2 (条件付き改良)

**HGK 接続点**:

| iGRPO 概念 | HGK 対応 | 摩擦/一致 |
|:-----------|:---------|:---------|
| 動的自己条件付け (d̂_θ) | Handoff = 前セッションの最良出力で条件付け | ✅ 一致。ただし HGK は「前セッションの最良」、iGRPO は「同一ステップ内の最良」 |
| Stage 1 (探索) | /zet, /noe+ (深い探索) | ✅ 構造的同型 |
| Stage 2 (改良) | /ene, /dev (条件付き実行) | ✅ Stage 2 のみ勾配更新 = HGK では実装のみがコードを変える |
| 暗黙的カリキュラム | 早期: 弱い Handoff, 後期: 高品質 Handoff | ✅ HGK の Handoff 品質は共進化で改善 |
| 推論時はドラフトなし | HGK は推論時にも Handoff を使う (常時) | ❌ 摩擦。iGRPO は訓練時のみ自己条件付け |

**新知見**: iGRPO の「coupled dynamical system」= HGK の共進化。θ が変わると d̂_θ も変わる = Creator が変わると BC/Handoff も変わる。

## 2. The Missing Reward (arXiv 2508.05619)

**核心**: Grounded-Agency Gap = 外部報酬なしで自律的に目標設定・追求する能力の欠如

**HGK 接続点**:

| TMR 概念 | HGK 対応 | 摩擦/一致 |
|:---------|:---------|:---------|
| Grounded-Agency Gap | HGK の BC 群が「疑似的」に埋める | ⚠️ 部分的。HGK は Creator 介入で Gap を補う (開いた系) |
| FEP で報酬を代替 | HGK の基盤 = FEP | ✅ 直接的一致 |
| 情報利得 > ヒューリスティック探索 | /noe の Phase 構造 | ✅ 構造化された情報獲得 |
| Energetic-Bounded Rationality | BC-18 (コンテキスト予算) | ✅ 驚くほど直接的対応 |
| 「一日の仕事を終える」行動 | /bye (セッション終了の自己判断) | ✅ TMR の予言を HGK は既に実装 |

**衝撃的一致**: TMR が「AIF エージェントは疲れたら休む」と論じている行動を、HGK は BC-18 + /bye で既に実装している。

## 射の発見

- **iGRPO → HGK**: Handoff を「静的 ICL」から「動的自己条件付け」に格上げする設計変更の示唆
- **TMR → HGK**: Grounded-Agency Gap の概念で「なぜ Creator 介入が必要か」を理論的に説明可能
- **TMR × iGRPO**: 報酬信号の「内在化」が両論文のテーマ。HGK は BC 群を内在的報酬として使う
