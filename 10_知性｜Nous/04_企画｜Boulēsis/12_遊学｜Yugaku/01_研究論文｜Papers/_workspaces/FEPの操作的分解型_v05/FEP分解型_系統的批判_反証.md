## 致命的欠陥 (論文が成立しない可能性)

### F1. 「操作的分解の型」は well-defined でない

**論文の主張**: FEP は 8 つの独立な操作的分解型を持つ。

**反論**: 「操作的分解の型」という概念自体に数学的定義がない。論文は以下の問いに答えていない:

- **型の同値関係は何か？** D4 (VFE = Accuracy − Complexity) と D5 (EFE = Ambiguity + Risk) は「同じ型」か「異なる型」か。両方とも「汎関数を2項に分解する」操作であり、同値関係の定義次第で型数は大幅に変わる。
- **独立性の基準は何か？** 論文は「D6 は D4 から導出可能」と主張するが、その「導出」の形式的定義がない。依存グラフは著者の主観的解釈に過ぎない。
- **数え上げの完全性は保証されない。** 「D1-D10 以外はない」を証明する方法がない。FEP 文献は膨大であり、見落とした分解が存在する可能性は排除できない。

**影響**: 型数「8」は著者の解釈に依存する恣意的な数字。異なる同値関係や粒度で数えれば 4 にも 15 にもなりうる。

### F2. 循環論法: HGK の座標が分解の枠組みを決定している

**論文の主張**: FEP の操作的分解を bottom-up に数えたら 8 型 (→7 座標) になった。

**反論**: 分解の列挙自体が HGK の座標を暗黙の prior として行われている。

- D3 (Inference ⊣ Action) を「独立な型」として数えるのは、HGK が Function (Explore↔Exploit) を座標としているからである。
- D8 (Precision weighting) を「型」として数えるのは、HGK が Precision を座標としているからである。
- D10 (Interoceptive gradient sign) を入れるのは、HGK が Valence を座標としているからである。

もし HGK を知らずに FEP だけを読んだ人が分解を列挙したら、D3 を D2 の内部構造（独立ではない）、D10 を D4 の特殊ケース（符号の区別は型の差ではない）と見なす可能性が高い。

**つまり**: FEP → 8 型 → HGK ではなく、**HGK → 8 型の選別基準 → 「8 型に見える」** という循環。

### F3. Fiber bundle 構成は trivial

**論文の主張**: dissipative flow は e-projection に射影され、solenoidal flow は gauge flow（ファイバー内）に対応する。

**反論**: この構成は数学的にほぼ自明であり、新しい洞察を含まない。

- 「$\mathfrak{I}$ が一定 → パラメータ $\theta$ が一定 → ファイバー内」は指数族の定義からの直接的帰結。定理と呼ぶには trivial すぎる。
- 全空間 X の次元は膨大（状態空間のすべての粒子の位置）であり、底空間 M の次元は小さい。ファイバーの次元が大きいのは自明。「solenoidal flow は射影されない」と言っているが、ほぼすべての flow がファイバー方向を持つ。
- 構造群の同定が完全に欠落しており、「fiber bundle」と呼ぶための必要条件（局所自明性）が検証されていない。

**影響**: §5 は「正しいことを言っているが、新しくない」――最悪の学術的立場。

## 重大な欠陥 (主張が大幅に弱まる)

### F4. d_eff の分析が指数族カテゴリカルに偏っている

**反論**: 論文は 3 つの「おもちゃモデル」で分析しているが:

- **模型 (c)** のカテゴリカルの Fisher metric $G = \text{diag}(1/p_i)$ は指数族カテゴリカルの **特殊な座標系 (natural parameters)** にのみ成立する。実際の active inference generative model はカテゴリカルではなく、深い階層的非線形モデルである。
- **模型 (b)** の階層ガウスは線形ガウスのみ。非線形の場合、Fisher metric の解析的計算は不可能であり、数値計算が必要。
- 「d_eff は模型依存」という結論は正しいが、**現実的な認知モデル**（例: 深層 active inference、POMDPs with deep transitions）での d_eff は不明のまま。

**影響**: おもちゃモデルでの結論を一般化する根拠が弱い。

### F5. Construction distance d は post-hoc な解釈

**論文の主張**: d=0, d=1, d=2 の階層は FEP の論理構造と一致する。

**反論**: 「追加仮定の数」の数え方が恣意的。

- D4 (VFE) は本当に「1つの追加仮定」(variational density) か？ VFE の定義には変分密度 q の選択 + Kullback-Leibler divergence の定義 + 期待値の交換定理が必要であり、「1つの仮定」と数えるのは過小評価。
- D9 (深い粒子) は「2つの追加仮定」か？ D2 の再帰的適用は「追加仮定」ではなく「同じ仮定の反復適用」とも読める。その場合 d=1 であるべき。
- d の値は HGK の d=0,1,2 と一致するよう後付けで調整されている疑い。

### F6. 反証プロトコルが反証実験で自壊している

**論文の主張**: P2 (H* ≈ 2.46 普遍定数) は部分的に棄却。

**反論**: これは論文自体の問題ではなく、**論文の物語が破綻していることの証拠**。

- §4 の結論は「d_eff は模型依存 → universal-7 は棄却」
- Gemini がこれを延長して「条件付きで回復できる」と主張
- Claude がそれを反証して「条件自体も棄却」
- 論文に書かれている結論は最終的に「d_eff は模型依存」のまま

しかし、論文は §3 の操作的型分析で 7 を回復しようとしている。この「planB」自体が ad hoc な回避策であり、最初から 7 を得るためにデザインされた感がある。

### F7. HGK との対応表 (P6) が評価不能

**論文の主張**: 7 つの操作的型が HGK の 7 座標に対応する。

**反論**: 対応の多くが「アナロジー」止まりであり、同型 (isomorphism) は 1 つもない。

- Value ↔ D2 (Internal/External) は「内部/外部」の概念的類似。だが D2 は 4 分割 (η,s,a,μ) であり、2 値 (Internal/External) ではない。
- Function ↔ D3 (Inference/Action) は「Explore/Exploit」とラベルを貼っただけ。
- Temporality ↔ D7 (VFE/EFE) は「過去/未来」と「推論/計画」の混同。VFE は現在の推論であり、「過去」ではない。

10 問中、厳密な同型は 0、準同型は 2-3、残りはラベルの類似。これは学術論文として主張するには弱すぎる。

## 中程度の欠陥

### F8. "implicit" の恣意性

「D1 (Helmholtz) は HGK に implicit に吸収」は都合が良すぎる。8 型あるのに 7 座標しかないので、1 つを「暗黙」とラベルして辻褄を合わせているだけ。なぜ D1 が implicit で D9 が explicit なのか？ その選択基準がない。

### F9. 非指数族への一般化が完全に欠如

Friston 2019 の NESS 密度は指数族に限定されない。多くの実用的な NESS は指数族ではない。§5 の fiber bundle 構成は指数族仮定に依存しており、一般的な FEP への適用可能性が不明。

### F10. 証明のスケッチが証明ではない

§5 の "Proof Sketch" は数学的に不完全であり、少なくとも以下が欠落:
- $\sigma$ の smoothness の証明 (局所エルゴード性の仮定の正当化)
- 局所自明性 (fiber bundle の定義条件) の検証
- 接続 (connection) の構成
