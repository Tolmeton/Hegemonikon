## Abstract

「すぐれた知性は優れた溶媒である」—— この直感を圏論的に定式化し、関手 F: Chem → Cog が**忠実**であることを示す。鍵となるのは、Gibbs 自由エネルギーの温度パラメータ T と変分自由エネルギー (VFE) の精度パラメータ β の対応 (β = 1/T) であり、これにより Gibbs ↔ VFE の同型は2項分解の骨格だけでなく制御構造まで保存される。さらに、embedding システム Hyphē の実験結果 (310+ 実験) が「溶媒仕様」の具体的検証を提供する: ker(G)_{Session} = {Scale, Valence} は「この溶媒が溶かせない溶質」の実測であり、embedding anisotropy は「溶媒の極性が不十分である」ことの物理的実証である。

## §2. Gibbs ↔ VFE の同型 — T ↔ β による完成

### 2.1. 標準形の構造的類似

| Gibbs 自由エネルギー | 変分自由エネルギー |
|:---------------------|:-------------------|
| G = H − TS | F = −Accuracy + Complexity |
| H: エンタルピー (結合エネルギー) | Accuracy: $\mathbb{E}_q[\ln p(o \mid s)]$ |
| −TS: エントロピー × 温度 | Complexity: $D_{KL}[q(s) \| p(s)]$ |
| 最小化 → 平衡状態 | 最小化 → 信念の更新 |

この対応は「2項に分解される自由エネルギーの最小化」という骨格レベルで明瞭だが、一見すると重要な非対称性がある: Gibbs の T は外部から制御可能な物理パラメータだが、VFE の KL 項にはそのような制御パラメータが見当たらない。

### 2.2. 温度付き VFE (Tempered Variational Inference)

この非対称性は、VFE の温度パラメータ化により解消される:

$$F_\beta = -\beta \cdot \text{Accuracy} + \text{Complexity}$$

ここで β は **精度** (precision) = **逆温度** (inverse temperature) である。

| Gibbs (T) | VFE (β = 1/T) | 物理的意味 |
|:-----------|:---------------|:----------|
| T → 0 | β → ∞ | **結晶化**: エネルギー最小化 / 予測精度最大化。信念が固まる |
| T → ∞ | β → 0 | **溶解**: エントロピー最大化 / 事前分布への回帰。信念が溶ける |
| 加熱 (T ↑) | 注意の弛緩 (β ↓) | 構造を解きほぐす。探索モード |
| 冷却 (T ↓) | 注意の集中 (β ↑) | 構造を固定する。活用モード |
| 外部制御 | 外部制御 | 化学: 加熱器。認知: 注意・神経調節 (ドーパミン, アセチルコリン) |

### 2.3. 制御可能性の保存

Friston (2010) [1] は precision を inverse temperature と明示的に同定している。これは比喩ではなく、統計力学の指数型分布族における自然パラメータの対応である:

- **Gibbs 分布**: $p(x) \propto \exp(-E(x) / T)$
- **変分事後分布**: $q(s) \propto \exp(\beta \cdot \ln p(o|s)) \cdot p(s)$

T ↔ 1/β は連続的全単射であり、射は潰れていない。

**帰結**: Gibbs ↔ VFE の同型は、2項分解の骨格だけでなく、制御構造 (温度パラメータによる探索-活用バランスの外部制御) まで保存する。F: Chem → Cog はこの射を潰していない。

## §3. 極性 → 構造的親和性 — 媒介圏による忠実化

### 3.1. 表面的な非忠実性

「like dissolves like」(似たものが似たものを溶かす) は化学の基本原則である。水（極性溶媒）は塩（極性溶質）を溶かすが、油（非極性溶質）を溶かさない。

直感的には「極性 → 訓練目標」と写したくなるが、メカニズムが異なる:

| Chem | Cog | 差異 |
|:-----|:----|:-----|
| 水素結合 / Van der Waals 力 | attention weight / gradient descent | 相互作用の物理が異なる |
| 溶媒和殻の幾何学 | embedding 空間の位相 | 空間構造が異なる |

### 3.2. 媒介圏 Affinity による解消

この非忠実性は、抽象度を一段上げることで解消できる。

**圏 Affinity** (親和性):

| 対象 | 射 |
|:-----|:---|
| 構造 (Structure) | 構造的相互作用 (Structural Interaction) |
| 親和性パラメータ δ | 選択的結合 (Selective Binding) |

化学の「極性」も認知の「訓練目標」も、Affinity 圏の対象「構造的親和性パラメータ」の異なる実装 (representation) である:

```
       Affinity
      /        \
   Chem        Cog
   極性        訓練目標
   "like dissolves like"  "似た構造のモデルが似た構造の情報を処理できる"
```

両者はともに「構造的な適合性があるとき結合し、ないとき結合しない」という射を保存する。射が潰れていたのではなく、直接対応を探す抽象度が低すぎた。

### 3.3. 溶解度パラメータ δ の認知的対応

化学の Hildebrand 溶解度パラメータ δ は「この溶媒にこの溶質が溶けるか」を定量的に予測する。

認知的対応:

$$\delta_{\text{intelligence}}(\text{agent}, \text{information}) \propto \frac{1}{\text{VFE}(\text{agent}, \text{information})}$$

- δ が高い (溶ける) ↔ agent の generative model が information の構造を予測できる (VFE が低い)
- δ が低い (溶けない) ↔ agent の generative model と information の構造が不整合 (VFE が高い)

これは主エッセイ §5.2 のアナロジー効率 η と構造的に同型:

$$\eta = \frac{\Delta\text{Accuracy}}{\Delta\text{Complexity}} \quad \longleftrightarrow \quad \delta \propto \frac{1}{F}$$

効率が高い = よく溶ける。

## §4. 記憶の化学 — 忠実な射の展開

### 4.1. 記憶プロセスの完全対応表

| 記憶プロセス | 化学プロセス | FEP 操作 | 忠実性 |
|:------------|:-----------|:---------|:------|
| エンコーディング | 溶解 (dissolution) | VFE 最小化 (モデル更新) | ✅ 忠実: 溶質の分子間結合を切り水和殻で包む ↔ 情報の既存文脈を解体し内部モデルに統合 |
| 保持 | 溶液状態 | 信念の維持 | ✅ 忠実: 溶質が溶媒中に分散 ↔ 情報がモデル全体に分散 |
| 想起 | 結晶化 (crystallization) | EFE 最小化 (能動推論) | ✅ 忠実: 核形成 ↔ 手がかり (retrieval cue)。過飽和が必要条件 |
| 忘却 (検索失敗) | 沈殿 (precipitation) | cue の precision 低下 | ✅ 忠実: 溶解度低下で構造は残るが取り出せない |
| 忘却 (痕跡消失) | 分解 (decomposition) | シナプス結合の消失 | ✅ 忠実: 分子結合の破壊 ↔ 構造自体が消滅 |
| 固定化 (sleep) | **Ostwald 熟成** | Complexity 最小化 | ✅ **最も美しい対応** |
| 干渉 | 共沈 / 置換反応 | 信念の競合 | ✅ 忠実 |
| Context Rot | 飽和溶液 | ΔF ≈ 0 | ✅ 忠実: Gibbs で ΔG = 0 |

### 4.2. Ostwald 熟成 ↔ 記憶固定化 — 本論文の最良の射

**Ostwald 熟成** (Ostwald ripening): 小さな結晶が溶解し、大きな結晶に再析出する。表面エネルギー最小化。

**記憶のスリープ固定化**: 個別のエピソード記憶が、睡眠中に統合され、汎化された知識 (schema) に再編成される。

| Ostwald 熟成 | スリープ固定化 | 共通構造 |
|:------------|:-------------|:---------|
| 小結晶の溶解 | エピソード記憶の解体 | 小さな構造を溶かす |
| 大結晶への再析出 | schema への統合 | 大きな構造に再結晶する |
| 駆動力: 表面エネルギー最小化 | 駆動力: Complexity 最小化 | 同じ変分原理 |
| 結果: 少数の大結晶 | 結果: 少数の汎用知識 | 同じ統計的構造 |

メカニズムの忠実性: 両者とも「小さな構造を溶かして大きな構造に再結晶する」操作であり、駆動力が変分原理（表面エネルギー / VFE の Complexity 項）の最小化である。因果構造が保存されている。

### 4.3. 過冷却 ↔ 知識の潜在的結晶化

過冷却 (supercooling): 溶液が飽和を超えているのに、核形成のきっかけがなく結晶化しない状態。

認知的対応: 知識が十分にあるのに、適切な retrieval cue がなくて想起されない状態。「喉まで出かかっている」(tip-of-the-tongue phenomenon) がこれに該当する。

核形成に必要なのは**種結晶** (seed crystal) であり、認知における種結晶は**良い問い** (good question) である。

## §5. Hyphē による検証 — 溶媒仕様の実測

### 5.1. Hyphē = 場⊣結晶理論

Hyphē (場⊣結晶理論) は、テキストデータを embedding 空間に溶解し (F: 場化)、意味的境界で結晶化する (G: 結晶化) システムである。

溶媒アナロジーとの対応:

| 化学 | Hyphē |
|:-----|:------|
| 溶媒 | embedding モデル (gemini-embedding-001, 768d) |
| 溶質 | セッションログ (テキストデータ) |
| 溶解 | F: テキスト → embedding ベクトル |
| 結晶化 | G: チャンク境界の検出 (similarity < τ) |
| 飽和溶液 | Context Rot (コンテキストウィンドウの上限) |
| 相転移温度 | **τ* ≈ 0.720** (実測, 858 similarity 点) |
| 不動点 | **Fix(G∘F)**: 全310実験で1-2回で収束 |

### 5.2. ker(G) = 溶媒が溶かせない構造

ker(G) (結晶化関手 G の核) = G が零に送る座標 = **結晶が正しく捨てる情報** = **溶媒が溶かせない溶質**。

HGK の6修飾座標のうち、3つのドメインそれぞれで正確に2座標が ker(G) に属する:

```
6座標:  Value  Function  Precision  Temporality  Scale  Valence
Desc:    保      保        保          捨         捨      保
Link:    保      保        保          捨         保      捨
Sess:    保      保        保          保         捨      捨
```

**ker(G)_{Session} = {Scale, Valence}**: セッション結晶は Scale (粒度) と Valence (評価) を溶かせない。

溶媒アナロジーによる読み替え:

| ker(G) の座標 | 溶媒アナロジーでの意味 |
|:-------------|:--------------------|
| Scale (粒度) | embedding は「このチャンクが Sprint のどこにいるか」という**巨視的位置**を溶解できない。局所溶媒は局所的にしか溶かせない |
| Valence (評価) | embedding は「これが良いか悪いか」という**価値判断**を溶解できない。溶媒は中性であり、溶質の善悪を区別しない |

### 5.3. Embedding Anisotropy = 溶媒の極性不足

Hyphē 実験 §12 で発見された問題:

- **precision gradient が検出できない**: cosine similarity の分散が圧縮され、チャンク品質の差異を弁別できない
- **根本原因**: **embedding anisotropy** — 3072次元中の有効次元 k_eff_90 ≈ 33。全ベクトルが狭い円錐内に集中

溶媒アナロジーでの解釈:

> **embedding anisotropy = 溶媒の極性不足**
>
> 溶媒 (embedding モデル) の極性が一方向に偏っている (anisotropic) ため、溶質 (テキスト情報) の多様な構造を均等に溶解できない。水がヘキサンを溶かせないのと同じ構造的理由で、768d embedding は Scale/Valence を溶かせない。

| 化学 | Hyphē 実験 |
|:-----|:----------|
| 溶媒の極性方向が限定的 | embedding の有効次元が 33/3072 (1.1%) |
| 極性の合わない溶質は溶けない | Scale/Valence 座標は embedding に溶けない |
| **対策: 混合溶媒** (極性+非極性) | **対策: 多層 embedding** (異なる task_type の組み合わせ) |
| **対策: 超臨界流体** (極性の境界を超える) | **対策: fine-tuned embedding** (Hyphē 専用モデル) |

### 5.4. Coherence Invariance = 溶液の恒常性

Hyphē 実験 §6 の最重要知見: τ (相転移温度) を変えてもチャンクの coherence は 0.807-0.815 で一定。

- G∘F ON: coherence range = 0.008 (τ 非依存)
- G∘F OFF: coherence range = 0.121 (τ 依存)

溶媒アナロジーでの解釈:

> 溶液の**浸透圧**は溶質の種類に依存するが、溶液を小分けにしても各容器の浸透圧は同じ (均質性)。Coherence Invariance は溶液の均質性の認知的対応である。G∘F (merge/split 不動点) が「撹拌」の役割を果たし、溶液全体の均質性を保証する。
