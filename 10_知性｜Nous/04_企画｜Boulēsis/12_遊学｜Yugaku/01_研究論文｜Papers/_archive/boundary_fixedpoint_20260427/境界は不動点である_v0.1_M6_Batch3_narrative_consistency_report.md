## 2. §3 / §5 接続

§5.1 冒頭に、§3.3 の第三項配置を既存圏論的 FEP 比較へ移す接続文が入った。
これにより、§5 は「圏論を使うかどうか」の論点ではなく、「境界の前段を扱うか、後段を扱うか」の同型問題として読める。

§5.6 の主張は、M2/M3 audit に合わせて限定された。
現在の文言は「本稿で比較した五つの既存研究の範囲では、closure operator の fixed point として Markov blanket を derive しない」であり、未調査の全圏論的 FEP への過剰一般化を避けている。

## 3. §5 / §6 接続

§6 冒頭は、§5 の「boundary がある後段」と「boundary の origin layer」の区別を受けて始まる。
そのため、autopoiesis 節は独立した寄り道ではなく、§5 の直交性主張を生命境界へ移す節として接続している。

§6.2 は Razeto-Barry の D1/D2 に基づき、自己境界を膜一般や膜成分の全内部産出と同一視しない。
§6.3 は §2 の T0 を呼び戻し、\(M_{\mathrm{auto}}(o,B)\)、\(L(o)\)、\(R(B)\)、\(B=L(R(B))\) の形で autopoiesis へ適用している。
これにより、§6 は §2 と §5 の双方へ戻る構造を持つ。

## 4. §3 / §6 接続

§6.5 は Bruineberg 批判を autopoiesis 文脈へ戻している。
Friston camp の autopoiesis-MB 接続を、条件付き独立から実在境界へ飛ぶ危険として整理し、T0 によって数学的層では dissolve するが、形而上学的層は bracket する、という §3 の規律を維持している。

このため、§6 は §3 の批判応答を再利用しているが、単なる反復ではない。
autopoiesis では「どの膜、どの代謝範囲、どの環境結合を自己に含めるか」が問題になるため、§3 の boundary realism 問題が生命境界の問題として具体化されている。

## 5. §6 / §7 接続

§7 は、§6 の近似不動点としての自己生成系から、時間発展中の blanket 探索へ移る。
単発の Markov blanket 検出ではなく、

\[
o_t\mapsto L(o_t)\mapsto R(L(o_t))
\]

の反復軌道を調べる計算課題として active inference を拡張している。

D 系列への接続も §7 末尾に追加された。
D4 は Galois 不動点を直接探索する active inference 実装、D5 は autopoiesis / cell biology への展開として明示され、§6 から §7 への派生線が見える。
