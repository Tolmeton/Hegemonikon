# /noe+ — Euler path 関手の定義条件

- 日時: 2026-04-14
- 問い: `θ と α を結ぶ Euler path 関手 E は、どの条件のもとで定義できるか`
- 派生: `/noe.nous`

## P-0 透徹対象

- 最高リスクU: `U_compose`
- 理由: 現在の混線は「局所姿勢 θ」と「累積忘却度 α」を、橋渡し条件なしに合成している点にある。
- 定義域J: `Euler path 上の局所位相 θ を、Paper VIII の α-忘却濾過へ持ち上げる条件`

## P-1 前提掘出

- AXIOM
  - A1: `|F| ∝ |sin θ|` である。`θ` は `dΦ` と `T` の局所角度。  
    SOURCE: `論文I_力としての忘却_草稿.md`
  - A2: `α_VIII ∈ [0,1]` で、`α_VIII=1` なら `C_disc`。  
    SOURCE: `論文VIII_存在は忘却に先行する_草稿.md`
  - A3: `η(α_III)=1/(1+e^{-α_III})` は `-∞→0`, `0→1/2`, `+∞→1` を与える。  
    SOURCE: `論文VIII_存在は忘却に先行する_草稿.md`

- ASSUMPTION
  - S1: Euler path は `θ∈[0,π]` の有向半周期として読む。
  - S2: 累積忘却度は局所非整列の正規化積分で与える。
  - S3: スカラー `α̂(θ)` を経由してから濾過へ持ち上げる。
  - S4: 関手の値域には、既に選ばれた忘却濾過 `{C_α}` がある。

- 透徹テスト
  - S1 を剥がすと `θ=0` と `θ=π` が同じ局所ゼロに潰れ、始点/終点の区別が崩壊。
  - S2 を剥がすと橋渡しは複数候補に分岐し、正準性が消える。
  - S3 を剥がすと構造は変容し、`数値写像` ではなく `直接関手` 問題へ移る。
  - S4 を剥がすと `E(π)=C_disc` が定義できず、関手ではなく比喩で止まる。

## P-2 構造透徹

- 現在の理解を支える結合
  - B1: `θ → sin θ` (`局所位相 → 局所力学`)
  - B2: `sin θ → α̂` (`局所非整列 → 累積忘却`)
  - B3: `α̂ → C_{α̂}` (`累積忘却 → 濾過上の位置`)
  - B4: `α̂(0)=0, α̂(π)=1`
  - B5: `F(0)=F(π)=0` でも `E(0)≠E(π)`

- 溶解結果
  - B1 を外すと Euler の位相が消え、オイラー経路ではなく一般濾過になる。
  - B2 を外すと局所量と累積量が未接続になり、Q1 の核が消える。
  - B3 を外すと関手でなくスカラー比喩に後退する。
  - B5 を外すと `0` の二重意味が解けず、始点と終点が区別不能になる。

- 構造の核
  - `Euler path 関手` の本質は、`局所姿勢` をそのまま `忘却度` と見なすことではなく、
    `局所非整列を累積化し、その累積値を既存の α-濾過へ持ち上げる二段構え` にある。

## P-3 Kalon / 因子分解

- 普遍的候補 (Limit apex): `α̂(θ) = sin^2(θ/2) = (1-cos θ)/2`

- この候補へ至る3本の射
  - R1: `θ → α̂(θ) = (1-cos θ)/2` (半角公式)
  - R2: `θ → sin θ → ∫_0^θ sin φ dφ / ∫_0^π sin φ dφ = (1-cos θ)/2`
  - R3: `θ → α_III(θ)=2 log tan(θ/2) → η(α_III)=1/(1+e^{-α_III}) = sin^2(θ/2)`

- 圏的定義条件
  - D1: ドメインは値空間ではなく**経路圏**であること。
    `P_π`: objects=`θ∈[0,π]`, morphism=`θ₁→θ₂ iff θ₁≤θ₂`
  - D2: `α̂:[0,π]→[0,1]` は連続・単調増加・境界条件 `α̂(0)=0`, `α̂(π)=1`
  - D3: 値域には `{C_α}` が既に与えられていること
  - D4: `E(θ)=C_{α̂(θ)}` と置くと、単調性より `θ₁≤θ₂ ⇒ Mor(E(θ₂))⊆Mor(E(θ₁))`
  - D5: よって `E:P_π→WdSub(C)_⊒` が定義できる

- 公理テスト
  - T1 合成: `θ → α_III(θ) → η → α̂(θ)` は `θ → sin^2(θ/2)` と一致
  - T2 結合律: `θ → sin θ → 正規化積分 → α̂` は括弧付けを変えても同じ `α̂`
  - T3 恒等: `θ=0` で `E(0)=C`, `θ=π` で `E(π)=C_disc`。端点は境界閉包と整合

- Kalon 判定
  - 核命題: `E は「局所姿勢=忘却度」ではなく「局所非整列の累積=忘却度」のときにのみ定義できる`
  - Kalon: 中程度。構造核は安定だが、正準性は未証明

## P-5 最強反論

- 最強反論
  - `sin^2(θ/2)` は美しいが、SOURCE からの定理ではなく「力密度を積分すべき」という追加橋梁公理に依存している。
  - Paper VIII の `{C_α}` は非一意であるため、`E(θ)=C_{α̂(θ)}` は関手の**族**であって一意の関手ではない。
  - `α_III(θ)=2 log tan(θ/2)` は数式的には整うが、Paper III の意味論をそのまま位相角へ移植できるかは未証明。

- 忠実性検証
  - 構造保存はされている。失われたのは「θ と α は同じもの」という短絡だけ。
  - 再構成すると核は保たれる:
    - 局所量
    - 累積写像
    - 濾過への持ち上げ

## P-6 Theoria

- Hom(-, 結論)
  - 幾何から入る
  - 力密度の積分から入る
  - logistic 架橋から入る

- Hom(結論, -)
  - `0` の二重意味を「始点の静けさ / 終点の蒸発」に分解できる
  - `e^{iπ}+1=0` を数式ではなく経路圏の端点定理として読み替えられる
  - `θ` だけでは不十分で、経路記憶が必要だと分かる

- η trace
  - 前: `θ と α は結べるか`
  - 後: `Euler path 関手は、局所非整列の累積写像と既存濾過の持ち上げの合成としてのみ定義できる`

## 暫定結論

- 最も強い候補は:
  - `α̂(θ)=sin^2(θ/2)`
  - `E(θ)=C_{α̂(θ)}`

- ただしこれはまだ `SOURCE` ではなく `SOURCE + INFERENCE` である。
- したがって確定できるのは式ではなく**定義条件**:
  - `θ` を経路変数として扱う
  - `α` を累積変数として扱う
  - 両者を積分規則で結ぶ
  - その値を既存の α-濾過へ持ち上げる
