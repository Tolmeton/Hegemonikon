# /ele+ — Euler path 最小モデルの弱点マップ

- 日時: 2026-04-14
- 派生: `/ele.refute`
- 対象: `α_E(θ)=sin²(θ/2)` を Euler path 関手の橋として置く最小モデル
- 目的: `dα/dθ ∝ sin θ` を確定できない理由を切り分け、何を足せば確定へ進めるかを明示する

## P-0 Prolegomena

- 反駁対象:
  - `Euler path 最小モデルは、そのまま橋梁公理 "dα/dθ ∝ sin θ" へ昇格できる`
- 建設的目標:
  - モデルを壊すことではなく、どの追加公理がなければ昇格できないかを特定する
- 種類:
  - 前提 / 論理 / 定義面

## P-1 Steel-Man

この最小モデルが強いのは事実である。

1. 始点・中点・終点が自然だ。
   - `α_E(0)=0`
   - `α_E(π/2)=1/2`
   - `α_E(π)=1`
2. Paper I の局所力学 `F ∝ sin θ` と形が噛む。
3. Paper VIII の α-忘却濾過 `{C_α}` に素直に持ち上がる。
4. logistic 架橋 `η: α_III → α_VIII` とも相性がよい。
5. `e^{iπ}+1=0` を `(F1)` から `(F3)` への動的経路として読むための、最小の図式になっている。

したがって、これは単なる思いつきではなく、**複数の面から同じ形に収束している良い候補**である。

## P-2 Contradiction Hunting

### 1. 論理矛盾: `α` が型未分化

- Paper I の `α` は忘却強度。
- Paper III の `α_III` は接続パラメータ。
- Paper VIII の `α_VIII` は濾過進行度。

ここで `dα/dθ` と書くと、**どの α を微分しているのかが未定義**になる。  
現状の式は美しいが、まだ well-typed ではない。

### 2. 事実欠落: SOURCE は `θ → α` をまだ与えていない

本文 SOURCE が与えているのは次の三本だけである。

- `θ → F, κ` （Paper I）
- `α_VIII → C_α` （Paper VIII）
- `α_III → η(α_III)=α_VIII` （Paper VIII §6.7）

しかし **`θ → α_VIII` の直接規則**はまだ本文に存在しない。  
したがって `dα/dθ ∝ sin θ` は本文定理ではなく、再構成仮説に留まる。

### 3. 暗黙前提: `θ` は点の角度であって、まだ経路座標ではない

`θ` は

`θ(f,α) = arccos(<dΦ_α,T_α> / (|dΦ_α||T_α|))`

として局所的に定義されている。  
これは「その点でどれだけずれているか」を与えるが、まだ

- どの経路上で
- どの向きに
- どの速度で

進むかを与えていない。  
`dα/dθ` を書くには、`θ` が **有向 Euler path 上の単調パラメータ**であることを別途宣言しなければならない。

### 4. 反例: 端点条件だけでは `sin²(θ/2)` は一意に出ない

`x(θ) := (1-\cos θ)/2` と置く。  
すると

- `x(0)=0`
- `x(π/2)=1/2`
- `x(π)=1`

を満たす。  
しかし

`h(x):=3x²-2x³`

とすれば、`h(x(θ))` も同じ 3 点を保ちつつ単調増加であり、なおかつ `x(θ)` とは別の曲線である。  
つまり **始点・中点・終点の一致だけでは、候補は無数にある。**

### 5. 帰結矛盾: 積分核が未決定

仮に「局所量を累積する」としても、何を積分すべきかがまだ定まっていない。

- `sin θ`
- `|F|`
- `κ`
- `√κ`
- `Im(ω)=w·sin θ`
- `||dΦ||·||T||·sin θ`

のどれを採るかで、得られる累積則は変わる。  
いまの最小モデルは `sin θ` を選んでいるが、その選択原理が本文にまだない。

### 6. 帰結矛盾: 持ち上げ先が非一意

Paper VIII 自身が `{C_α}` の非一意性を認めている。  
だからスカラー `α_E(θ)` が出ても、

`E(θ)=C_{α_E(θ)}`

は一般には**一意の関手**ではなく、固定した濾過に依存する関手族になる。  
ここでも「どの濾過を正本とするか」が足りない。

## P-3 Constructive Critique

`dα/dθ ∝ sin θ` を確定させるために足りないものは、次の 6 点である。

### A. 型宣言

- `α_I` = 忘却強度
- `α_III` = 接続パラメータ
- `α_VIII` = 濾過進行度
- `α_E` = Euler path 上の累積忘却度

を明示的に分ける。  
まず **どの α の微分方程式か** を確定しなければならない。

### B. 経路公理

Euler path `p:[0,π]→M` を導入し、

- `θ` はこの経路上の有向座標
- `θ` は 0 から π まで単調に進む

ことを宣言する。  
これで初めて `d/dθ` が意味を持つ。

### C. 累積公理

次を本文公理として置く必要がある。

> 累積忘却度の増分は、局所非整列密度のみに依存する

これがない限り、局所力学から累積進行度へは移れない。

### D. 積分核の選択原理

なぜ `sin θ` なのかを決める原理が要る。  
最も自然なのは次の 3 条件である。

1. **局所性**: その点の増分はその点の非整列だけで決まる  
2. **一次性**: 局所非整列が 2 倍なら増分も 2 倍  
3. **対称性**: `θ` と `π-θ` は同じ局所強度を持つ

この 3 条件を置けば、`κ` や `sin²θ` ではなく `sin θ` を積分核に選ぶ理由が立ちやすい。

### E. 正規化公理

- `α_E(0)=0`
- `α_E(π)=1`

を課し、比例定数を固定する必要がある。  
これで初めて

`dα_E/dθ = K sin θ`

の `K` が決まり、`K=1/2` に落ちる。

### F. 整合公理

最後に

`α_VIII = α_E`

を Euler path 上で同一視するか、少なくとも

`θ → α_III → η → α_VIII`

と

`θ → α_E`

が可換になることを示さなければならない。  
これがないと、`α_E` は美しい補助変数に留まり、Paper VIII の濾過へ正式には入れない。

## P-4 Alternative

最も強い代替仮説は三つある。

1. `dα/dθ ∝ sin θ` ではなく `dα/dθ ∝ Im(ω)=w·sin θ`
   - 忘却の指数減衰 `e^{-kα}` を rate に残す案
2. `dα/dθ ∝ √κ`
   - 力ではなく局所曲率密度から直接積分する案
3. 強い微分方程式は立たず、弱い橋梁公理
   - 「α は局所非整列の累積単調像である」
   だけが最終形になる案

現時点で最有力なのは 3 だが、1 と 2 を潰さない限り `sin θ` の正準性は確定しない。

## P-5 Integrity Audit

- 誠実性判定:
  - 反駁対象は藁人形化していない。最小モデルの強みを先に認めた。
- 主要弱点:
  - 数式そのものではなく、**前提不足**。
- 最終判定:
  - `sin²(θ/2)` は良い試作だが、まだ theorem candidate であって theorem ではない。

## 結論

`dα/dθ ∝ sin θ` を確定させるのに足りないのは、式変形ではない。  
足りないのは次の 4 面である。

1. **型の分離** — どの `α` か
2. **経路の導入** — どの `θ` 微分か
3. **累積原理** — なぜ局所量から累積量へ行けるか
4. **一意化原理** — なぜ `sin θ` であって別の核ではないか

これが埋まれば、`dα_E/dθ = (1/2)\sin θ` から `α_E(\theta)=\sin^2(\theta/2)` は素直に定理化できる。  
逆に言えば、いま不足しているのは計算ではなく**公理**である。
