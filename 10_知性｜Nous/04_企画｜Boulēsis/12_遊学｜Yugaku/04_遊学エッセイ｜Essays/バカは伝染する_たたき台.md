# バカは伝染する
## — 集団的VFE最小化の悲劇

**ステータス**: 構造たたき台 (2026-03-16)
**シリーズ位置**: Scale (Micro→Macro) への拡張 — 個体の愚かさが集団を侵食する構造

---

## コンセプト

### シリーズ接続

| 接続元 | 接続の射 |
|:--|:--|
| ①構造を見ろ | 個体の VFE → 本作は**集団の VFE** |
| ②馬鹿ほど自信を持つ | 個体の $\pi_0$ 過剰 → 本作は**他者の $\pi_s$ にノイズを注入する** |
| ⑦ビジネス書 | model-free 学習 → 本作は model-free な行動が**集団に伝播する**構造 |
| ⑤LLMの身体 | Context Rot ≅ 集団の劣化。**構造的同型** |

### 核心命題（結論先行で §1 に出すもの）

1. **バカは伝染する** — これは比喩ではない。Nested Markov Blanket の構造上、ある個体の行動は他の個体の感覚入力となるため、行動の質の低さは**物理的に**伝播する
2. **「やる気のある無能」は集団にとって最も有害** — 高い能動推論（行動する）＋低い知覚推論（理解していない）の組み合わせは、集団の sensory channel にノイズを注入し続ける砲台である
3. **「やる気のある無能は排除せよ」は工学的に正しい** — これは差別ではなく信号処理: 集団の SNR (信号対雑音比) を保つためには、ノイズ源を除去するのが最も効率的な最適化である

### 忘却関手としての位置

$$U_{\text{social}}: \mathbf{Collective} \to \mathbf{Individual}$$

**忘却される構造: Nested MB の結合 — 自分の行動が他者の入力になるという再帰構造**。 バカは自分の Markov blanket の「外」を忘れる。自分の行動が他者にとっての環境であるという事実を忘却する。

## 構造案 (8節, ~15頁)

### §1　あなたの周りのバカは偶然ではない

**役割**: 挑発 + 結論先行

- フック: 「あなたの周りに馬鹿が多いと感じるなら、二つの可能性がある。一つは、あなたが馬鹿であること — 馬鹿から見ると世界は馬鹿だらけに見える（②参照）。もう一つは、本稿で数学的に示す構造的理由: **馬鹿は伝染する**。」
- 結論3点を箇条書きで先出し
- 「これは道徳論ではなく、Markov blanket の力学である。」

---

### §2　集団の Markov blanket — Nested MB

**役割**: FEP の集団レベルへの拡張を導入

#### 2.1　個体の MB から集団の MB へ

- §2 of LLM body 論文との接続: MB は個体の境界だけでなく、**集団の境界**も定義する
- Friston (2013): MB は入れ子 (nested) 構造をなす — 細胞→器官→個体→集団→社会

$$\text{MB}_{\text{cell}} \subset \text{MB}_{\text{organ}} \subset \text{MB}_{\text{individual}} \subset \text{MB}_{\text{group}}$$

#### 2.2　集団の VFE

- 集団全体の VFE を定義:

$$F_{\text{group}} = \sum_{i=1}^{N} F_i + \underbrace{\sum_{i \neq j} I(a_i; s_j)}_{\text{相互作用項}}$$

- 第1項: 個体の VFE の和
- 第2項: **ある個体の行動 $a_i$ が他の個体の感覚入力 $s_j$ に与える影響** — これが「伝染」の数学的基盤

#### 2.3　因果の経路: 行動 → 他者の知覚

- 個体 A の行動 $a_A$ → 環境を変化させる → 個体 B の sensory state $s_B$ に影響
- **A の行動の質が低い (VFE_A が高い) → B の入力にノイズが混入 → B の VFE が上昇**
- GIGO の集団版: ゴミ行動はゴミ入力を生む

### §3　やる気のある無能 — 最も危険な個体

**役割**: 核心概念の FEP 的定式化

#### 3.1　四象限分類

| | 低 $\pi_{\text{active}}$ (行動しない) | 高 $\pi_{\text{active}}$ (行動する) |
|:--|:--|:--|
| **高 $\pi_{\text{sensory}}$ (理解している)** | 有能な怠惰 — 害なし | **有能な行動者 — 理想** |
| **低 $\pi_{\text{sensory}}$ (理解していない)** | 無能な怠惰 — **害なし** | **やる気のある無能 — 最大の害** |

- ゼークトの四分類 (旧プロイセン参謀本部の人事原則) との接続:
  - 「怠惰な有能」→ 参謀向き (最小努力で最大効果)
  - 「勤勉な無能」→ **即刻排除** (積極的に害を撒く)

#### 3.2　なぜ「やる気のある無能」が最も有害か

FEP 的分析:

$$\text{ノイズ注入量} = \pi_{\text{active}} \cdot (1 - \text{quality}(q_i))$$

- $\pi_{\text{active}}$ が高い = **たくさん行動する** → 環境への出力が多い → 他者の入力に影響する頻度が高い
- $\text{quality}(q_i)$ が低い = **行動の質が低い** → 出力がノイズである確率が高い

> やる気のある無能は**高出力のノイズ源**である。

- 「やる気のない無能」は害がない — 行動しないのでノイズを出力しない ($\pi_{\text{active}} \approx 0$)
- 「やる気のある有能」は最善: 高出力の信号源

#### 3.3　ゼークトの定理の FEP 的証明

> **定理 (やる気のある無能の排除原理)**:
> 集団の $F_{\text{group}}$ を最小化する最も効率的な操作は、$\pi_{\text{active}}$ が高く $\text{quality}(q_i)$ が低い個体を排除することである。

証明の骨格:
- $\frac{\partial F_{\text{group}}}{\partial N}$ を個体の追加/除去で評価
- ノイズ源の除去は、信号の質を向上させるより少ないコストで $F_{\text{group}}$ を減少させる
- 情報理論: ノイズ除去 > 信号増幅（シャノンの定理の帰結）

### §4　伝染の力学 — なぜバカは増えるのか

**役割**: 正帰還ループの分析

#### 4.1　伝染ループ

1. 無能 A が低品質の行動を出力する
2. B の sensory state にノイズが混入する
3. B の generative model が歪められる (ノイズを信号と誤認)
4. B の行動の質が低下する
5. C の sensory state にノイズが混入する → 1 に戻る

$$\text{noise}(a_A) \xrightarrow{s_B} \Delta q_B \downarrow \xrightarrow{a_B} \text{noise}(a_B) \xrightarrow{s_C} \Delta q_C \downarrow \xrightarrow{a_C} \cdots$$

#### 4.2　⑤ Context Rot との構造的同型

| | Context Rot (⑤) | バカの伝染 |
|:--|:--|:--|
| 系 | LLM セッション | 人間集団 (Nested MB) |
| 劣化原因 | コンテキスト蓄積 → チャネル崩壊 | 無能の行動蓄積 → 信号劣化 |
| 0-cell 崩壊 | 情報源の区別不能 | 有能/無能の行動の区別不能 |
| 1-cell 劣化 | 処理パイプラインの単純化 | 判断基準の画一化 |
| α trivial 化 | 代替経路の消失 | 創造的解法の消失 |
| Γ/Q 崩壊 | 学習と慣性の区別不能 | 革新と慣習の区別不能 |

> **集団の劣化は LLM の Context Rot と同型の力学を持つ。** 薄い集団的 MB は、個体が注入するノイズに対して脆弱である。

#### 4.3　閾値効果 — 臨界点

- 集団内のノイズ源（やる気のある無能）の割合が閾値を超えると、集団全体の VFE が正帰還ループに入る
- 閾値以下: 有能な多数がノイズを吸収 → 系は安定
- 閾値以上: ノイズが信号を上回る → **連鎖崩壊** → 系は劣化の一途
- これはパーコレーション理論と同型: ノイズ源が臨界確率を超えると無限クラスターが発生

### §5　集団の免疫系 — 何がバカの伝染を防ぐか

**役割**: 解決策の FEP 的定式化

#### 5.1　排除 (最も効率的)

- ゼークトの原則: ノイズ源を物理的に除去する
- FEP: $\frac{\partial F_{\text{group}}}{\partial N}$ が最大の個体から除去
- 工学的: チャネルの SNR を保つ最も直接的な方法

#### 5.2　隔離 (MB の分割)

- 全員を排除できない場合: MB の構造を分割する
- 具体: チーム分け、権限制限、情報のファイアウォール
- FEP: 集団 MB の Nested 構造を意図的に深くする → ノイズの伝播範囲を制限

#### 5.3　教育 (最もコストが高い)

- ノイズ源を信号源に変換する: $\text{quality}(q_i) \uparrow$
- FEP: 個体の generative model を更新させる
- 問題: ②で見たように、$\pi_0$ が高い個体は学習が停止している → **教育が最も効きにくい対象こそが最も教育が必要**

---

### §6　反論の取り込み

#### 6.1　「排除は差別ではないか」

- 排除の基準: 能力ではなく**行動の質** × **行動の頻度**
- 有能だが行動しない者は排除不要（害がない）
- 無能だが行動しない者も排除不要（害がない）
- **行動の質が低いのに行動の頻度が高い** → これのみが排除対象
- これは属性による差別ではなく、系のSNR保全

#### 6.2　「集団は多様性を必要とする」

- 正しい。しかし**多様性 ≠ ノイズ**
- 多様性: 異なる関手 $F_1, F_2, ...$ を持つ個体の集合 → 集団の圏の射が豊かになる
- ノイズ: 構造のない出力を高頻度で発する → 射を破壊する
- **多様性は射の豊饒化。ノイズは射の破壊。方向が逆**

#### 6.3　「やる気のないバカの方が有害では？」

- §3.1 の四象限で示した通り、やる気のない者は（有能でも無能でも）**環境への出力が少ない**
- FEP: $\pi_{\text{active}} \approx 0$ → 他者の sensory state への影響はほぼゼロ
- やる気のないバカは**自分が損をするだけ**。やる気のあるバカは**周囲を巻き添えにする**

### §7　圏論的定式化 — 集団の忘却関手

**役割**: 忘却関手 $U_{\text{social}}$ の厳密な定式化

#### 7.1　集団の圏

集団 $G$ を圏として定義:
- **対象**: 集団の各個体 $g_1, g_2, ..., g_n$
- **射**: 個体間の情報伝達経路 $f_{ij}: g_i \to g_j$ （$a_i \to s_j$ の経路）
- **合成**: 情報の伝播: $f_{jk} \circ f_{ij} = f_{ik}$ (A→B→C の伝播)

#### 7.2　ノイズ注入 = 射の劣化

- 有能な個体: $f_{ij}$ が**忠実** (faithful) — 情報が保存される
- やる気のある無能: $f_{ij}$ が**非忠実** (unfaithful) — 情報が失われ、ノイズが付加される
- ノイズ注入の蓄積 = 合成 $f_{jk} \circ f_{ij}$ の忠実性が単調に低下する

#### 7.3　忘却関手 $U_{\text{social}}$

$$U_{\text{social}}: \mathbf{Group} \to \mathbf{Individuals}$$

- $U$ が忘れるもの: 射 $f_{ij}$ の**構造**（他者との結合、自分の行動が他者に影響する再帰構造）
- バカは $U$ を適用した世界で生きている: 自分の行動が他者に影響するという射が見えない
- ①の忘却関手（対象だけ見て射を忘れる）の**社会スケール版**

---

### §8　結語 — バカを置くな

**役割**: 三段構成

#### 非情な事実

集団の VFE 最小化は、個体の VFE 最小化とは構造的に異なる。個体は自分の generative model を改善すればよい。しかし集団は — Nested MB の構造上 — 最も弱いチャネルの品質に全体が引きずられる。鎖の強度は最も弱い輪で決まる。

やる気のある無能は、集団にとって最も弱い輪ではない。**最も弱い輪を量産する工場**である。高出力のノイズ源が一つあれば、周囲の SNR は不可逆的に劣化する。

#### しかし希望

だが、この構造を理解した者には対策がある。排除・隔離・教育 — この優先順位は、信号処理の原則と Nested MB の力学から導かれる。

#### 再帰的閉鎖

最後に一つ、痛い問いを投げておく。この文章を読みながら「あいつのことだ」と思った読者へ: **あなたが周囲に注入しているのは、信号か、ノイズか。** 

もしその問いに即答できるなら — ②を読み直すことを勧める。即答 = 高い $\pi_0$ = 疑いなき確信。最も危険な状態は、自分がノイズ源であることに気づいていないことだ。
