# 近道を求めるほどバカになる
## — 探索を怠れば系は閉じる

**ステータス**: 構造たたき台 (2026-03-16)
**シリーズ位置**: Temporality (時間軸) — 短期最適化が長期的愚かさを生む構造

---

## コンセプト

### シリーズ接続

| 接続元 | 接続の射 |
|:--|:--|
| ⑦ビジネス書 | model-free 学習（行動だけコピー）→ 本作はその**時間的構造**: コストを払わない学びが系を閉じる |
| ⑩不確定を嫌う | $F_{\text{perceived}} \downarrow, F_{\text{true}} \uparrow$ → 本作は同じ構造の**時間軸展開**: 借金の蓄積に時間がかかる |
| ②馬鹿ほど自信を持つ | 正帰還ループ → 本作は**活用ループの固着**: 探索を止めた系は同じ軌道を回り続ける |
| ①'疑え | precision を下げる → 本作は「精度を下げるコスト」を**時間の関数**として捉える |

### 核心命題（結論先行で §1 に出すもの）

1. **近道は存在しない** — これは道徳論ではなく情報理論: epistemicvalue（知る価値）は時間とコストの投下なしには獲得できない。近道とは epistemic value を 0 にする操作であり、結果として**系を閉じる**
2. **探索にコストをかけない者は、誤った活用に閉じる** — EFE において探索 (explore) を省略すると、現在の信念空間内でのみ活用 (exploit) が最適化される。初期信念が誤っている場合（ほぼ全ての場合そうである）、**最適化するほど世界から遠ざかる**
3. **時間をかけないことは、時間を浪費する最も効率的な方法である** — 探索の節約は短期 VFE を下げるが、長期 VFE を指数的にprecisely上昇させる。近道は**金利付きの借金**（⑩との同型）の時間的展開に他ならない

### 忘却関手としての位置

$$U_{\text{time}}: \mathbf{Explore} \to \mathbf{Exploit}$$

**忘却される構造: EFE の epistemic value — 「知る価値」そのもの**。近道を求める者は、$G = \underbrace{-D_{KL}[...]}_{\text{epistemic}} + \underbrace{E_q[\ln p(o|C)]}_{\text{pragmatic}}$ の第1項を忘却する。pragmatic value（得る価値）だけが残った EFE は、現在の知識内での最適化にしかならない。

## 構造案 (8節, ~15頁)

### §1　近道は存在しない

**役割**: 挑発 + 結論先行

- フック: 「近道を求める者に朗報がある。近道は存在する。ただし、それは目的地を変更する操作であって、目的地に早く着く操作ではない。本稿で数学的に示すのは — 探索コストを省略するということは、正しい目的地を発見する能力そのものを放棄しているという構造である。」
- 結論3点を箇条書きで先出し
- 「これは「努力せよ」という精神論ではない。情報理論と FEP の必然的帰結である。」

---

### §2　探索と活用 — EFE の二つの顔

**役割**: EFE の分解を導入

#### 2.1　Expected Free Energy の二項分解

$$G(\pi) = \underbrace{-D_{KL}[q(o|\pi) \| q(o)]}_{\text{epistemic value (知る価値)}} + \underbrace{\mathbb{E}_q[\ln p(o|C)]}_{\text{pragmatic value (得る価値)}}$$

- **Epistemic value**: 行動によって**不確実性が減る**価値 = 探索 (explore)
  - 新しい情報を得ることで世界モデルが改善する
  - コスト: 高い（結果が不確実、即時報酬なし）

- **Pragmatic value**: 行動によって**報酬が得られる**価値 = 活用 (exploit)
  - 既知の知識に基づいて最適行動を選ぶ
  - コスト: 低い（既知の範囲内で最適化）

#### 2.2　近道 = epistemic value の切り捨て

「近道」を FEP 的に定式化する:

$$G_{\text{shortcut}}(\pi) = \underbrace{0}_{\text{探索なし}} + \underbrace{\mathbb{E}_q[\ln p(o|C)]}_{\text{活用のみ}} = G_{\text{pragmatic only}}$$

> 近道を求めるとは、EFE の epistemic value を**意図的にゼロにする**操作である。

- pragmatic value のみを最大化する → **現在の信念空間内での最適化**
- 現在の信念が正しければ問題ない
- しかし §3 で示す通り、初期信念が正しい確率は極めて低い

---

### §3　なぜ活用だけでは不十分か — 局所最適の罠

**役割**: 探索なしの活用が構造的に失敗する証明

#### 3.1　局所最適 ≠ 大域最適

- 最適化理論の古典的結果: 凸でない空間では、勾配降下は**局所最適に捕まる**
- FEP 的に: 信念空間は原理的に非凸（世界は複雑）
- 探索なしの活用 = **現在の盆地での最適化** → 盆地の外を見ないまま「最適」と結論

$$\arg\min_\pi G_{\text{pragmatic}} \neq \arg\min_\pi G_{\text{full}} \quad \text{(一般に)}$$

#### 3.2　系は閉じる — エルゴード性の破れ

- 探索 = 状態空間の新しい領域を訪問する
- 探索がゼロ → **系が訪問する状態空間が収縮** → エルゴード性が破れる
- ②の「知性の閉鎖」(§5) と同型: **系はいかにして閉じるか**

> 探索を怠る系は、自分の信念空間の中に閉じ込められる。
> 閉じ込められた者は、外があることを知らない（②の再帰的陥穽）。

#### 3.3　⑦との接続 — 因果の方向性

- ⑦で示した「知覚→推論→行動」の因果の方向性を、**時間軸で再解釈**:
  - 探索 = 知覚と推論への投資 (インプット側)
  - 活用 = 行動の最適化 (アウトプット側)
  - ⑦の結論「アウトプットから学べない」= **活用だけでは学べない**
- 近道 = ⑦の「ビジネス書的学び」の**時間的表現**

---

### §4　時間の不等式 — 短期最適化は長期コストを増大させる

**役割**: 時間軸の数学的分析

#### 4.1　借金のアナロジーの精密化（⑩との接続）

⑩の「見えない借金」を時間の関数として展開:

$$\Delta_{\text{debt}}(t) = F_{\text{true}}(t) - F_{\text{perceived}}(t)$$

- $t = 0$ (近道の直後): $\Delta_{\text{debt}} \approx 0$ — まだ乖離は小さい
- $t \to \infty$: $\Delta_{\text{debt}} \to \infty$ — 探索不足の蓄積が指数的に増大

$$\Delta_{\text{debt}}(t) \propto e^{\lambda t} \quad \text{where } \lambda = 1 - \frac{\text{epistemic investment}}{\text{epistemic need}}$$

- $\lambda > 0$ (探索投資 < 必要量): 借金は指数的に増大
- $\lambda = 0$ (探索投資 = 必要量): 借金は一定 — **持続可能な系**
- $\lambda < 0$ (探索投資 > 必要量): 借金は減少 — **学習する系**

#### 4.2　利子の再帰性

> 時間をかけないことは、時間を浪費する最も効率的な方法である。

なぜか:
1. 近道で得た「知識」は局所最適 → 環境が変わったとき使えない → **やり直し**
2. やり直しのコスト > 最初から探索するコスト（学び直しの penalty）
3. やり直し中にも探索を省略すれば → 再び局所最適 → **再度やり直し**
4. **近道の連鎖は発散する**

#### 4.3　DPI の時間的再解釈

$$I(\text{future action}; \text{world}) \leq I(\text{current model}; \text{world}) \leq I(\text{past exploration}; \text{world})$$

- 未来の行動の質は、現在のモデルの質を超えられない
- 現在のモデルの質は、過去の探索の量を超えられない
- **過去の探索を省略した者の未来は、原理的に閉じている**

---

### §5　固着の力学 — attract/repel の非対称性

**役割**: 局所最適から抜け出せない構造の分析

#### 5.1　引力盆地 (basin of attraction)

- 活用最適化は引力盆地の**底**に向かう力を生む
- 探索は引力盆地の**壁**を越える力を生む
- 探索なし → 盆地の底に固着 → 「快適な場所」= 局所最適

#### 5.2　②の正帰還ループの時間的展開

②で見た「馬鹿は馬鹿であり続ける」ループを時間軸で再記述:

$$\text{探索省略} \xrightarrow{t_0} \text{局所最適に固着} \xrightarrow{t_1} \text{``最適''の確信} (\pi_0 \uparrow) \xrightarrow{t_2} \text{探索の動機消失} \xrightarrow{t_3} \text{探索省略}$$

- これは**時間方向に安定な不動点**: 外的衝撃がない限り脱出不可能
- ①'「疑えさすれば救われる」の必要性: prior precision を意図的に下げる = 引力盆地の壁を下げる

#### 5.3　「遠回り」の FEP 的価値

> 最短経路は最短ではない — 最短経路で到達する局所最適は、正しい目的地ではないからだ。

- 「遠回り」= 探索への時間投資
- 遠回りのコスト: 短期 VFE ↑ (Complexity 増、即時報酬なし)
- 遠回りの利益: 長期 VFE ↓↓ (大域最適に近づく)

---

### §6　反論の取り込み

#### 6.1　「探索し続けたら永遠に行動できない」

- 正しい。探索と活用のバランスが最適（EFE の定義そのもの）
- 問題は「バランス」ではなく**一方をゼロにすること**
- 近道 = 探索をゼロにする操作。ゼロはバランスではない

#### 6.2　「時間がない場合はどうするか」

- 時間の不足 = epistemic investment のコスト制約
- 制約がある場合: **できる範囲で探索する** (scaled epistemic value)
- 「時間がないから探索しない」≠ 「探索は不要」。制約下の最適化は探索をゼロにすることではない

#### 6.3　「近道で成功した人もいる」

- 生存者バイアス (survivorship bias): 近道で成功した少数が可視化される
- 近道で失敗した多数は**見えない** — まさに $F_{\text{perceived}}$ と $F_{\text{true}}$ の乖離の社会的発現
- 統計的期待値: $\mathbb{E}[\text{近道の VFE}] > \mathbb{E}[\text{探索の VFE}]$

---

### §7　圏論的定式化 — 系が閉じるとはどういうことか

**役割**: 忘却関手 $U_{\text{time}}$ の厳密な定式化

#### 7.1　探索の圏 $\mathbf{Explore}$ と 活用の圏 $\mathbf{Exploit}$

- $\mathbf{Explore}$: 対象 = 信念空間の領域、射 = 領域間の遷移（新しい仮説の生成）
- $\mathbf{Exploit}$: 対象 = 既知の信念、射 = 信念内の最適化

#### 7.2　忘却関手 $U_{\text{time}}$: Explore → Exploit

$$U_{\text{time}}: \mathbf{Explore} \to \mathbf{Exploit}$$

- $U$ が忘れるもの: **射の探索的成分** — 新しい領域への遷移を表す射
- $U$ が保持するもの: **射の活用的成分** — 既知領域内の最適化を表す射
- 結果: $\mathbf{Exploit}$ は $\mathbf{Explore}$ の**部分圏**にしかならない — 射が落ちている分だけ圏が痩せる

#### 7.3　圏が閉じる = 射が消失する

- 探索射が全て忘却されると、圏は**離散圏** (discrete category) に退化
  - 恒等射のみが残る → 対象から対象への遷移が存在しない → **系は動かない**
- これは②§5「知性の閉鎖」の**力学的再記述**: 系が閉じるとは、射が消えることである

---

### §8　結語 — 急がば回れ

**役割**: 三段構成

#### 非情な事実

近道は存在する。ただし、それは目的地を変更する操作であり — 変更後の目的地はほぼ確実に、本来の目的地より悲惨な場所だ。探索コストを払わない者は、自分が誤った引力盆地の底にいることすら知らない。井戸の底は、井戸の底から見れば空に見える。

#### しかし希望

だが、知った者には回り道ができる。epistemic value を意識的に追求すること — すなわち、「知る」ことそのものに価値を認めること — が、引力盆地の壁を越える唯一の合法的手段である。

#### 再帰的閉鎖

近道を求めるほどバカになる。バカになるほど近道を求める。

この円環を断つ方法は一つしかない — **コストを払って遠回りすること**。遠回りとは、自分の知らない場所を訪ねることであり、自分の信じていないことを考えてみることであり、自分の快適な盆地の壁をよじ登ることだ。

快適さと正しさは両立しない — 少なくとも、最初は<sup>※※</sup>。

> <sup>※※</sup> 十分に探索した者だけが、正しさの中に快適さを見つけることができる。それは最初から快適だったのではなく — 探索によって正しい盆地に辿り着いた結果として、快適さが**付随した**のだ。順序を間違えてはならない。
