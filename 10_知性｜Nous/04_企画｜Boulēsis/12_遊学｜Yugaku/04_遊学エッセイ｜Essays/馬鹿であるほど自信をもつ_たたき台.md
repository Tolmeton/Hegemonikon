# 馬鹿であるほど自信をもつ
## — 主観の客観化がもたらすお痛いナルシシズム

**ステータス**: 構造たたき台 (2026-03-16)
**三部作位置**: 第2作（「構造を見ろ」の **WHY NOT** 補完）

## コンセプト

### 三部作の構造

### 核心命題（結論先行で §1 に出すもの）

1. **馬鹿ほど自信を持つ**のは偶然でも性格でもない。未知に無知なほど既知の prior precision が上がるという、知性の構造的必然である
2. **主観の客観化** — 自分の推論（暫定的仮説）を普遍的事実と取り違える — は、Markov blanket の内と外を混同する非忠実関手の作用である
3. しかし悲しいかな、この「愚かさ」は社会適応においては合理的に機能する。**正しさは適応の手段であって目的ではない**

### 副題の姉妹版

本作のテーマは鏡像的に二つの読みを持つ:
- **「馬鹿であるほど自信をもつ」** — 無知が確信を生む構造（順方向）
- **「自信を持つほど馬鹿になる」** — 確信が学習を止める構造（逆方向）

両者は FEP 的に同一のフィードバックループの二つの相であり、本作ではこの **正帰還ループ** を分解する。

### §2　なぜ馬鹿は自信をもつのか — VFE が見えない世界

**役割**: 基礎概念の導入（VFE を第1作から引き継ぎつつ、precision に焦点を移す）

#### 2.1　復習: VFE と「バカ」の定義

- 第1作の定義を簡潔に再導入: バカ = VFE が高い状態
- ただし第1作では **なぜ VFE が高いのか** — 「公理の質」で説明した
- 本作では **なぜ VFE が高いことに気づかないのか** を問う

#### 2.2　未知の未知 — Dunning-Kruger の FEP 的再定式化

- Dunning-Kruger 効果の俗流理解を一旦棚上げし、FEP で再定式化する
- 核心:

> **未知に無知** = 自分の generative model が世界の一部しかカバーしていないことを、モデル自身が検出できない状態

- VFE は「モデルの外の世界」を測定できない。VFE が測定する Accuracy は、**モデルが観測した範囲内での** 予測精度に過ぎない
- したがって: モデルが狭いほど、モデル内での Accuracy は高く見える（観測範囲が限定されているから）
- ここに悲劇がある: **無知であるほど、自分の VFE は低く見える**

数式:

$$F_{\text{perceived}} = -\text{Accuracy}_{\text{observed}} + \text{Complexity}_{\text{model}}$$

$$F_{\text{true}} = -\text{Accuracy}_{\text{全域}} + \text{Complexity}_{\text{model}}$$

$\text{Accuracy}_{\text{observed}} \geq \text{Accuracy}_{\text{全域}}$ が一般に成立する（観測範囲を広げれば予測が外れる事例が増える）。

直感的に言えば: 井の中の蛙は、井戸の中では最も正確な天気予報士である。

#### 2.3　GIGO の再帰 — ゴミでゴミを評価する

- 第1作 §2 の GIGO を拡張: ゴミを評価するのもゴミ（自分のモデル）なので、ゴミがゴミであることに気づけない
- **自己評価の GIGO**: 評価基準自体が貧弱なモデルに依存している → メタ認知の不全

メタファー三連:
- 曲がった定規で自分の曲がり具合を測っても、直線だと結論される
- 色盲の画家は自分の絵が正確だと確信する — 見えないものは存在しない
- 壊れた体重計が示す数値を信じる限り、ダイエットは永遠に成功する

### §3　主観の客観化 — 非忠実関手の病理

**役割**: 圏論的定式化。「主観を客観と取り違える」を射の構造で厳密に述べる

#### 3.1　二つの系: SubjとObj

二つの系を定義する:

- **Subj** (主観系): ある主体の内部モデルが持つ対象と射（信念と推論）
- **Obj** (客観系): 世界が実際に持つ対象と射（事象と因果）

全ての知的主体は、暗黙にある関手 $\Phi: \text{Subj} \to \text{Obj}$ を想定している — 「自分が思っていること」を「世界で起きていること」に写す操作。

#### 3.2　非忠実性 — 射が潰れる

- 「主観の客観化」 = $\Phi$ が **非忠実** (unfaithful) であることに気づかないこと
- 非忠実: Subj における異なる射（異なる推論経路）が Obj で同じ射（同じ事実）に写される → **推論の多様性が失われる**
- より深刻なケース: Subj に存在する射が Obj に**対応物を持たない**。主体は「A→B は自明」と信じているが、**世界にはその因果関係が存在しない** — 捏造された射

定義ブロック:

> **主観の客観化** = 関手 $\Phi: \text{Subj} \to \text{Obj}$ が非忠実であるにもかかわらず、これを忠実だと信じている状態。
> すなわち、「自分の推論」と「世界の事実」の区別がついていない認知状態。

#### 3.3　なぜ馬鹿ほど $\Phi$ を忠実だと信じるのか

- §2.2 に接続: モデルが狭いほど Subj の射が少ない → 射間の区別が少ない → 「射が潰れている」ことを検出する機会が減る
- 極端なケース: Subj に射が1本しかなければ、その1本が Obj のどの射に対応するかは**問うことすらできない**
- これが「未知に無知なほど既知を重視し盲信する」の圏論的表現

直感的に言えば: 語彙が100語しかない人間は、自分の言葉で世界を完全に記述できると確信する — 記述できないものの存在を想像する言葉がないからだ。

### §4　断言の数学 — precision の正帰還ループ

**役割**: 数学的証明の核心部。prior precision → 確信 → 学習停止 → さらなる確信のループを定式化

#### 4.1　Precision とは何か

- Prior precision $\pi_0$ の導入: 信念の「硬さ」を制御するパラメータ
- ベイズ更新の基本:

$$q^*(s) \propto p(o \mid s)^{\pi_{\text{sensory}}} \cdot p_0(s)^{\pi_0}$$

$\pi_0 \to \infty$ のとき $q^*(s) \to p_0(s)$: **いかなるデータも信念を更新しない**

これは定理であり、意見ではない<sup>※</sup>。

> <sup>※</sup> 第1作 §8.3 脚注 ※※ で示した定理の展開。第1作では結語に1行で触れただけだったが、本作ではこれを中心的に扱う。三部作の核心がこの定理に収束している。

#### 4.2　正帰還ループ — 断言が断言を生む

核心の構造: 以下の4ステップが**正帰還ループ** (positive feedback loop) を形成する。

1. **未知に無知** → 観測範囲が狭い → $\text{Accuracy}_{\text{observed}}$ が高く見える (§2.2)
2. **見かけの成功** → 「自分のモデルは正しい」→ $\pi_0$ がさらに上昇
3. **先鋭化** → $\pi_0$ ↑ → 新たなデータが信念を更新できない → 学習停止
4. **固着** → 学習が止まるので未知は未知のまま → 1 に戻る

$$\text{ignorance} \xrightarrow{\text{narrow obs.}} \text{high perceived Acc.} \xrightarrow{\pi_0 \uparrow} \text{learning stops} \xrightarrow{\text{no update}} \text{ignorance}$$

> **「馬鹿であるほど自信をもつ」** = このループが安定的に回転している状態。
> **「自信を持つほど馬鹿になる」** = 同じループを逆方向から記述したもの。

直感的に言えば: 井の中の蛙は、蛙であるがゆえに井戸の外を知らず、知らないがゆえに井戸を世界と信じ、信じるがゆえに井戸から出ず、出ないがゆえに蛙であり続ける。

#### 4.3　断言 = $\pi_0 \to \infty$ の発話的表現

- 「絶対に〜だ」「間違いなく〜である」— これらの言語形式は、話者の prior precision が高いことの直接的な表出
- したがって: **断言を頻繁に発する者は、$\pi_0$ が高い** ≈ **学習が止まっている**

> 愚か者ほど断言を求め、断言を発する。

これは侮蔑ではなく、precision の機能の記述である。

### §5　知性の閉鎖 — 系はいかにして閉じるか

**役割**: Creator メモの「知性と言う系は閉じる」を FEP/エルゴード性で厳密に展開

#### 5.1　「主観的合理 = 客観的事実」が閉鎖の瞬間

- 知性を一つの**力学系**として捉える
- 開いた系: 環境との情報交換がある → 学習が継続する → VFE が減少する
- 閉じた系: 環境との情報交換が停止する → prior のみで動作する → VFE は変化しない

> 知性という系は、推論という主観的仮説を絶対的な客観的事実と同一視することで閉じる。

FEP 的に: $\Phi$ を忠実と信じた瞬間、prediction error を情報ではなくノイズとして処理する → **系のエルゴード性が破れる** → 状態空間の一部しか訪問しなくなる

#### 5.2　「たかだか一主体の中での合理」

- 圏論における**局所的性質**と**大域的性質**の区別に対応する
- Subj の中で整合的 (locally consistent) であることは、Obj との整合性 (globally consistent) を保証しない
- 忠実関手の欠如 = 局所整合性が大域整合性を偽装している状態

> たかだか「一主体の中での合理」であることを忘れた時に、その合理は不合理になる。

メタファー三連:
- チェス盤の一角だけを見て「ここには敵がいない」と確信する将棋指しは、盤全体を見ていない
- 地図の一部が正確であることは、地図全体の正確さを保証しない
- 自分の家の中が片付いていることは、街全体が清潔であることを意味しない

### §6　愚かさの生存価 — なぜ盲信は社会で機能するか

**役割**: 反論の取り込み + Creator の洞察「悲しいかな自身の主観に酔える愚かさがこの社会においては役に立つ」の定式化

#### 6.1　EFE の分解 — epistemic value vs pragmatic value

Expected Free Energy (EFE) の分解:

$$G = \underbrace{-D_{KL}[q(o \mid \pi) \| q(o)]}_{\text{epistemic value (知る価値)}} + \underbrace{\mathbb{E}_{q}[\ln p(o \mid C)]}_{\text{pragmatic value (得る価値)}}$$

- **Epistemic value**: 行動によって不確実性が減る価値 = 探索 (explore)
- **Pragmatic value**: 行動によって報酬（社会的成功等）が得られる価値 = 活用 (exploit)

#### 6.2　盲信は pragmatic に最適化された戦略

- 断言は社会的に以下の利点を持つ:
  - **意思決定コストの削減**: 迷わない → 行動が速い → 機会を逃さない
  - **集団への信号**: 確信を表明する者はリーダーとして認識される（確信バイアス）
  - **認知負荷の低減**: 疑いを放棄すれば認知リソースが行動に回せる

- FEP 的に: pragmatic value を最大化し epistemic value を**放棄した**状態

> 正しさはあくまで世界への適応という目的を果たすための一つの「手段」でしかない。

#### 6.3　しかし — 長期 VFE は増大する

- 短期: 盲信は低コスト高リターン（pragmatic value で正当化）
- 長期: 環境が変化したとき、更新できない prior は致命的になる
- ダイナソーの比喩: 恐竜は6500万年間 pragmatic に最適だった。隕石が落ちるまでは

直感的に言えば: 盲信は晴天限定の傘なし戦略である。雨が降らないうちは合理的に見える。

### §7　馬鹿の自認 — 最適 precision の探索

**役割**: Creator の結語「馬鹿を自認しつつ適度に主観を客観化する人間がもっとも賢いのかもしれない」を FEP 的に展開

#### 7.1　最適 precision は 0 でも ∞ でもない

- $\pi_0 = \infty$: 盲信。学習停止。短期合理的、長期致命的
- $\pi_0 = 0$: 虚無。全否定。精度ゼロの信念は判断基盤を失う
- **最適 $\pi_0^*$** が存在する: prior を十分に柔らかく保ちつつ、行動の指針となる程度に固い

> **「馬鹿を自認する」** = 自分の $\pi_0$ が最適でない可能性を常に保持する
> = メタ認知的な precision の制御

- 「疑え」（第3作）への橋渡し: 最適 precision をどう較正するかは第3作のテーマ

#### 7.2　「絶対」は危険信号

姉妹版テーマの回収:

> 強い信念ほど、疑う際に得られうるサプライズが大きい。

これは情報理論的に自明:

$$\mathcal{S}(o) = -\ln p(o)$$

**確率が低い（＝信念と矛盾する）観測ほど、サプライズ量が大きい。**

すなわち: 「絶対にAである」と信じている者がA以外を発見したとき、そのサプライズ量は理論的に**最大**となる。ブレイクスルーの温床は「最も疑われていない前提」の直下にある。

#### 7.3　不合理が合理をもたらす

> 主体の中での合理を逸脱する不合理が、新たなる合理をもたらす。

FEP 的に: prior を一時的に(explore方向に)逸脱させることが、**epistemic value の獲得** に対応する。これは能動推論 (active inference) における exploratory behavior そのものである。

「馬鹿になる勇気」= $\pi_0$ を意図的に下げる — 「疑え」のテーマ本体

### §8　結語 — 適度に馬鹿であれ

**役割**: 三段構成（スタイルガイド §7 準拠）

#### 非情な事実

現実は非情である。正帰還ループは**安定的**に回転する — 馬鹿は馬鹿であり続けるための自己強化メカニズムを内蔵している。無知であること自体は罪ではないが、無知に気づくメカニズムが無知そのものに阻害されるという、この再帰的な陥穽は容易には抜け出せない。

#### しかし希望

だが、この正帰還ループを断ち切る方法がないわけではない。

筆者の完全な主観ではあるが、最も賢いのは**馬鹿を自認する者**である。馬鹿を自認するとは、自分の $\Phi$ が非忠実であること — 自分の主観が客観と一致しない可能性を — 構造的に受け入れることだ。

> 馬鹿を自認しつつ適度に主観を客観化する人間が、もっとも賢いのかもしれない。

#### 再帰的閉鎖

しかし、馬鹿であることを毛嫌いして適応を損ねることもまた馬鹿である。正しさは適応の手段であり、目的ではない。盲信は短期的に機能し、懐疑は長期的に機能する。最適はその間のどこかにある — そして、その「どこか」を探し続けること自体が、VFE を最小化し続ける営みに他ならない。

愚か者ほど断言を求め、断言を発する。
適度に馬鹿であれ — だが、適度に疑え<sup>※※</sup>。

> <sup>※※</sup> 「どう疑うか」— これが第3作「疑え、さすれば救われる」の主題である。
