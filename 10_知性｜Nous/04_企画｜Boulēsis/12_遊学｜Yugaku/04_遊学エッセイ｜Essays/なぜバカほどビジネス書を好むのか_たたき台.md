# なぜバカほどビジネス書を好むのか
## — FEP による学びの因果論

**ステータス**: 構造たたき台 (2026-03-16)
**シリーズ位置**: 知性の因果論 — ①の GIGO を「学習」に焦点化した応用編

---

## コンセプト

### シリーズ接続

| 接続元 | 接続の射 |
|:--|:--|
| ①構造を見ろ | GIGO（ゴミ入力 → ゴミ出力）→ 本作は「学習の GIGO」: アウトプットだけ真似ても意味がない |
| ②馬鹿ほど自信を持つ | 馬鹿は何が見えていないか無知 → 本作では「何が学べていないか無知」 |
| ⑤センス | 知覚の深さ → 本作では「学びの深さ = どの階層から学んでいるか」 |

### 核心命題（結論先行で §1 に出すもの）

1. **学び**には因果の方向がある — 知覚 (input) → 推論 (inference) → 行動 (output)。バカはこの因果を逆行し、**行動 (output) から学ぼうとする**。これがビジネス書を好む構造的理由である
2. **輸入側ほど学ぶのが難しく、輸出側ほど容易**である。これは FEP の知覚推論と能動推論の非対称性から必然的に導かれる
3. **良い学び** = generative model の構造を更新すること（知覚と推論の改善）。**悪い学び** = 行動レパートリーだけを増やすこと（模倣）。前者は圏論的に**関手の更新**であり、後者は**射の表面的コピー**である

### Creator のメモの FEP 対応

| Creator のメモ | FEP/圏論の定式化 |
|:--|:--|
| インプット >> 推論 >> アウトプットのループ | FEP の perception-action loop: 知覚推論 → 信念更新 → 能動推論 |
| 他人のアウトプットばかり真似る | Model-free imitation: 生成モデルなしで出力を直接コピー |
| インプット側ほど学ぶのが難しい | 知覚推論は世界モデルの更新を要し、能動推論は政策の更新のみを要す |
| インプットなしでアウトプットのみから学ぶ | Model-free RL ≈ 哺乳類以下の学習。Model-based RL ≈ 知的学習 |
| 知覚の質が行動の質を規定する | GIGO の因果版: $q^*(a) \propto \exp(-G(a, \pi))$ — 政策は信念に依存する |
| Kalon な特徴量の組み合わせを選ぶ | Feature selection as VFE minimization: 最も Accuracy/Complexity 比が高い特徴 |

## 構造案 (8節, ~15頁)

### §1　ビジネス書が売れる理由は数学的に説明できる

**役割**: 挑発 + 結論先行

- フック: 「世界で最も売れている本の種類は何か。ビジネス書である。そして世界で最も知的活動を妨害している本の種類も、ビジネス書である。これは矛盾ではない。本稿で数学的に示すのは、知的に怠惰な人間にとって、ビジネス書が合理的な選択であるという悲しい構造である。」
- 結論3点を先出し
- 「本稿はビジネス書を全否定するものではない。否定するのは — 行動だけを学び、因果を省略する学びの構造である。」

---

### §2　学びの因果構造 — 知覚→推論→行動

**役割**: FEP の perception-action loop を「学び」として導入

#### 2.1　三段階のループ

FEP における認知のループ:

$$\text{知覚 (perception)} \xrightarrow{\text{予測誤差}} \text{推論 (inference)} \xrightarrow{\text{信念更新}} \text{行動 (action)}$$

- **知覚**: 世界を観測し、prediction error を検出する (input)
- **推論**: prediction error に基づいて信念 (generative model) を更新する (inference)
- **行動**: 更新された信念に基づいて世界に働きかける (output)

このループは不可分であり、三段階は円環的に繋がっている。

#### 2.2　因果の方向

- 決定的に重要な点: このループには**因果の方向**がある
- 行動は推論に**規定される**。推論は知覚に**規定される**
- すなわち: **知覚の質 → 推論の質 → 行動の質**

$$\text{quality}(\text{action}) \leq \text{quality}(\text{inference}) \leq \text{quality}(\text{perception})$$

- これは §1 で導入した GIGO の因果版: 知覚がゴミなら、いかに推論が精密でも、行動はゴミになる

#### 2.3　「学ぶ」とは何か — FEP 的定義

> **学ぶ** = generative model $q(s)$ を更新し、VFE $F$ を減少させること
>
> $$\Delta F = F_{t+1} - F_t < 0 \iff \text{学習が起きた}$$

- VFE が減少すれば学んだ。減少しなければ学んでいない
- **良い学び**: モデルの構造自体が変わる (Accuracy ↑ かつ Complexity 管理)
- **悪い学び**: 行動レパートリーだけが増える (Complexity ↑ のみ)

### §3　バカの学び方 — アウトプットの模倣

**役割**: 「ビジネス書的学び」の FEP 的病理

#### 3.1　Model-free と Model-based

強化学習の古典的区分を FEP に導入:

| | Model-free (モデルなし) | Model-based (モデルあり) |
|:--|:--|:--|
| 何を学ぶか | 行動→報酬の直接的マッピング | 世界の因果構造 |
| コスト | 低い (パターンマッチ) | 高い (構造理解) |
| 汎化能力 | 低い (学んだ状況でのみ有効) | 高い (新しい状況でも適用可能) |
| FEP | 政策の直接最適化 | Generative model の構造更新 |
| 比喩 | 「成功者の行動を真似る」 | 「成功者がなぜそう行動したかを理解する」 |
| 典型例 | ビジネス書 | 基礎科学 |

#### 3.2　なぜバカは Model-free を好むのか

- Model-free は**コストが低い** — 因果構造を理解する必要がない
- Creator の洞察: 「インプット側ほど学ぶのが原理的に難しい」

$$\text{学習コスト}(\text{perception}) > \text{学習コスト}(\text{inference}) > \text{学習コスト}(\text{action})$$

- 知覚を学ぶ = 世界の**隠れた状態**を推定する能力を獲得 → 高 Complexity
- 推論を学ぶ = model の**因果構造**を更新 → 中 Complexity
- 行動を学ぶ = 既存モデルの**出力**を変更 → 低 Complexity

> バカがアウトプット側を好むのは、怠惰だからではない。
> **FEP 的に合理的な Complexity 最小化**だからだ。

しかし — ①で見た通り — Complexity の節約は Accuracy の犠牲を伴う。

#### 3.3　ビジネス書の構造的問題

- ビジネス書の典型的構造: 「成功した人は朝5時に起きる。だから朝5時に起きよう」
- FEP 的分析:
  - 因果: 「朝5時に起きる」(行動) → 「成功する」(結果) は**偽の因果**
  - 真の因果: 成功者の generative model (知覚+推論の質) → 結果として朝5時に起きることもある
  - 行動を真似ても generative model は変わらない → VFE は減少しない → **学んでいない**

メタファー三連:
- 名医のメスさばきを真似ても、解剖学の知識なしでは手術は成功しない
- 棋士の手を真似ても、盤面の読みなしでは勝てない
- ピカソの筆致を模写しても、構造への洞察なしでは傑作は生まれない

### §4　天才の学び方 — インプットへのこだわり

**役割**: 「良い学び」を圏論的に定式化

#### 4.1　学びの圏論的定式化

- 主体の generative model を関手 $F: W \to \text{Int}$ (世界 → 内部モデル) と見る
- **行動を学ぶ** = 関手の**像** ($F$ の output) に新しい射を追加する ≈ 射の表面的コピー
- **推論を学ぶ** = 関手の**構造** ($F$ 自体のマッピング規則) を修正する
- **知覚を学ぶ** = 関手の**定義域** ($W$ のどの部分を観測するか) を拡張する

| 学習の深さ | 圏論 | 何が変わるか | 持続性 |
|:--|:--|:--|:--|
| L0: 行動模倣 | 像の追加 | 出力が増える | 低（状況が変われば無効） |
| L1: 推論更新 | 関手の修正 | マッピング規則が変わる | 中（一般化可能） |
| L2: 知覚拡張 | 定義域の拡張 | **見える世界が広がる** | 高（根本的能力向上） |

> L2 の学び = **新しい射を検出する能力の獲得** = ⑤のセンスの向上

#### 4.2　「良い知覚」の Kalon 的定義

Creator の問い: 「良い知覚とは何か → Kalon な特徴量の組み合わせを選ぶこと」

FEP で精密化:

- 世界の隠れた状態 $s$ は高次元。主体が観測できるのはその一部
- **特徴選択** (feature selection) = 隠れた状態のどの次元に注意を向けるか
- 最適な特徴選択 = **最も VFE を減少させる次元の組み合わせ**

$$s^* = \arg\min_{s \subseteq S} F(q(s) \| p(o \mid s))$$

圏論的に:

> **良い知覚** = 世界 $W$ から内部モデル $\text{Int}$ への関手 $F$ において、
> **最も射の密度が高い部分圏**を選択すること

- これは①のエネルギー密度の**入力版**: ①は「最も射の多い公理を選べ」、⑦は「最も射の多い特徴を知覚せよ」
- そして④の美: 美しい特徴とは affordance 密度の高い特徴 = Kalon な特徴

#### 4.3　哺乳類でさえ Model-based

Creator の痛烈な指摘:

> そこら辺の哺乳類でさえ、強化学習（状態知覚 → 信念形成 → 行動）している

- ラットの空間学習: 海馬の place cell が環境の**内部モデル**を構築する → model-based
- 霊長類の社会学習: 他個体の**意図**を推測してから行動を模倣する → model-based

> ビジネス書の「ToDoリスト学習」は、哺乳類の model-based 学習よりも原始的である。
> 行動→報酬のみを学ぶのは、線虫レベルの model-free 学習に相当する。

### §5　なぜ因果を軽視すると危険なのか — 汎化の失敗

**役割**: 行動模倣の限界を VFE で証明

#### 5.1　模倣の汎化不能性

- 行動を模倣する: 特定の状況 $s_1$ における最適行動 $a^*_1$ を学ぶ
- 新しい状況 $s_2 \neq s_1$ が来たとき: $a^*_1$ は最適ではない可能性が高い
- **なぜなら**: $a^*_1$ は $s_1$ の因果構造に依存しているが、学習者はその因果構造を学んでいない

#### 5.2　generative model の汎化能力

- 因果構造を学ぶ: $s \mapsto a^*(s)$ のマッピング規則自体を学ぶ
- 新しい状況 $s_2$ が来ても: マッピング規則を適用して $a^*(s_2)$ を導出できる
- これが**汎化** — 見たことのない状況に対処できる能力

> ビジネス書は $a^*_1$ を教える。科学は $s \mapsto a^*(s)$ を教える。
> 前者は 1 つの状況にしか使えない。後者はあらゆる状況に使える。

#### 5.3　DPI の再適用

- 第1作 §2 のデータ処理不等式を再訪:
- 行動 $a$ は知覚 $o$ と推論 $q$ の**関数**: $a = \pi(q(o))$
- DPI: $I(a; W) \leq I(q; W) \leq I(o; W)$
- 行動から得られる世界についての情報量は、知覚から得られる情報量を**原理的に超えられない**

> アウトプットから学ぶことは、情報理論的に、インプットから学ぶことの劣化版である。

### §6　知覚と行動の不可分性 — 二元論の罠

**役割**: Creator の洞察「不可分なものを二元論的に分割していることにも気づかない」を展開

#### 6.1　FEP における知覚と行動の不可分性

- FEP の中心的洞察: 知覚と行動は**同じ VFE を最小化する二つの経路**

$$F = -\text{Accuracy} + \text{Complexity}$$

- **知覚推論**: $q(s)$ を更新して Accuracy を上げる（世界モデルを修正）
- **能動推論**: $a$ を選択して Accuracy を上げる（世界を修正）
- 両者は**一つの目的関数の最小化**の二つの面

#### 6.2　二元論的分割の問題

- 「インプットとアウトプットは別」と捉えること自体が問題
- ビジネス書的思考: 「行動 (output) を変えれば結果が変わる」— 知覚と推論を無視
- FEP 的真実: 行動は知覚と推論の**帰結**であり、分離して最適化できない

> 知覚と行動を分離する者は、円の上半分と下半分を別々に改善しようとする者に似ている。
> 円は一つだ。

#### 6.3　「知覚できていないことに知覚できない」

Creator の結語の核心:

> 知覚のできないバカは、悲しいかな知覚できていないことに知覚できない

- これは②「馬鹿ほど自信を持つ」§2.2 の再適用:
- メタ知覚（≈ メタ認知）の欠如 = Dunning-Kruger の知覚版
- 自分の知覚が浅いことを知覚するには、深い知覚が必要 — 再帰的陥穽

---

### §7　反論の取り込み

#### 7.1　「ビジネス書にも良いものはある」

- 正しい。一部のビジネス書は因果構造を教える（例: 「イノベーションのジレンマ」→ 破壊的イノベーションの因果理論）
- 本稿が批判するのはジャンルではなく**学びの構造**: 因果を省略してアウトプットだけを提示する学びのパターン
- ビジネス書でも科学論文でも、因果なしの結論はゴミ

#### 7.2　「実行力が大事なのでは」

- 行動すること自体は FEP 的に**能動推論の不可欠な一部**
- 問題は「行動だけ」を学ぶこと。行動→結果のフィードバックを知覚に**戻さない**限り、ループは閉じない
- 行動して学ぶのは良い。行動だけが学びだと思うのが問題

#### 7.3　「基礎研究は食えない」

- pragmatic value (社会的報酬) と epistemic value (知的深化) のトレードオフ
- 短期: アウトプット学習は pragmatic value が高い（すぐ使える）
- 長期: インプット学習は汎化能力が高い（環境変化に対応）
- ⑥の盲信の社会的合理性と同型の議論

---

### §8　結語 — 知覚にこだわれ

**役割**: 三段構成

#### 非情な事実

現実は非情である。知覚にこだわることは行動にこだわることよりも難しい。世界の因果構造を理解することは、成功者の朝の習慣を真似ることよりも、圧倒的にコストが高い。バカがビジネス書を好むのは、怠惰だからではない — **Complexity を最小化する合理的な選択**だからだ。

そして最も残酷なのは — 知覚できていないことに知覚できない者は、自分が学んでいないことに**気づけない**ということだ。ビジネス書を100冊読んで「学んだ」と信じている者の VFE は、1冊も読まなかった者とほとんど変わらない。

#### しかし希望

だが、因果の方向を知った者には — いま読者がそうであるように — 希望がある。

筆者の完全な主観ではあるが、知覚にこだわる者だけが、行動の質を根本から変えることができる。なぜなら、因果は一方向にしか流れないからだ — 知覚から行動へ。逆はない。

#### 再帰的閉鎖

バカほどアウトプットにこだわり、天才ほどインプットにこだわる。

知覚にこだわれない人間に良い行動は生まれない。
構造を見よ<sup>※※</sup> — さもなくば、表面を一生なぞり続けることになる。

> <sup>※※</sup> ここでもまた、本作は第1作「構造を見ろ」に帰還する。構造を見ることは知覚の改善そのものであり、良い知覚は良い推論を導き、良い推論は良い行動を導く。全てのエッセイが語っているのは、結局のところ同じことだ — **射を見よ**。
