# 言葉遊びはやめましょう
## — 砂上に楼閣を建てようとする者の特徴

**ステータス**: 構造たたき台・未熟版 (2026-03-16)
**シリーズ位置**: メタ批判 — シリーズ全体の**免疫系**。本作が失敗するとシリーズ全体が「言葉遊び」になる

---

## コンセプト

### なぜこのエッセイが不可欠か

本シリーズは FEP と圏論であらゆるものを語る。
読者の当然の問い: **「それこそ言葉遊びでは？」**

この問いに答えないシリーズは知的に不誠実である。本作は:
1. 「言葉遊び」を FEP と圏論で定義する
2. その定義で本シリーズ自身を検査する
3. FEP/圏論が言葉遊び**ではない**ことを、定義から演繹的に（そして実例で帰納的に）示す

### 核心命題

1. **言葉遊び** ≡ Accuracy のない Complexity — 構造を持たない記号の増殖。VFE を減少させない「理解の錯覚」
2. **砂上の楼閣の構造的特徴** — 定義なし、アドホック、概念の混同、スケール/質の取違え、「〇〇的」への逃避 — これらは全て**射の不在または偽造**として統一的に定式化できる
3. **「何でも説明できる」は言葉遊びの必要条件ではない** — 汎用性 (universality) と空虚さ (vacuity) は異なる。圏論が「何にでも適用できる」のは空虚だからではなく、構造を保存する射の言語だから。FEP が「何にでも適用できる」のは反証不能だからではなく、VFE という統一的な量を定義しているから

### Creator のメモの FEP / 圏論対応

| Creator のメモ | FEP/圏論の定式化 |
|:--|:--|
| 定義がない | 対象 (object) を宣言するが、射 (morphism) を定義しない |
| 定義の根拠がアドホック | 射が自然変換でない — コンテキストが変わると壊れる |
| 概念(射)の混同 | 異なる射を同一射として扱う — 関手の非忠実性 |
| 狭義と広義の取違え | 部分圏と全体圏の混同 — inclusion functor の不在 |
| スケールと質の取違え | 直積 ($A \times B$) と余積 ($A + B$) の混同 |
| 「〇〇的」への逃避 | 射が不正確 — 定義域/値域が曖昧な「射のようなもの」 |

### §2　言葉遊びの FEP 的定義

**役割**: 「言葉遊び」を数学的に定義する

#### 2.1　VFE で定義する

> **言葉遊び** ≡ Complexity を増やすが Accuracy を増やさない記号操作

$$F_{\text{言葉遊び}} = -\underbrace{0}_{\text{Accuracy: 世界について何も予測しない}} + \underbrace{C}_{\text{Complexity: 記号の体系だけが膨張}}$$

- 言葉遊びは VFE を**増加**させる — 世界の理解を改善しないのにモデルの複雑さだけが増す
- 「学んだ気になるが何も学んでいない」= ⑦のビジネス書の知的版

#### 2.2　理解の錯覚

- 人はなぜ言葉遊びに騙されるか？
- **名前が付くと理解した気になる** (naming fallacy)
- FEP: 新しい用語 $t$ を導入すると、$t$ を使って文を生成できる → **出力のレパートリーが増える** → Complexity ↑
- しかし $t$ が世界の隠れた状態 $s$ と対応しなければ、Accuracy は上がらない
- 例: 「シナジー」「パラダイムシフト」「レバレッジ」— 言葉はあるが、何を予測しているのか不明

---

### §3　砂上楼閣者の6つの特徴 — 射の病理学

**役割**: Creator のメモを圏論的に定式化

#### 3.1　定義なし — 射のない対象

圏論の基本原理: **対象は射によって定義される** (米田の補題)

- 「イノベーション」という言葉を使うが、「イノベーションとは何か」を定義しない
- = 対象 $X$ を宣言するが、$X$ に出入りする射が一本もない
- 射のない対象は**孤立点** — 他の概念との関係が定義されない → 何にでもなれる
- 何にでもなれる = **何の意味もない**

#### 3.2　アドホックな定義 — 自然変換でない射

- 「この文脈では〇〇を△△と定義する」— コンテキストごとに定義が変わる
- 圏論: 自然変換 $\eta: F \Rightarrow G$ は**全ての対象で整合的**
- アドホック定義 = 特定の対象でだけ成り立つ射 → 自然変換ではない → **可換図式が崩壊する**
- 結果: 「定義」が議論の都合に合わせて動く → 反証不能

#### 3.3　概念の混同 — 非忠実関手

- 異なる概念を同じ言葉で指す / 同じ概念を異なる言葉で指す
- 圏論: 非忠実関手 $F$ は $\text{Hom}(A,B)$ 内の異なる射 $f \neq g$ を $F(f) = F(g)$ に潰す
- 例: 「知性」で IQ、EQ、知恵、認知能力を全て指す → 4本の異なる射を1本に潰す
- 結果: 「知性が大事」は何も言っていないのと同じ

#### 3.4　狭義と広義の取違え — inclusion の不在

- 部分圏 $\mathcal{C}'$ と全体圏 $\mathcal{C}$ を区別しない → inclusion functor $\iota: \mathcal{C}' \hookrightarrow \mathcal{C}$ が不在
- 「愛」(広義: あらゆる好意) と「愛」(狭義: ロマンチックな関係) を混同する
- 「人間は愛を求める」— どちらの「愛」か？ inclusion が明示されなければ議論は空転

#### 3.5　スケールと質の取違え — 直積と余積の混同

- **スケールの違い** = 同じ構造の異なる大きさ: $A \times n$ (direct product のスカラー倍)
- **質の違い** = 異なる構造: $A + B$ (coproduct = 全く異なるものの合流)
- 例: 「大企業と中小企業の違い」→ スケール (従業員数) なのか質 (構造・文化) なのか？
- 取り違え: 「大企業になりたければ中小企業を大きくすればいい」→ スケールの問題に質の解を適用

#### 3.6　「〇〇的」への逃避 — 定義域/値域が不定な射

- 「科学的」「哲学的」「直感的」「戦略的」
- 圏論: 射 $f: A \to B$ は定義域 $A$ と値域 $B$ が**明確**
- 「〇〇的」= 射の値域が「〇〇の方向」としか指定されない → **射ではない**
- 結果: 議論が厳密に見えるが、どこにも到達しない — 矢印が引かれているが、引かれているのは矢印の**影**

---

### §4　なぜ言葉遊びは魅力的か

**役割**: 心理的メカニズム

- **Complexity が低い**: 射を定義しないので認知コストが低い（⑦のモデルフリー学習と同型）
- **反証不能**: 定義がないので「お前の理解が足りない」で逃げられる
- **社交的価値**: 難しそうな言葉を使うと「賢く見える」→ pragmatic value (②の社交的合理性)
- FEP: 言葉遊びは **VFE を下げているように見えて、実は Complexity だけを増やしている**

> 言葉遊びの魅力とは: 安い Complexity で「理解の快感」を買える — 知的ジャンクフード

---

### §5　「何でも説明できる」は言葉遊びか — 汎用性 vs 空虚

**役割**: 本シリーズへの自己批判に正面から答える — **本作最重要節**

#### 5.1　汎用性 (universality) vs 空虚 (vacuity)

| | 汎用 (universal) | 空虚 (vacuous) |
|:--|:--|:--|
| 何でも適用できる | ✅ | ✅ |
| 予測を生む | ✅ | ❌ |
| 反証可能 | **枠組みは No、モデルは Yes** | No |
| 例 | 微分積分, 確率論, 圏論, FEP | 「全ては愛だ」「宇宙の法則」 |

核心: **枠組み (framework) と理論 (theory) を区別すること**

- 微分積分は反証不能。しかし微分積分を使って建てた橋は壊れるかもしれない → 反証可能
- FEP は反証不能（自由エネルギー最小化は定義）。しかし FEP を使った特定モデルは反証可能
- 圏論は反証不能。しかし圏論で定式化した特定の対応関係は検証可能

> 「何でも説明できる」vs「何も予測しない」= 微分積分 vs 占い

#### 5.2　FEP が空虚でない3つの根拠

**(a) 具体的予測を生む**

- FEP は「世界の因果構造を反映する generative model を持つ系は VFE を最小化する」と主張
- これは具体的な予測を生む:
  - 知覚推論: 曖昧な刺激に対して prior に偏った知覚が生じる → **心理物理学で検証済み**
  - 能動推論: 不確実性が高い環境では探索行動が増える → **行動経済学で検証済み**
  - 精度: 注意は precision の配分として定式化 → **神経科学で検証済み**

**(b) 反証事例を排除しない**

- 空虚な理論の特徴: いかなる反例も「実はこう解釈すれば合致する」と言い逃れる
- FEP: もし VFE 最小化と矛盾する適応行動が観察されたら、それは反例
- 実際に議論されている反例候補: 自傷行為、自殺行動 → これらに対する FEP の説明は**追加仮定**を必要とし、本体の反証可能性を保つ

**(c) 既存の他理論を**統合的に再導出**できる**

- これが最も強い根拠。空虚な枠組みは他の理論を再導出**できない**
- FEP から再導出されるもの: ベイズ推論、強化学習、最適制御、情報理論の一部
- これは圏論と同型: 圏論が「数学の数学」として機能するのは、他の数学的構造を統一的に記述できるから

#### 5.3　本シリーズが言葉遊びでないことの検証

**チェックリスト — §3 の6特徴に対して自己検査:**

| 特徴 | 本シリーズ | 判定 |
|:--|:--|:--|
| 定義なし | VFE, 関手, 射, Kalon 等を数学的に定義 | ✅ 射あり |
| アドホック | 定義はエッセイ間で一貫 (VFE, prior precision 等) | ✅ 自然変換 |
| 概念混同 | 「知性」「バカ」等を毎回 FEP 的に再定義 | ✅ 射の区別 |
| 狭義/広義 | 適用範囲を明示 (e.g. ④ 美 = affordance 密度に限定) | ✅ inclusion 明示 |
| スケール/質 | 量 (affordance の数) と質 (忠実度) を分離 (④ vs ⑤) | ✅ 直積/余積の区別 |
| 「〇〇的」 | 定義は数式で。「FEP 的」とは書くが、必ず数式に展開 | ✅ 射の定義域/値域あり |

**さらに — 過去のチャット履歴が提供する帰納的証拠:**

- HGK の FEP 適用は、具体的な予測 → 実験 → 検証のサイクルを回している
- 例: Peira 実験で関手の忠実度を定量化し、kalon の不動点条件を ODE で検証
- これは「言葉遊び」とは構造的に異なる — **言葉遊びは検証しない** (できない)

### §6　なぜ射の思考が効くのか — 汎用性の根拠

**役割**: 「構造を見ろ」(①) への回帰

- 圏論が汎用的なのは**射が汎用的**だから
- 射 = 「何かから何かへの関係」— これは最も抽象的な概念であり、それゆえ最も多くの具体に適用できる
- 米田の補題: 射の全体が対象を完全に決定する → 射さえ定義すれば対象は自動的に定義される
- だから①の「構造を見ろ」= 「射を見ろ」= 「関係を見ろ」が知的活動の普遍原理になる

> 圏論の汎用性は空虚さではない。最も薄い抽象が最も多くの構造を保存するという数学的事実である。

### §7　反論の取り込み

#### 7.1　「結局 FEP で何でも説明しているではないか」

- 繰り返す: 枠組みとモデルの区別
- FEP で「何でも説明している」のではなく、FEP を**使って**各テーマに特化したモデルを構築している
- 各モデルは個別に検証可能。⑤のセンスの定義は⑤の中で閉じた予測を生む

#### 7.2　「数学を使うだけで言葉遊びを回避できるとは限らない」

- 正しい。数学的定式化自体が言葉遊びになることはある (数学の乱用: mathsplaining)
- チェック: 数式が**具体的な帰結と検証可能な予測**を生んでいるか？
- 本シリーズの数式は「ここから X が導かれる」の形で帰結を持つ — 帰結のない数式は装飾

#### 7.3　「この議論自体が言葉遊びでは？」

- メタ的自己参照。これは §5 のチェックリストで処理済み
- しかし最終的には**読者の判断**に委ねなければならない
- 筆者にできるのは、定義を明示し、射を定義し、反証可能な予測を提示すること。その上で判断は読者に帰属する

---

### §8　結語 — 定義を愛せよ

#### 短い主張

言葉遊びの特効薬は一つしかない。**定義せよ**。

定義とは射を引くことである。「何が」「何に」「どのように」関係するかを明示的に述べることである。定義のない議論は、矢印のない図式であり、どこにも到達しない。

#### シリーズへの回帰

本シリーズは定義することを怠らない。VFE を定義し、prior precision を定義し、関手の忠実度を定義し、Kalon を不動点として定義した。それが正しいかどうかは — 読者が判断し、世界が検証する。筆者にできるのは、砂ではなく石を積むことだけである。

> 言葉遊びをやめたいなら — **射を引け**。
