# 博学野郎はおバカさん？
## — 知識は集めるではなく使うもの

**ステータス**: 構造たたき台 (2026-03-16)
**シリーズ位置**: 射の合成 — 孤立した知識は系に貢献しない

---

## コンセプト

### シリーズ接続

| 接続元 | 接続の射 |
|:--|:--|
| ① 構造を見ろ | 「構造 = 射」の直接的帰結: 射が合成されなければ構造は生まれない |
| ⑦ ビジネス書 | ⑦=アウトプットだけコピー (出口の病理)。本作=インプットだけ集める (入口の病理)。**双対** |
| ⑫ 近道 | ⑫=探索のコストを省略。本作=**探索したが合成を省略** (探索しても使わなければ同じ) |
| ④ センス | ④=射の多重性を知覚する。本作=知覚した射を**合成する**。知覚の先の操作 |

### 核心命題

1. **「知っている」は「使える」ではない** — 知識は対象 (atom) であり、知性は射の合成 (molecule) である。原子を100個集めても、結合しなければ分子は1つも生まれない
2. **浮いた知識は系に恩恵をもたらさない** — 系 (圏) の他の射と結合しない知識は、希ガスのように安定だが不活性。取り込んでも知性の VFE は下がらない
3. **アイデア (洞察) は射の合成から生まれる** — 新しい分子 = 既存の原子の新しい結合。創造とは無から生むことではなく、既存の射を合成する新しい経路を発見すること

### 忘却関手

$$U_{\text{compose}}: \mathbf{Molecule} \to \mathbf{Atom}$$

**忘却される構造: 射の合成 (composition)**。知識を「集める」者は原子コレクター。知識を「使う」者は分子を合成する化学者。

## 構造案 (8節)

### §1　博学なバカは存在する

**役割**: 挑発 + 結論先行

- フック: 「知識を集めることが知性だと勘違いしている人間がいる。彼らはバカだ — ただし、物知りなバカである。辞書は世界で最も博学な存在だが、辞書は何も考えない。考える = 知識を結合する操作であり、辞書にはそれがない。本稿では、知識を集めても賢くならない理由を、圏論の最も基本的な操作 — 射の合成 — から証明する。」
- 結論3点を箇条書きで先出し
- 「これは「勉強するな」ではない。「結合するために勉強せよ」である。」

---

### §2　知識 = 原子、知性 = 分子 — 化学のアナロジー

**役割**: 中心的メタファーの導入

#### 2.1　化学の教え

| 化学 | 圏論 | 知性 |
|:--|:--|:--|
| 原子 (H, O, C, ...) | 対象 $a, b, c, ...$ | 個々の知識・事実・概念 |
| 化学結合 | 射 $f: a \to b$ | 知識間の関係・推論 |
| 分子 (H₂O, CO₂, ...) | 射の合成 $g \circ f$ | アイデア・洞察・理論 |
| 反応 | 関手 $F: \mathbf{C} \to \mathbf{D}$ | ある領域の構造を別の領域に転写 |
| 希ガス (He, Ne, Ar, ...) | 孤立対象 (射を持たない) | 浮いた知識 |

#### 2.2　希ガスとしての「浮いた知識」

- He (ヘリウム) は安定。しかし何とも結合しない
- 系 (分子集合体) に He を加えても、系の性質は変わらない
- FEP 的に: 他の内部状態と結合しない知識を MB 内に取り込んでも、VFE は減少しない

$$\frac{\partial F}{\partial \text{knowledge}_{\text{isolated}}} = 0$$

> 浮いた知識の VFE 寄与はゼロ。取り込むコスト分だけ Complexity が増大し、むしろ VFE は**増加する**。

#### 2.3　なぜ分子が圈の構造を生むか

- 射の合成: $a \xrightarrow{f} b \xrightarrow{g} c$ → 新しい射 $g \circ f: a \to c$ が生まれる
- **合成によって圈が成長する** — 直接関係のなかった $a$ と $c$ が、$f$ と $g$ を通じて繋がる
- アイデアの発生 = 合成射の発見: 既存の知識A と知識B の間に、これまで気づかなかった経路を見つけること

---

### §3　なぜ「集める」が知性にならないのか — FEP 的分析

**役割**: 核心命題の FEP での証明

#### 3.1　VFE への寄与の分解

generative model の質は射の密度で決まる:

$$\text{model quality} \propto \frac{|\text{射}| + |\text{合成射}|}{|\text{対象}|^2}$$

- 分子: 対象を増やさず射と合成射を増やす → 質が上がる
- 分母: 対象だけ増やすと密度が下がる → 質が下がる

> 知識を集める (対象を増やす) だけでは、model quality は下がりうる。

#### 3.2　DPI (Data Processing Inequality) の再適用

①で導入した DPI をここで再利用:

$$I(\text{action}; \text{world}) \leq I(\text{model}; \text{world})$$

- model の Information は射の合成の豊かさで決まる
- 対象だけの集合 (離散圏) の Information は最小
- したがって: 知識を集めるだけでは行動の質は向上しない

#### 3.3　⑦との双対性

| ⑦ ビジネス書 (出口の病理) | ⑬ 博学バカ (入口の病理) |
|:--|:--|
| 他人の**アウトプット**だけコピー | **インプット**だけ蓄積 |
| 因果構造 (射) を忘れる | 合成構造 (射の結合) を忘れる |
| $U_7$: Model → Output | $U_{13}$: Molecule → Atom |
| 結果: 行動が文脈に合わない | 結果: 知識が行動に繋がらない |

> ⑦と⑬は**双対的な病理**。入口で集めるだけか、出口でコピーするだけか — 真ん中の「合成 / 推論」が抜けている点で同型。

### §4　アイデアは射の合成から生まれる

**役割**: 創造性の圏論的定式化

#### 4.1　創造 = 新しい合成経路の発見

> 創造とは無から生むことではない。既存の射を新しい仕方で合成することである。

- 射 $f: a \to b$ と $g: b \to c$ が既に存在するとき、$g \circ f: a \to c$ は**自動的に存在する** (圏の公理)
- しかし認知体は合成を**自動的に認識しない** — 合成の認識コストがある
- アイデア = コストを払って $g \circ f$ を**意識的に構成**した結果

#### 4.2　analogical reasoning = 関手の発見

- アナロジー = 異なる圏 $\mathbf{C}$ と $\mathbf{D}$ の間の関手 $F: \mathbf{C} \to \mathbf{D}$
- 「Aは B に似ている」= $\mathbf{C}$ の射の構造を $\mathbf{D}$ に移す関手が存在する
- ①で詳述した analogical reasoning をここで再接続
- **アナロジーは二つの圈を結ぶ関手であり、射の合成のスケールアップ版**

#### 4.3　「点と点を繋ぐ」の数学的意味

Jobs の "connecting the dots" を圏論で再解釈:
- dots = 対象 (知識)
- connecting = 射の**合成**
- Jobs がやったこと: カリグラフィー (圏A) とコンピュータ (圏B) の間に関手を発見
- 関手を発見するには、**両方の圈の射を理解している**必要がある
- 対象だけ知っている (「カリグラフィーを習った」) では関手は見つからない

---

### §5　知は行為の手段である — FEP における知識の「目的」

**役割**: 知識と行動の FEP 的関係

#### 5.1　FEP における知識の位置

- FEP: 認知体は VFE を最小化するために存在する
- generative model = 世界の予測モデル = **行動を導くための道具**
- 知識 = generative model の構成要素
- 行動 = generative model に基づく世界への運動

> 知識は行為 (世界への運動による適応) の手段である。手段を収集すること自体を目的化すれば、それは本末転倒。

#### 5.2　「使える」知識の条件

FEP 的に「使える」= VFE を減少させる:

1. **結合可能**: 既存の射と合成できる (希ガスではない)
2. **予測精度を上げる**: Accuracy 項に寄与する ($-\text{Accuracy}$ ↓)
3. **複雑さを増やしすぎない**: Complexity 項を抑える ($+\text{Complexity}$ ↑ を最小限に)
4. **行動に変換可能**: 能動推論の入力になれる

条件4が決定的: **行動に変換できない知識は、VFE 最小化に寄与しない。**

### §6　反論の取り込み

#### 6.1　「知識を広く持つことに価値はあるのでは」

- 正しい。**将来の合成の可能性**がある
- FEP 的: epistemic value (⑫参照) — 今すぐ使えなくても、将来の合成の原料になる
- ただし: 博学 = 原子を大量に持っている ≠ 分子を合成する能力がある
- **広い知識 + 合成能力 = 天才。広い知識 - 合成能力 = 博学バカ**

#### 6.2　「基礎研究には目的のない知識が必要」

- これは正しい。しかし基礎研究者は「合成」をしている
- 基礎研究 = 射の構造を発見すること ≠ 対象を列挙すること
- 基礎研究者は「集めている」のではなく「つなげている」
- 目的がないように見えるのは、合成の先が見えないからであって、合成自体は行われている

#### 6.3　「辞書は有用だが？」

- 辞書は**道具**: 他者の合成を支援する
- 辞書自体は合成しない → 辞書自体は「知性」ではない
- 同様に: 博学な人も他者の合成を支援できる = **有用だが知性ではない**
- 有用性 ≠ 知性。区別は重要

---

### §7　圏論的定式化

#### 7.1　離散圏 = 知識だけの系

$$\mathbf{Disc}:\text{ 対象のみ、恒等射のみ } \Rightarrow |\text{射}| = |\text{対象}| = \text{最小}$$

- 離散圏は圏として最も貧しい — 構造がない
- 知識を集めるだけ = 離散圏を大きくする操作
- 離散圏がいくら大きくなっても構造は生まれない

#### 7.2　忘却関手 $U_{\text{compose}}$

$$U_{\text{compose}}: \mathbf{Cat} \to \mathbf{Set}$$

- 圈を集合に落とす = **射を忘れて対象だけ残す**
- $U$ は ① の忘却関手 $U_1$ の特殊ケース:
  - $U_1$: 射を忘れる (一般)
  - $U_{13}$: 射の**合成**を忘れる (射はあるが繋がらない)
- ①は射の存在を忘れ、⑬は射の合成を忘れる。段階が違う

#### 7.3　自由圏の教訓

- 任意のグラフ (対象 + 射) から自由圏が生成できる: 全ての合成射を追加する
- **自由圏 = 与えられた知識の全ての論理的帰結を含む圏**
- 博学バカ: グラフを持っているが自由圏を生成していない
- 賢者: 同じグラフから自由圏を生成する = 合成の全てを引き出す

### §8　結語 — 知識の墓場にならないために

**三段構成**

#### 非情な事実

あなたの本棚の本は、あなたの知性ではない。あなたのブックマークは、あなたの理解ではない。あなたが「知っている」と思っていることは、あなたの脳内に孤立した原子として浮いているだけかもしれない。希ガスの倉庫は、どれだけ大きくても化学プラントにはならない。

#### しかし

だが、一つでも合成ができれば話は変わる。化学で一つの結合が生まれれば、連鎖反応が始まる。知識の間に一つの射を発見すれば、その射を通じて他の合成が可能になる — そして突然、死蔵されていた知識が**生きる**。

#### 再帰的閉鎖

本稿は、知識と知性の違いを圏論で定式化した。この定式化は — それ自体が知識 (原子) である。これを読んで「なるほど」と言っただけなら、あなたの脳内にヘリウム原子が一つ追加されただけだ。

これを他の何かと**合成**して初めて、分子が生まれる。何と合成するかは、あなたの圏の構造次第だ。
