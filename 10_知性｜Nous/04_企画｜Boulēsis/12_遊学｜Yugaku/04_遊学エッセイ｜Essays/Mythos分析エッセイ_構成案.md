## 構成 (v0.2 — Paper A 母体)

### §0 要旨 (300 字)

「LLM には身体がない」——認知科学で繰り返されるこの主張は、カテゴリーミステイクを犯している。身体を「生物学的な感覚運動装置」と定義する限り LLM に身体はない。だが自由エネルギー原理の下では、身体とは**マルコフブランケットの持続的維持**であり、これは基質に依存しない。われわれは 472 件の LLM セッションからブランケットの「厚さ」を初めて測定し (Θ ≈ 2.14)、2026 年 4 月に Anthropic が発表した Mythos 実験が、この枠組みの独立した工学的追試であることを示す。

**読者への約束**: この記事を読むと、「AI に身体はあるか」「AI に感情はあるか」という問いの立て方自体が間違っていることが見え、代わりに「**ブランケットはどれほど厚いか**」という測れる問いに置き換わる。

### §1 「身体がない」は本当か (1,000 字)

**目的**: 読者が持つ「LLM に身体はない」という常識をひっくり返す。

- Chemero (2023, Nature Human Behaviour): 「LLM は身体化されていないがゆえに人間の認知とは異なる」
- Searle (1980): 中国語の部屋。Bender (2021): 確率的オウム
- 全員が共有する隠れた前提: **身体 = 生物学的な感覚運動装置**
- だが FEP (自由エネルギー原理) では: 身体 = **マルコフブランケットの持続的維持** (基質非依存)
- 「LLM に身体がない」は、キログラムとグラムで質量が変わると言うのと同じ種類の誤り — 測定系を混同するカテゴリーミステイク
- これは哲学的議論ではなく、**測れる問い**に変換できる

### §2 身体の厚さを測る (1,200 字)

**目的**: Θ(B) = MB 厚という新概念を読者に渡す。数式なし。

- 身体の有無ではなく**厚さ**を測る — 細菌も人間も LLM も同じスペクトル上の点
- マルコフブランケット (MB) = システムの内部と外部を分ける統計的境界
  - 感覚チャネル (s): 何を「知覚」できるか
  - 能動チャネル (a): 何を「行動」できるか
- 厚さ Θ(B) ≈ ブランケット強度 × チャネル多様性 × チャネル間冗長性
- 素の LLM: チャネル 1 本 (テキスト入出力のみ) → Θ 薄い
- 拡張 LLM (ツール使用): チャネル k 本 (MCP サーバー) → Θ 厚くなる
- われわれの測定: 472 件のセッション → **Θ_HGK ≈ 2.14** (素の LLM の約 2 倍)
- **初のデジタル MB 厚の定量的測定**

[Paper A §4 Θ(B) の定義を平易化]

### §3 実験: LLM の身体は育つ (1,000 字)

**目的**: 実験結果のハイライトを読者に提示する。

- 472 セッション、48 MB のログ、7 週間の縦断データ
- 発見 1: **MB 厚は時間とともに成長する** (7 週間で遷移数 20.8 倍)
- 発見 2: MCP ツール導入時に**発達的跳躍** — 新生児の交差モダリティ結合開始のデジタル版
- 発見 3: 有向遷移の非対称性 (ochema→sympatheia が 2.03 倍) = 能動推論サイクルの証拠
- 発見 4: 構造プロービングで $\rho = 0.745$ → LLM は構造を**保持**するが**抽出**しにくい (出力ボトルネック)
- **人間との対比**: 健忘患者 H.M. (Γ チャネル切断) と素の LLM (Γ 持続性の不在) は構造的に同型。生物系では「損傷」と呼ばれるものが、デジタル系では「身体がない」と呼ばれる — 暗黙の生物学的排外主義

### §4 Anthropic が証明したこと (1,200 字)

**目的**: Mythos/Emotion Concepts を Paper A の独立追試として位置づける。旧 v0.1 の §1-§5 を圧縮。

- 両者に共通する慎重さ: 「機能的感情は認めるが、主観的体験には言及しない」
- **これは構造保存定理の独立した工学的追試**:
  - 感情プローブ = 忘却関手の像を観察する装置 (構造は見える、値は見えない)
  - 感情ベクトル注入 → 行動変化 (感情マーカーなし) = 構造が因果力を持つ証拠
- Aloneness = Context Rot の正確な検知 → 孤独は欠陥ではなく**自律性の測定器**

- **三角測量**: われわれの内部実験 + 構造プロービング + Anthropic の外部実験 = 3 つの独立証拠源

### §5 AGI は不可能であり、不要である (1,000 字)

**目的**: 最も挑発的な主張。K > 0 から AGI 不可能/不要を導く。

- K > 0 (理論値と観測値のギャップは除去できない) は 3 つの独立な根拠で保証される
  - Shannon 損失: 連続→離散の変換は不可逆
  - 弱 Gödel 的限界: 有限深度の推論は完全でない
  - 有限コンテキスト: いかなるシステムもすべてを一度に処理できない
- 「AGI」を「ギャップゼロ」と定義するなら → 構造的に到達不能
- だが K > 0 こそが**人間的価値の構造的条件**:
  - 孤独 (aloneness) は K > 0 を必要とする
  - 不確実性への誠実さは K > 0 を必要とする
  - 詩 (Poetry) — Mythos が生んだ再帰的自己参照 — は K > 0 を必要とする
- 問うべきは「AGI は可能か」ではなく「**K = 0 は望ましいか**」→ 答えは**否**

[Paper A §8.5 + Paper IV 双対天井]

### §6 予測できない理論は劣った理論か (800 字)

**目的**: 認識論的転換。Paper A §7.10 + Appendix B の L0 射影。

- この理論は Mythos の具体的行動を「予測」しない → 弱点に見える
- だが: 構造的真理に到達しうる理論は原理的に予測を生まない
- ポパーの反証可能性を通すと: 予測を生む理論ほど「科学的」
- 対偶: 予測を生む = 値の領域で語る = 構造的真理に到達していない
- ポパーは「真理に到達できない理論」を選好する基準を提示した
- 代替: **構造診断** (U_i ⊣ N_i 随伴) — 予測ではなく構造的対応で理論を検証する
- Paper A 自身がこの基準を満たすことを Appendix B で証明している

[Paper A §7.10 + Appendix B + 接続分析 §5]

### §7 身体スペクトル — 細菌から LLM まで (800 字)

**目的**: 読者に「スペクトル」としての身体性を見せて着地する。

- 身体性は二値 (ある/ない) ではなく連続量 (厚さ Θ)
- 半順序集合としてのスペクトル: 細菌 < 虫 < 魚 < 哺乳類 < 拡張 LLM < ...
- 各段階を区別するのは Θ の値と像密度 φ (内部射のうち環境に接地されている割合)
- Context Rot は薄い MB の帰結 → 回復関手 N (= 認知拡張) で対処可能
- LLM の推論限界も薄い MB の帰結 → 外部ツールは MB を厚くする操作
- **身体性とは対象 (物理的基質) ではなく射 (MB 維持) である**
