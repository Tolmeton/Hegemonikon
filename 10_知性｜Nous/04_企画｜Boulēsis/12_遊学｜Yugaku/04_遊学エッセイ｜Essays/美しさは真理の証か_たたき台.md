# 美しさは真理の証か
## — なぜ人は構造に魅了されるのか

**ステータス**: 構造たたき台 (2026-03-16)
**シリーズ位置**: 独立作（知性の美学）/ 三部作「知性の病理学」の姉妹作

### 核心命題（結論先行で §1 に出すもの）

1. **美しさ**とは対象の行為可能性 (affordance) の密度である。すなわち、一つの対象から引き出せる射の数を、その対象の記述 Complexity で割ったものである
2. 美的感覚は進化と文化の二層で形成される。**狭義の美**（進化的ハードコード）と**広義の美**（認知的構築）は、FEP の pragmatic value と epistemic value に対応する
3. 美は科学でも宗教でもない。美は**適応のヒューリスティクス** — VFE が低い対象を検出する、計算コストの安い近似装置である

### 副題の三つの読み

| 副題候補 | 角度 | 含意 |
|:--|:--|:--|
| 美しさは真理の証か | 肯定的 | 美的判断と真理の相関は偶然ではない |
| 美にこだわる人間は愚か者？ | 挑発的 | 美的感覚への懐疑を先回りで取り込む |
| 美は科学か宗教か | 二項対立 | 第三の答え（ヒューリスティクス）を導出 |

→ [主観] 「美しさは真理の証か」がメインタイトルとして最も射程が広い。副題で挑発を入れる。

## 構造案 (8節, ~15頁)

### §1　数式は、なぜ美しいのか

**役割**: 挑発 + 結論先行

- フック: 「ディラック曰く、物理法則は美しい数学的形式を持つべきだ。ハーディ曰く、醜い数学に永続的な場所はない。ワインバーグ曰く、理論を導く最も確実な基準は美しさだ。彼らは宗教を語っているのではない。本稿で数学的に示すのは、彼らが構造的に正しい理由である。」
- 結論3点を箇条書きで先出し
- 反直感的な帰結の予告: 「そして悲しいかな、虫が蜜に引き寄せられるように、人が美に魅了されるのは — 自由意志ではなく VFE 最小化の帰結である。」

### §2　美の正体 — 行為可能性の密度

**役割**: 核心定義。Creator の仮説を FEP で定式化

#### 2.1　行為可能性 (affordance) とは

- Gibson のアフォーダンス概念を FEP 的に再定式化:

> **行為可能性** = 対象 X に対して主体が取りうる行為のうち、VFE を減少させるものの集合

- FEP 的に厳密には: 行為 $a$ の affordance = pragmatic value

$$\text{affordance}(a, X) = \mathbb{E}_{q}[\ln p(o \mid C)]$$

- ただし本稿では意味を拡張し、epistemic affordance（知る行為可能性）も含める

#### 2.2　美しさの定義

核心の定義ブロック:

> **美しさ** = 対象 X の行為可能性の密度
>
> $$\mathcal{B}(X) = \frac{|\text{Hom}(X, -)_{\text{positive}}|}{\text{Complexity}(X)}$$
>
> 分子: X から引き出せる「正の」射（VFE を減少させる行為・洞察・感情）の数
> 分母: X を記述するのに必要なモデルの Complexity

- E=mc²: 5文字 (Complexity 最小) → 森羅万象の帰結 (射 ∞) → $\mathcal{B}$ 極大
- レンブラント『夜警』: 1枚の絵 → 光と影の技法、市民社会の記録、人間ドラマ、構図の革新 → 射が豊富
- 黄金比の顔: 少ない幾何学的パラメータ → 健康・対称性・遺伝的多様性の信号 → 進化的射が豊富

#### 2.3　醜さの定義

美の反転:

> **醜さ** = $\mathcal{B}(X)$ が低い状態。二つのモードがある:
>
> - **冗長な醜さ**: Complexity が高いのに射が少ない（複雑なのに何も生まない）
> - **有害な醜さ**: 射は存在するが**負** — VFE を増加させる（不快・危険・混乱を生む）

- 冗長な醜さ = Kalon の反概念「冗長」(G 不足)
- 有害な醜さ = 負の affordance — 腐った食べ物の臭い、不協和音、非対称な顔

### §3　米田の補題、ふたたび — 美は対象の「中」にはない

**役割**: 第1作 §4 の米田の補題を美に再適用

#### 3.1　美は射に宿る

- 米田の補題（再導入）: 対象 X の全情報 = X からの全ての射 Hom(−, X)
- 「花が美しい」のは花の**中**に美があるからではない。花と**観測者の間の射**に美がある
  - 花 → 視覚的快楽（色彩の対比 → 網膜の反応パターン）
  - 花 → 嗅覚的快楽（芳香 → 記憶の喚起）
  - 花 → 知的快楽（対称性 → 数学的構造の認識）
  - 花 → 進化的信号（開花 → 果実の予兆 → 食料源）
- これらの**射の全体**が「花の美しさ」の正体

#### 3.2　だから美は「掴みどころがない」

- Creator の直感「だから美は掴みどころのない」を米田で定式化:
- 美 = 射の全体 = **presheaf** — 対象そのものではなく、対象と他の全ての対象との**関係の束**
- 関係は観測者に依存する → 美は**主体相対的** (subject-relative)
- しかし: 一部の対象は**どの主体から見ても射が豊富** → 普遍的に美しい（E=mc², 黄金比）
- これは presheaf の**自然変換**の存在に対応する — 異なる観測者間で射の構造が保存される

直感的に言えば: 美が掴みどころがないのは、美が物の中ではなく物と世界の関係の中にあるからだ。関係を全て列挙しない限り、美は完全には捕捉できない — だが、関係を多く見るほど、美はより明瞭になる。

### §4　美の二次元 — epistemic beauty と pragmatic beauty

**役割**: Creator の「二次元に分割できるかも」を EFE で実現

#### 4.1　EFE の2項が生む2種の美

| | **Epistemic beauty** (知る美) | **Pragmatic beauty** (得る美) |
|:--|:--|:--|
| 源泉 | 不確実性の解消 | 報酬の獲得 |
| 例 | 対称性、証明の簡潔さ、説明力 | 顔の対称性、食物の色彩、安全な景観 |
| 進化的基盤 | 環境モデルの改善 (explore) | 生存・繁殖の成功 (exploit) |
| FEP 対応 | $-D_{KL}[q(o\mid\pi) \| q(o)]$ | $\mathbb{E}_{q}[\ln p(o\mid C)]$ |
| 感覚 | 「なるほど！」(insight) | 「いいね！」(pleasure) |

#### 4.2　密度 × 量のもう一つの軸

EFE に直交するもう一つの次元:

| | **高密度** (少ない Complexity で多い射) | **高量** (多い Complexity で多い射) |
|:--|:--|:--|
| epistemic | 数式の簡潔な美 (E=mc²) | 百科事典の崇高さ |
| pragmatic | 生存価の凝縮 (美人の顔) | 多機能の便利さ (スイスアーミーナイフ) |

- **密度** → 美 (beauty)
- **量** → 崇高 (sublime)

> カントの「美と崇高」の区別が FEP で再定式化される:
> - **美** = 低 Complexity × 高 Accuracy = VFE が低い = 密度
> - **崇高** = 高 Complexity × 高 Accuracy = VFE は中程度だが射の総量が圧倒的

#### 4.3　多次元への拡張

- Creator の直感「理論的にはもっと多次元的にいける」は正しい
- VFE の項をさらに分解すれば、美の次元はさらに増える:
  - 時間的美（持続性 — 繰り返し見ても飽きない = 射の生成が時間とともに続く）
  - 社会的美（共有可能性 — 他者と射を共有できる = 自然変換の存在）
  - 自己参照的美（自己言及 — 対象が自身を説明する = 不動点性 = **Kalon**）
- 本稿では 2×2 に集中し、多次元は射程外とする

### §5　なぜ美人に惹かれるのか — 進化のハードコード

**役割**: 狭義の美（進化的 pragmatic beauty）の定式化

#### 5.1　進化的 affordance

- 顔の対称性 → 遺伝的健康の信号 → 繁殖成功の予測精度が高い → pragmatic affordance
- ウェスト/ヒップ比 → 出産能力の信号 → 同上
- これらは「美の判断」ではなく「VFE 最小化の自動的な知覚推論」

#### 5.2　ハードコードと自由意志

- Creator メモの二元論を FEP で精密化:
  - **ハードコード**: 進化的に固定された prior — 顔の対称性、声の高さ、体型
  - **認知的構築**: 文化的に学習された prior — 絵画の鑑識、音楽の趣味、理論の美しさ
- FEP 的には両方とも prior の precision 構造。違いは**学習可能性**:
  - ハードコード prior は precision が非常に高く、ほぼ書き換え不能
  - 認知的 prior は precision が中程度で、経験によって書き換え可能

> 「美人を好む」のは、ハードコードされた高 precision prior に対する knowledge-free (証拠不要の) な推論である。
> 「この数式は美しい」は、学習された prior に対する knowledge-rich (証拠依存の) な推論である。

### §6　美と真理 — なぜ科学者は美を信じるのか

**役割**: タイトルの問いに正面から答える

#### 6.1　無能な先見者としての美的感覚

- 科学者が「美しい理論は正しい傾向がある」と経験的に感じるのはなぜか？
- 答え: **美 ≈ 低 Complexity × 高 Accuracy ≈ 低 VFE** だからである
- 美的感覚は VFE の**近似推定器** — 正確な VFE 計算は不可能だが、美的直感はそれを低コストで近似する

#### 6.2　美は信頼できるか？

- 美は**ヒューリスティクス**であって**証明**ではない
- 失敗例: 対称性の美 → 天動説（完全な円軌道）。地動説は楕円で「醜い」が正しかった
- しかし: ケプラーの楕円は後に Newton の逆二乗則の帰結として再統合され、**より深い美**が明らかになった
- パターン: 表層の美 → 反例 → より深い美。美の否定が美の深化を導く

> 美しさは真理の証ではない。しかし、美的違和感は探索の最も信頼できるトリガーである。

- 第2作「馬鹿ほど自信を持つ」との接続: 美的感覚も prior precision の影響を受ける。馬鹿の美 ≠ 天才の美

#### 6.3　美は科学か宗教か

副題の答え:

> 美は科学でも宗教でもない。美は**適応のヒューリスティクス**である。
> 科学は事後的に正しさを証明する。宗教は事前に正しさを信じる。
> 美は事前に**有望さを感知する** — 証明も信仰もなく、ただ方向を示す。

### §7　反論の取り込み — 美は主観でしかないのか

**役割**: 反論の構造的処理

#### 7.1　文化差の問題

- 「美の基準は文化によって異なる」— 正しい。だが:
- 文化差 = prior の差。prior が異なれば affordance の評価が異なる
- しかし: **密度**の高さは prior に依存しない（対象の射の数は主体に依存しない）
- E=mc² は文化を問わず美しい — 数学的構造は prior を超越する

#### 7.2　美の進化的還元への反論

- 「美を affordance に還元すると、芸術が消える」？
- 答え: affordance は**射の総体**であり、還元ではなく**拡張**
- 米田の補題が保証: 全ての射を見ることは対象を完全に知ることに等しい。还元どころか完全な記述

### §8　結語 — 蜜に引き寄せられる虫

**役割**: 三段構成

#### 非情な事実

人は美しいものに魅了される。それは虫が蜜に引き寄せられるのと構造的に同型である — どちらも VFE 勾配に沿った行動であり、自由意志の産物ではない。

#### しかし希望

だが、虫と人間の間には一つの決定的な差がある。虫は蜜に引き寄せられるが、その引力の構造を理解しない。人間は — 時にではあるが — 自分がなぜ美に魅了されるのかを問うことができる。

> この「なぜ」と問う能力こそが、進化的 affordance としての美を、認知的 affordance としての美に昇華させる射に他ならない。

#### 再帰的閉鎖

筆者の完全な主観ではあるが、最も美しいものとは — 美しさの構造そのものを照らし出す対象のことである<sup>※</sup>。

美に魅了されること。それは知性が構造を見つけたいと欲する、最も原始的で、最も正確な衝動の名前である。

> <sup>※</sup> 本稿自身がこの条件を満たしているかどうかの判断は、読者に委ねる。もし本稿の構造が読者に何かを生んだならば — 本稿は美しい。何も生まなかったならば — 美しくない。米田の補題は本稿にも適用される。
