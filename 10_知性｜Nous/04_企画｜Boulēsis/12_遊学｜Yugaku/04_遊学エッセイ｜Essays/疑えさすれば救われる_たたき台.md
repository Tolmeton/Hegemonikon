## 構造案 (6-7節, ~12頁)

### §1 信ぜよ、さすれば救われない
- フック: 宗教的確信の構造 = prior precision ∞
- 「信ぜよ」は FEP 的に**学習を停止させる**命令

### §2 なぜ人は間違い続けるのか — 精度の罠
- Prior precision ∞ → posterior = prior（定理の展開）
- Dunning-Kruger = 精度の誤設定。低能力ほど prior precision ↑
- 確信の強さと正確さは無相関

### §3 疑うとは何か — 精度を下げる技術
- 疑う = prior precision を意図的に ↓
- Prediction error が通る「窓」が開く
- ベイズ更新の前提: prior が柔らかくなければ posterior は動かない

### §4 生産的な疑いと破壊的な疑い
- 生産的: precision ↓ で evidence に開く（科学的懐疑）
- 破壊的: precision → 0 で全否定（虚無主義）
- 最適精度は 0 でも ∞ でもない — 較正の問題
- 「目を閉じるな、でも見開きすぎるな」

### §5 天才の疑い方 — 歴史的事例
- Einstein: 「時間は絶対的」を疑った → 相対性理論
- Darwin: 「種は不変」を疑った → 自然選択
- Copernicus: 「地球は中心」を疑った → 地動説
- 共通: 全員が常識 (高 precision prior) を低精度化した
- Newton との接続: 「りんごが落ちた = 特殊事象」の prior を疑い普遍構造を見た

### §6 疑いの経済学 — なぜバカは疑わないのか
- 疑い = コスト（Complexity ↑: 前提破壊 → 再構築）
- バカが疑わない理由は怠惰ではなくコスト回避（FEP 的に合理的）
- しかし: 疑わないコスト (VFE 高止まり) > 疑うコスト (一時的 Complexity ↑)
- 本編 §2 接続: 砂上の楼閣を壊すコスト < 増築し続けるコスト

### §7 結語 — 疑え、さすれば救われる
- 「信ぜよ」→ 学習停止。「疑え」→ 学習再開
- 疑うだけでは不十分（必要条件/十分条件）
- 疑った後に構造を見る力が要る → 本編へのリンク
- 最終行案: 「疑うことは知覚の門を開く。だが、門の向こうに何があるかを見るには — 構造を見る目が要る。」

## 移送された素材 (「構造を見ろ」旧 §8.3 から)

> Prior の precision (精度) が無限大のとき、ベイズ更新は posterior = prior を返す。
> つまり**どれだけ証拠が積まれても信念が更新されない**。
> これは「自分が正しいと確信しすぎている系は、原理的に学べない」ことの数学的な表現である。

- **本たたき台での位置**: §2 (精度の罠) の核心部分。§1 「信ぜよ = precision ∞」の数学的裏付けとして使える
- **接続**: 「構造を見ろ」§8.3 結語の伏線脚注「しかし『確かめる』とは何か」が本稿への誘導
