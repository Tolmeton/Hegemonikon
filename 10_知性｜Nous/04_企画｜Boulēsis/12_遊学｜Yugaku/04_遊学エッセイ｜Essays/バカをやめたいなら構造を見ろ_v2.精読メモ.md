### C-10 §7.4 自然変換の説明を自然言語化 ✓

- **Creator の指摘**: 数式「α_Y ∘ F(f) = G(f) ∘ α_X」の正方形が可換、という数学語彙がやさしくない。自然言語での説明が欲しい
- **旧版の問題**: §4.1 で合成記号 ∘ を丁寧に導入した配慮が §7.4 でいきなり「正方形が可換」という高抽象の語彙を出すことで台無しになっていた
- **処置**:
  - 冒頭を自然言語化: 「関手 $F$ による記述から関手 $G$ による記述への一貫した変換」
  - 「一貫した」の意味を 2 ステップで分解: (a) 各対象への翻訳手続き、(b) C の射に対する整合性
  - 数式を残した上で、「二つの道のり」のメタファーで可換性を説明: (道のり 1) C で動いてから翻訳 / (道のり 2) 先に翻訳してから D で動く
  - 具体例として原子の古典軌道関手 $F_{\text{classical}}$ と量子軌道関手 $F_{\text{quantum}}$ を導入
  - Bohr の水素原子モデルが一部の射では整合し、電子スピン / 量子もつれで破綻する事実を「自然変換が全域では成立しない」例として提示
  - §7.1 の脳/コンピュータ層別忠実性と論理的に接続
- **効果**: 数学語彙のまま残しつつ、自然言語の層を追加。読者の吸収度が大幅に改善
- **適用**: v2.5.0

### C-11 §7.4 「強力に裏付ける」→ Creator 案で書き換え ✓

- **Creator の指摘**: 「この歴史的事実は、本稿の主張を強力に裏付ける」は言い過ぎ。自画自賛が過ぎてキモい。事実から主張への飛躍を感じる
- **旧版の問題**:
  - 論理構造: (1) Eilenberg-Mac Lane が自然変換を定式化したかった → (2) 自然変換 = アナロジーの質の評価 → (3) 圏論の動機 = アナロジー評価 → (4) 本稿の主張が強力に裏付けられる
  - (1)→(2)は本稿の解釈、(2)→(3)は過度の一般化、(3)→(4)は論理的飛躍
  - 「彼らは暗黙に…認識していた」は外部権威への訴え (argumentum ad verecundiam)
- **Creator 提示の書き換え案 (採用)**:
  > Eilenberg と Mac Lane が圏論を創始したときの動機の一つが自然変換の定式化だったという事実は、「複数の関手の比較」が圏論の中心的関心事の一つであることを示している。本稿のように関手をアナロジーと読む場合、この「複数の関手の比較」は「複数のアナロジーのうちどれが本質を捉えているか」という質的評価の問題に翻訳される。これは、Eilenberg-Mac Lane が本稿と同じ意図を持っていたわけではないが、本稿の読み方が圏論の内的な論理と整合することを示唆すると言えよう。
- **変更点**:
  - 「強力に裏付ける」→「整合することを示唆すると言えよう」(弱化 + 推論性)
  - 「彼らは暗黙に…認識していた」→「Eilenberg-Mac Lane が本稿と同じ意図を持っていたわけではない」(明示的な自己制限)
  - 事実 (圏論創始の動機) と解釈 (本稿の読み方) を明示的に分離

### C-04 §5.2 効率表の整合性問題 (5 件同時修正) ✓

#### C-04-1 記号定義ブロック追加 ✓
- 表の直後に $n, m, c_s, c_o, c_a, c_F, \Delta_s, \Delta_p, \Delta_t, \Delta_m, T(m), |\text{Mor}(D)|$ の意味を一括定義
- $T(m)$ の関数形が公理系に依存することも注記

#### C-04-2 帰納の Accuracy 正確化 (平方根則) ✓
- **問題**: 旧表 Accuracy = $k \cdot \Delta_p$ (n 非依存) が本文「統計的検定力の平方根則」(L194) と不整合。旧表だと効率 = 定数/n (線形減少) になり、平方根則 (1/√n) より強い減少を主張していた
- **処置**: Accuracy を $\Delta_p \cdot \sqrt{n}$ に正確化、効率を $\Delta_p / (c_o \sqrt{n})$ に修正。本文も「平方根で逓減」を明示
- **脚注 ‖ の更新**: 「Accuracy 増分を $\sqrt{n}$ に比例させるのは、統計的検定力の平方根則 (standard error $\propto 1/\sqrt{n}$) に対応する近似」と明示

#### C-04-3 演繹の `T(m)` / `T` 表記不整合 ✓
- **問題**: 表で「定理 $T(m)$ 個」と書きながら効率式では `$T \cdot \Delta_t$` と `T` を使用 (関数と変数の混在)
- **処置**: Accuracy を `$T(m) \cdot \Delta_t$`、効率を `$T(m) \Delta_t / (m c_a)$` に統一

#### C-04-4 アナロジー行に脚注 ♮ 新設 ✓
- **問題**: $|\text{Mor}(D)|$ への比例が暗黙に忠実関手を前提しているが、表では前提が明示されていない
- **処置**: 脚注 ♮ 新設:「$|\text{Mor}(D)|$ への比例は忠実関手 (§7.1) を前提とした heuristic な概算。非忠実関手の場合は射が潰れるためこの単純比例は成立しない。正確な定量化は §5.3 の $\Delta_F$ を参照」

#### C-04-5 記号 $\delta$ → $\Delta_s$ 統一 ✓
- **問題**: 暗記だけ $\delta$ (単独記号)、他は $\Delta_p, \Delta_t, \Delta_m$ (添字付き)
- **処置**: 暗記の $\delta$ を $\Delta_s$ (s = single fact) に変更。$\Delta_*$ 系で統一
