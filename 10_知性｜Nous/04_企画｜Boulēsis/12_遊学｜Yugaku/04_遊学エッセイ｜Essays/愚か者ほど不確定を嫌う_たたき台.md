# 愚か者ほど不確定を嫌う
## — 楽に逃げたら不幸になる理由

## コンセプト

### シリーズ接続

| 接続元 | 接続の射 |
|:--|:--|
| ②馬鹿ほど自信を持つ | $\pi_0$ 過剰 (知的) → 本作は $\pi_0$ 過剰 (情動的): 不確実性が**嫌だから**拒否する |
| ⑧世界を否定する | バカの世界否定 → 本作はその**情動的動機**: 不確定を受容する苦痛から逃げていたこと |
| ③狂っているは褒め言葉 | 「世界に近い」= 不確定を受容できている → 本作はその逆: 確定を欲するほど世界から遠ざかる |

### 核心命題

1. **現実は不確定的である** — エントロピーは原理的にゼロにならない。認知体はこの不確実性を嫌う（VFE 最小化の衝動として）
2. **認知的不協和 = 見えない借金** — 知覚を歪めて主観に合わせる ($\pi_0 \uparrow\uparrow$) と、主観的な予測誤差は減るが、**真の予測誤差** (世界との乖離) は無慈悲に増え続ける
3. **楽に逃げると不幸になる** — 真の予測誤差の蓄積 = 環境からの罰 (不適応) の蓄積。認知的借金はいつか返済を迫られる

### Creator のメモの FEP 対応

| Creator のメモ | FEP の定式化 |
|:--|:--|
| 現実は不確定的 | 世界の隠れ状態 $s$ の分布 $p(s)$ は原理的にデルタ関数にならない |
| 認知体は不確実性を嫌う | VFE 最小化の衝動: $F \to \min$ (Complexity 項 = KL divergence を下げたい) |
| 知覚を主観に合わせる | $\pi_0 \gg \pi_s$: prior が感覚入力を上書き → **知覚の主観化** |
| 真なる予測誤差は増え続ける | $F_{\text{perceived}} \downarrow$ だが $F_{\text{true}} = D_{KL}[q(s) \| p(s \mid o_{\text{real}})] \uparrow$ |
| 認知的不協和 = 見えない借金 | $\Delta_{\text{debt}} = F_{\text{true}} - F_{\text{perceived}} > 0$ が累積 |
| 不幸をもたらす逃避 | 蓄積した借金 $\sum \Delta_{\text{debt}}$ はいつか環境フィードバック (罰) として返済を迫られる |

## 構造案 (核心の骨格のみ — 肉付け未定)

### §1　なぜ人は確定を求めるのか

- FEP 的回答: 不確実性 = VFE の増大 → 生存への脅威信号
- 不確実性を感じること自体がストレス（コルチゾール上昇）
- **しかし**: 不確実性を人工的に消去すること ≠ 不確実性が消えたこと

---

### §2　二つの予測誤差 — 主観的 vs 真の

ここが核心:

$$F_{\text{perceived}} = D_{KL}[q(s) \| \tilde{p}(s)] \quad \text{(主観が信じている世界との距離)}$$

$$F_{\text{true}} = D_{KL}[q(s) \| p(s \mid o)] \quad \text{(本当の世界との距離)}$$

- $\tilde{p}(s)$ = 歪められた世界モデル（prior が支配した知覚）
- $p(s \mid o)$ = 真の posterior（データに基づく世界）

**認知的逃避**: $\pi_0 \uparrow\uparrow$ → $\tilde{p} \to q$ → $F_{\text{perceived}} \to 0$（「自分は正しい」と感じる）

**真の帰結**: $F_{\text{true}} \uparrow\uparrow$（世界からの乖離は増大し続ける）

> 認知的不協和の解消 = **$F_{\text{perceived}}$ を下げながら $F_{\text{true}}$ を上げる操作** = 見えない借金

---

### §3　楽と不幸の因果

- 「楽」= $F_{\text{perceived}}$ の即時的減少 → 短期的快楽
- 「不幸」= $F_{\text{true}}$ の蓄積が閾値を超え、環境からのフィードバック (罰) が来る
- 例: 浪費 (「お金はある」と信じる = $F_{\text{perceived}} \downarrow$) → 破産 ($F_{\text{true}}$ が現実化)
- 例: 健康軽視 (「健康だ」と信じる) → 生活習慣病 → 「真の予測誤差」の清算

> 不確実性から逃げるコスト = **金利付きの借金** — 逃げるほど利子が膨らむ

---

### §4　愚か者の逃避 vs 賢者の受容

| | 愚か者 | 賢者 |
|:--|:--|:--|
| 不確実性への反応 | $\pi_0 \uparrow$ (prior で上書き) | $\pi_s \uparrow$ (感覚入力を受容) |
| 主観的体験 | 安心 → 長期的不幸 | 不安 → 長期的適応 |
| VFE の挙動 | $F_{\text{perceived}} \downarrow$, $F_{\text{true}} \uparrow$ | $F_{\text{perceived}} \uparrow$ (一時), $F_{\text{true}} \downarrow$ |
| 比喩 | 借金で暮らす | 倹約して投資する |

> 不確実性を受容するのは**楽ではない**。しかし不確実性から逃げるのは**もっと楽ではない** — ただし請求書が後から来る

---

### §5　反論の取り込み (骨格)

- 「全てを疑えば生きられない」→ 正しい。不確実性の受容は**全てを疑う**ことではない。prior precision を**適切に**設定すること (⑧のニーバーの祈り参照)
- 「楽観主義は健康に良い」→ 部分的に正しい。しかし楽観主義 ≠ 現実否認。楽観的な generative model と、知覚を歪める prior precision の過剰は別物
- 「不確実性に強い人は生まれつき？」→ 環境要因 (安全基地) + 訓練可能 (マインドフルネス ≈ $\pi_s$ の意図的向上)

---

### §6　結語 (骨格)

不確実性は消えない。消えないものを消そうとする行為は — 定義からして — 失敗する。

不確実性への正しい態度は、消すことでも逃げることでもなく、**精度を設定すること**。何が確実で、何が不確実かを区別し、不確実なものについては**不確実なまま**保持する能力 — それが②で定義した「賢さ」の情動的基盤である。
