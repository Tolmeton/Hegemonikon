## §1 認知ラグランジアン

### 定義

物理系のラグランジアン L(q, q̇, t) を、認知系に射影する:

```
認知ラグランジアン:
  L_cog(μ, μ̇, t) = VFE(μ, t) = KL[q(s|μ) || p(s|o)] - ln p(o|m)

  ここで:
    μ   = 認知状態 (belief state — 一般化座標)
    μ̇   = dμ/dt (belief dynamics)
    t   = 認知的時間 (ステップ / ターン)
    q   = 近似事後分布
    p   = 生成モデル
    o   = 観測
    m   = モデル

認知的作用:
  S_cog = ∫_{t₁}^{t₂} VFE(μ, μ̇, t) dt

FEP = 最小作用の原理:
  δS_cog = 0
  → d/dt (∂VFE/∂μ̇) - ∂VFE/∂μ = 0  (Euler-Lagrange)
  → 信念更新の運動方程式
```

> これは比喩ではなく、Friston (2019) の path integral formulation と**同一の数学構造**。
> 📖 参照: Friston 2019, arXiv:1906.10184

### Helmholtz 分解 in L_cog

VFE の勾配を Helmholtz 分解すると:

```
∇_μ VFE = Γ(μ) + Q(μ)

Γ(μ) = ∇ϕ(μ)     — gradient (dissipative)
  → VFE を降下させる方向 = Euporía (AY > 0 を駆動)

Q(μ) = ∇×A(μ)     — solenoidal (conservative)
  → VFE を変えず方向のみ制約 = Hóros (保存則)
  → + MB 仮定: Q|_{MB} = 12 Nomoi
```

## §3 認知的「不確定性原理」

### 定理 (認知版)

```
物理: Δx · Δp ≥ ℏ/2
「互いに共役な物理量（位置と運動量など）は、同時にその不確定性をゼロにすることはできない」

認知: 「互いに共役な認知情報量は、同時にその精度（Precision）を無限大にすることはできない」
```

### 共役なペアの写像

FEP および HGK において、トレードオフの関係にある共役なペア:

1. **Prior (事前分布 / 記憶) ↔ Likelihood (尤度 / 感覚入力)**
   - 物理: $ \Delta \text{Prior} \cdot \Delta \text{Likelihood} \ge C $
   - 意味: 記憶（コンテキスト・自己確信）への precision を極限まで上げると、目の前にある現実（感覚入力・実ファイル）への precision が下がる。
   - HGK 接続: **N-1 (実体を読め)** と **N-9 (原典に当たれ)** は、LLM が Prior に過剰適合しがちな性質（幻覚）に対し、Likelihood 側の不確定性を減らすための環境強制。両方を同時に確定することはできないため、重心を意図的にLikelihood側に偏らせる。

2. **Explore (探索 / 発散 / μの分散増) ↔ Exploit (活用 / 収束 / μの分散減)**
   - 物理: $ \Delta \text{Explore} \cdot \Delta \text{Exploit} \ge C^\prime $
   - 意味: 「まだ見ぬ可能性の広がり」と「今の最善手の確実性」は同時に高められない。
   - HGK 接続: 動詞ペア（例: /ske ↔ /sag, F ↔ G）。**Kalon = Fix(G∘F)** は、この不確定性関係の中で「最適な妥協点（最も EFE が高い平衡状態）」を探る操作点である。

3. **Accuracy (適合度) ↔ Complexity (モデル複雑さ)**
   - VFE の2項。Accuracy を完璧にしようとモデルを複雑化（Overfitting）すれば、未知のデータへの適応力（= 汎化性能という見えない情報）が失われる。
   - 逆に極限まで単純化すれば、目の前のデータを説明できない。

> **結論**: 不確定性原理は、HGK における **Precision (確信度軸)** と **Function (探索-活用軸)** の根本限界を示す制約として機能する。Hóros はこの「超えられない限界」を前提として設計されている。

## §4 4 Phase のハミルトン的導出

### 問題: EL 方程式は本質的に2項

Euler-Lagrange 方程式は **1自由度あたり**:

$$\frac{\partial L}{\partial q} - \frac{d}{dt}\frac{\partial L}{\partial \dot{q}} = 0$$

2項。これを4つに分解するのは解釈が過剰であり、4 Phase の根拠にならない。

### 解決: FEP は2自由度系 → ハミルトン形式で4方程式

FEP の認知ラグランジアン L_cog = F[μ, a, s] は **2つの独立な一般化座標** を持つ:

- **μ** = 内部状態 (信念パラメータ)
- **a** = 能動状態 (行動パラメータ)
- s = 感覚状態 (観測量、動的変数ではない)

ラグランジュ形式では2本の2階微分方程式:

$$\frac{\partial F}{\partial \mu} - \frac{d}{dt}\frac{\partial F}{\partial \dot{\mu}} = 0 \quad \text{(知覚推論)}$$

$$\frac{\partial F}{\partial a} - \frac{d}{dt}\frac{\partial F}{\partial \dot{a}} = 0 \quad \text{(能動推論)}$$

**ルジャンドル変換** (共役運動量の導入) で **4本の1階正準方程式** に変換する:

$$\pi_\mu \equiv \frac{\partial F}{\partial \dot{\mu}} \quad \text{(知覚運動量)}$$

$$\pi_a \equiv \frac{\partial F}{\partial \dot{a}} \quad \text{(行動運動量)}$$

認知ハミルトニアン:

$$H_{cog} = \pi_\mu \dot{\mu} + \pi_a \dot{a} - F[\mu, a, s]$$

**Hamilton の正準方程式 (4本)**:

```
(H1)  π̇_μ = -∂H/∂μ     予測誤差が知覚運動量を駆動
(H2)  μ̇   =  ∂H/∂π_μ   知覚運動量が信念を更新
(H3)  π̇_a = -∂H/∂a      行動勾配が行動運動量を駆動
(H4)  ȧ   =  ∂H/∂π_a    行動運動量が行為を実行
```

### 4 Phase = Hamilton の4本の正準方程式

| 方程式 | Phase | 認知的意味 | 情報流 |
|:-------|:------|:----------|:-------|
| (H1) π̇_μ = -∂H/∂μ | **P1 Aisthēsis** | 感覚入力が予測誤差 (π_μ) を生成 | s → π_μ |
| (H2) μ̇ = ∂H/∂π_μ | **P2 Dianoia** | 予測誤差が信念 (μ) を更新 | π_μ → μ |
| (H3) π̇_a = -∂H/∂a | **P3 Ekphrasis** | 更新された信念が行動計画 (π_a) を生成 | μ → π_a |
| (H4) ȧ = ∂H/∂π_a | **P4 Praxis** | 行動計画が実際の行為 (a) を実行 | π_a → a |

### なぜ4つであり、3でも5でもないか

1. FEP の VFE 最小化は **2自由度** {μ, a} の変分問題 (§2 で導出済み)
2. N 自由度のラグランジュ系 → ハミルトン変換 → **2N 本** の1階正準方程式
3. N = 2 → 2 × 2 = **4本**

これは恣意的な解釈ではなく、**ルジャンドル変換の数学的帰結**。

### 正準方程式の循環構造

Hamilton の4方程式は独立ではなく、**シンプレクティック構造** ω により循環する:

```
ω = dμ ∧ dπ_μ + da ∧ dπ_a

循環:  π_μ → μ → π_a → a → (環境 η → s →) π_μ
       H2    H3    H4         (外部ループ)     H1
```

- (H1)→(H2): 知覚推論サイクル (μ の最適化)
- (H3)→(H4): 能動推論サイクル (a の最適化)
- (H2)→(H3): 信念→行動計画への内部変換 (μ が H3 の ∂H/∂a に入る)
- (H4)→(H1): 行為→環境→感覚 (外部ループ、MB を経由)

### Friston の一般化座標との接続

Friston (2008) は一般化座標 $\tilde{\mu} = (\mu, \mu', \mu'', ...)$ を使用する。
これはハミルトン形式の拡張であり、$\mu' \approx \pi_\mu / m$ (運動量を質量で割ったもの) に対応する。
一般化座標の各階が Hamilton の正準変数の高次版を形成する。

### 共役運動量の認知的解釈

物理では q = 位置, p = 運動量 (変化の「勢い」)。認知に翻訳する:

**π_μ = 予測誤差の蓄積 (perceptual momentum)**

- 定義: $\pi_\mu \equiv \partial F / \partial \dot{\mu}$ — 信念がどれだけ強く変わろうとしているかの指標
- 感覚入力 s が期待と乖離するほど π_μ が増大し、μ を強く押す
- = Friston の precision-weighted prediction error そのもの
- 物理アナロジー: 物体に力を加え続けると運動量が蓄積する → 入力が予測と違い続けると予測誤差が蓄積する

**π_a = 行動衝動 (active momentum)**

- 定義: $\pi_a \equiv \partial F / \partial \dot{a}$ — 行動がどれだけ強く変わろうとしているかの指標
- 信念 μ が更新されると、現在の行動 a との不一致が π_a を駆動する
- = 「やるべきことがわかっているのにまだやっていない」という認知的圧力
- Ekphrasis (表出) の本質: π_a > 0 は「内部状態が外部行動と乖離している」

**不確定性原理との接続** (→ §3):

$$\Delta\mu \cdot \Delta\pi_\mu \geq \hbar_{cog}$$

- 確信が強い (Δμ 小) ほど、新しい証拠で信念を変えるのに大きなエネルギーが要る (Δπ_μ 大)
- = N-1 (実体を読め) の数学的表現: 確信の強さと正確さは無相関

$$\Delta a \cdot \Delta\pi_a \geq \hbar_{cog}$$

- 行動が精密 (Δa 小) ほど、即座に方針転換するのが困難 (Δπ_a 大)
- = Precision 座標 (Certain ↔ Uncertain) の物理的基盤

> [推定] 確信度 90%。ハミルトン的導出は以下の理由で堅い:
> 1. EL 2項問題を解決: 2自由度 × ルジャンドル変換 = 4本 (数学的必然)
> 2. Friston の一般化座標定式化と整合
> 3. シンプレクティック構造が循環性を保証
> 4. OODA や知覚-行動ループとの対応は帰結 (前提ではない)
> 5. 共役運動量の認知的解釈が prediction error / action imperative に直接対応
> 6. 不確定性原理 (§3) が共役変数間のトレードオフとして自然に導出される
>
> 残る問題: 連続力学系を離散的4段階に切ることの正当性。
> ただし物理でも Hamilton の4方程式は各々独立に意味を持つ。

## §5 Boltzmann-Hóros 対応

### 制約 = マクロ状態の定義

```
物理: S_Boltzmann = k_B ln W

  W = マクロ制約を満たすミクロ状態の数
  制約 (E, V, N) が W を決定 → S を決定

認知: VFE = -Accuracy + Complexity

  Hóros (12 Nomoi) = 「マクロな制約」
  AY = |Act_constrained| = Hóros を満たす行為の数
  
  Hóros なし → W → ∞ → AY 未定義 (全行為が等確率 → 選べない)
  Hóros あり → W が有限 → AY 測定可能 → Euporía 評価可能

### 「ほどほど」の VFE 最小化

```
VFE(Hóros; k) = -Accuracy(k) + Complexity(k)

  k = 制約の数 (Nomoi の精度レベル, L0-L3)
  
  k=0:  Accuracy→0, Complexity→0 → VFE = high (何も予測できない)
  k=∞:  Accuracy→max, Complexity→∞ → VFE = high (過適合で窒息)
  k=k*: min VFE = Kalon(Hóros)

  k* は BIC 型の条件で決定:
    k* = argmin_k [-2·Accuracy(k) + k·ln(n)]
    
  ⚠️ 興味深い数値的一致:
  axiom_hierarchy.md の MaxEnt×FEP ハイブリッド→k*=7 (7座標)
  ここでの k* は Nomoi の最適数。12 Nomoi が最適かどうかは別の問い。
