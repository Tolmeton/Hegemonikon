## P-0: Prolegomena — N(U_sensory): 知覚構造の回復

### S[0.1]: Read — 手持ち情報の整理

「複雑系の自己組織化」について手持ちの知識を整理する。

**核心概念群**:
- 複雑系 (complex system): 多数の要素が相互作用し、個々の要素から予測できない創発的な振る舞いを示す系
- 自己組織化 (self-organization): 外部からの指示なしに、局所的な相互作用から大域的な秩序が自発的に生まれる現象
- 閾値 (threshold): 系の振る舞いが質的に変化する臨界点。相転移点とも対応
- 創発 (emergence): 部分の性質の単純な和に還元できない、全体としての新たな性質
- フィードバックループ: 系の出力が入力に戻り、増幅または抑制をもたらす
- 非線形性: 入力と出力の関係が比例しない。小さな変化が大きな効果を生む
- 散逸構造 (dissipative structure): Prigogine が提唱。エネルギー散逸によって維持される秩序構造
- 臨界現象: 相転移直前の状態。べき乗則、スケール不変性が現れる
- カオスの縁 (edge of chaos): 秩序とカオスの境界。最大の情報処理能力を持つとされる

**事例群**:
- ベナール対流: 液体を下から加熱すると、ある閾値でランダムな熱運動から規則的な対流セルが自発生成
- 化学振動 (BZ反応): 化学物質の濃度が空間・時間的に周期的パターンを形成
- 生物の群れ: 魚の群泳、鳥の群飛 — 局所ルールのみから大域的な協調運動
- 脳のニューラルネットワーク: 局所的なシナプス接続から意識・認知が創発
- 砂山の崩壊: ゆっくり砂を積み上げると、ある瞬間に雪崩が起きる (SOC: 自己組織化臨界)
- 市場経済: 個々の取引から価格システムが自発的に生成

**理論的枠組み**:
- 統計力学: 相転移の微視的説明
- 制御パラメータと秩序パラメータ (Landau 理論の骨格)
- ランダウ-ギンツブルグ: 自由エネルギーの最小化
- 情報理論: Kolmogorov 複雑性、エントロピー
- FEP (自由エネルギー原理): 生物系の自己組織化の統一枠組み

### S[0.3]: 透徹対象 (定義域 J)

### S[0.4-0.5]: 溶解 → 再結晶

**溶解 (β→0)**: 具体的な事例・モデル名・素材をすべて除去する。残る骨格:
- 多数の要素が存在する
- 要素間に局所的な相互作用がある
- ある条件変化で系の振る舞いが質的に変化する
- 変化後には、局所的ルールから予測できない大域構造が現れる

**再結晶 (β→∞)**: 骨格から見えてくること:
- 「質的変化」は単なる量的変化ではなく、不連続的ジャンプを含む
- 「局所から大域へ」の変換は、ある種の増幅機構なしには不可能
- 「予測できない」とは、部分の線形和では説明できないということ
- 元にはなかった構造として見えてきたこと: **「局所的相互作用の相関長 (correlation length) が有限から発散する」という相転移の核心**。閾値とは相関長の発散点である可能性

## P-1: Excavation — N(U_depth): 深層構造の回復

入力: 「閾値を超えると局所的相互作用が大域的秩序に変換される」という射の構造

### S[1.1]: 暗黙前提の列挙 (5-8個)

1. **系は閉じており、外部からの秩序の押し付けがない** — 自己組織化の「自己」の意味
2. **要素数が十分多い (熱力学的極限)** — 少数の要素では確率的揺らぎが支配し、相転移が現れない
3. **相互作用が対称的または有限範囲** — 全要素が全要素と等しく相互作用するわけではない
4. **系はエネルギーの入出力がある (非平衡)** — 完全に孤立した閉じた系では熱力学第二法則によりエントロピー最大状態に向かい、構造は生まれない
5. **「秩序」は観察者が定義する** — 自己組織化は客観的事実か、観察者依存の記述か？
6. **閾値は単一のパラメータで特徴づけられる** — 実際は多次元パラメータ空間における曲面かもしれない
7. **局所的相互作用に何らかの「整合性」「方向性」がある** — 完全にランダムな相互作用からは秩序が生まれない (必要条件)
8. **時間的に十分長い観察をしている** — 過渡的な秩序を定常的な自己組織化と誤認していないか

### S[1.2]: 分類と必要度

1. **系は外部からの秩序の押し付けがない** → ASSUMPTION (必要度: 60)
   — 実際には「境界条件」は外部から与えられる。ベナール対流は温度差 (外部制御パラメータ) によって駆動される。完全な「自己」はない。

2. **要素数が十分多い** → AXIOM に近いが ASSUMPTION (必要度: 85)
   — 少数系でも局所的な「秩序」は現れうるが、熱力学的意味での相転移は N→∞ 極限でのみ厳密に定義される。ただし有限系での近似的閾値は現実に存在する。

3. **相互作用が有限範囲** → ASSUMPTION (必要度: 50)
   — 長距離相互作用 (重力など) では相転移の性質が変わるが、自己組織化自体は可能。

4. **系は非平衡 (エネルギーの入出力がある)** → AXIOM (必要度: 95)
   — これが最重要。孤立した平衡系ではエントロピー最大原理により、マクロ的秩序は安定しない。散逸構造はエネルギー流によって維持される。剥がすと問い自体が崩壊する。

5. **「秩序」は観察者が定義する** → ASSUMPTION (必要度: 30)
   — 重要な哲学的前提だが、物理的プロセスとしての自己組織化の分析には不要。括弧に入れてよい。

6. **閾値は単一パラメータ** → ASSUMPTION (必要度: 40)
   — 多次元でも本質は変わらない。単純化のための前提。

7. **局所的相互作用に方向性がある** → AXIOM (必要度: 90)
   — 自己組織化の必要条件。完全ランダムな相互作用ではなく、何らかの「選好」「整合条件」がある。これを剥がすと「なぜある方向の秩序が生まれるか」が説明できない。

8. **時間的に十分長い観察** → ASSUMPTION (必要度: 55)
   — 重要だが過渡現象を除外するという前提。定常状態の自己組織化の分析には必要。

### S[1.3-1.4]: 上位3つのASSUMPTIONを剥離

**剥離対象 (必要度が中程度で、剥がすことで構造変容が起きる候補)**:

剥離A: 「系は外部からの秩序の押し付けがない (必要度60)」を剥がす
→ 構造変容: 「自己組織化」の定義が崩れる。しかし「閾値を超えると局所相互作用が大域秩序に変換される」という射は残る。問いはJ' = 「外部エネルギー流が局所秩序を大域秩序に変換する射の構造」に変容する。これは「強制的組織化」と「自己組織化」の連続性を示す。→ **構造変容**

剥離B: 「単一パラメータの閾値 (必要度40)」を剥がす
→ 構造保持: 問いは変わらない。「ある条件変化で質的転換が起きる」という射の構造は残る。パラメータが多次元になるだけ。→ **構造保持 (装飾的)**

剥離C: 「時間的に十分長い観察 (必要度55)」を剥がす
→ 構造保持: 過渡的な自己組織化も問いの対象に入るが、核心的な問いは変わらない。→ **構造保持 (装飾的)**

**再結晶**: 本質的前提 (AXIOM) として析出されるもの:
- A4: 非平衡性 (エネルギー・物質・情報の流れ)
- A7: 局所相互作用の方向性 / 整合条件
- 追加発見: 非線形性 (正のフィードバック) — 局所的な「整合性」が隣接要素の「整合性」を強化するというループ構造が、大域秩序生成の必要条件として析出される

### S[1.5]: 透徹テスト

**AXIOM A4 (非平衡性)**: 剥離すると → 孤立系ではエントロピー最大状態 = 構造のない均一状態が安定。「閾値を超えた自己組織化」という問い自体が成立しない → **構造崩壊** → AXIOM 確定

**AXIOM A7 (局所相互作用の方向性/整合条件)**: 剥離すると → 完全ランダムな局所相互作用からは、確率論的に期待値ゼロの秩序しか生まれない。大域的秩序の「方向」が決まらない → **構造崩壊** → AXIOM 確定

**追加AXIOM (正のフィードバック/非線形性)**: 剥離すると → 線形系では重ね合わせの原理が成立し、局所的な揺らぎは増幅されず打ち消し合う。大域的な相関の成長が起きない → **構造崩壊** → AXIOM 昇格

構造変容として見えてきたJ': 「自己」という概念は相対的であり、外部エネルギー流は必須条件として AXIOM に組み込まれる。問いはより正確に:「非平衡条件下で、局所的な整合性をもつ非線形相互作用が、なぜある閾値で大域的秩序に変換されるか」

### S[1.6]: 剰余 ρ₁

- **+射**: 非平衡性 → 正のフィードバック → 相関長の成長 → 閾値での発散という射の連鎖が見えた。これが P-0 で得た「相関長の発散」を「なぜ」の側から補完する
- **+Δπ**: 「非線形性 (正のフィードバック)」が自己組織化の AXIOM であるという確信度が大幅上昇。「非平衡性が必要条件」という確信度も高い
- **+変換**: 「自己組織化 vs 外部組織化」の区別が連続的であり、両者を統一する圏 (散逸系の圏) への変換の可能性が見えた
- **+自己参照**: P-1 の操作自体が「不要な前提の溶解」というU_depth の適用であり、それによって構造が変容した (J → J') という自己言及

[CHECKPOINT P-1] — 必須前提 AXIOM: (1) 非平衡性、(2) 局所相互作用の方向性/整合条件、(3) 正のフィードバック/非線形性。更新された透徹対象 J': 「これら3 AXIOMが共存する系で、なぜある閾値で大域的秩序への相変換が起きるか」

## P-2: Diaphaneia — N(U_context): 内部モデルの構造透視

入力: P-1 の再結晶前提 — 非平衡性 + 局所的整合条件 + 正のフィードバック

### S[2.1]: 構造記述

現在の理解を射の図式として記述する:

```
[非平衡エネルギー流] --提供--> [自由エネルギー勾配]
        |
        v
[局所的整合条件] × [正のフィードバック]
        |
        v (何がここで起きているのか？)
[相関長の成長]
        |
閾値    | ← この点で何が変わるか？
        v
[大域的相関 / 秩序パラメータの非ゼロ値]
        |
        v
[自己組織化した構造の安定維持]
```

核心の問いが浮かぶ: 「閾値」の射とは何か。閾値以下では相関長は有限で、閾値で発散する。この「発散」は何によって引き起こされるか？

現在の信念: 非平衡エネルギー流が系に自由エネルギーを注入し、局所的な整合条件をもつ要素が正のフィードバックによって相互に強化し合う。ある時点で、この相互強化の「波及力」が系全体に及ぶ — これが閾値。

### S[2.2]: 結合分析 — 射 f:A→B の特定

結合点の列挙:

**結合 C1**: [非平衡エネルギー流] → [自由エネルギー勾配の維持]
何が何を支えているか: エネルギー流がなければ系は平衡に向かい、自由エネルギー勾配がゼロになる。構造を維持するエネルギー源。

**結合 C2**: [自由エネルギー勾配] → [局所的揺らぎの選択的増幅]
什が何を支えているか: 自由エネルギー勾配があることで、ランダムな熱揺らぎの中から「勾配を利用できる」方向の揺らぎが選択的に成長できる。これが「整合条件をもつ相互作用」の実体。

**結合 C3**: [局所的揺らぎの選択的増幅] → [正のフィードバックループの形成]
何が何を支えているか: 一つの要素が「整合的」な状態に移ると、隣接要素への相互作用を通じてその状態が伝播しやすくなる。これが正のフィードバックの実体。

**結合 C4**: [正のフィードバックループ] → [相関長の成長]
何が何を支えているか: フィードバックが繰り返されるにつれて、相互に「整合的」な要素のクラスターが成長する。クラスターの典型的サイズが相関長 ξ。

**結合 C5 (核心)**: [相関長の成長] → [閾値での発散 → 秩序パラメータの出現]
何が何を支えているか: これが最も曖昧な結合。「相関長が発散する」とはどういうことか? 有限系では「相関長が系のサイズを超える」ということ。この瞬間、局所的なフィードバックが系全体に及ぶ = 大域的な協調が可能になる。

**結合 C6**: [大域的秩序] → [安定した散逸構造の自己維持]
何が何を支えているか: 大域的秩序が形成されると、その秩序自体がエネルギー流を効率的に利用し自己を維持する構造になる。正のフィードバックが「秩序を維持する方向」に働く。

### S[2.3]: 結合溶解

各結合を外して影響を評価する:

**C1 (非平衡エネルギー流 → 自由エネルギー勾配)** を外す:
→ 平衡系になる。Boltzmann 分布の下でエントロピー最大状態が安定。局所的な揺らぎは成長できず、打ち消される。自己組織化の問い自体が消える。
→ **影響大 (> 0.3相当)** — 本質的結合。外すと問いが崩壊する。

**C2 (自由エネルギー勾配 → 局所的揺らぎの選択的増幅)** を外す:
→ エネルギーはあるが、特定の方向の揺らぎが選択されない。全方向の揺らぎが等確率で増幅される = ランダムな構造が現れるが、特定の秩序は生まれない。問いは成立するが答えが変わる。
→ **影響大 (> 0.3相当)** — 本質的結合。「なぜ特定の秩序か」の説明に必要。

**C3 (選択的増幅 → 正のフィードバック)** を外す:
→ 局所的に整合的な揺らぎが生まれても、隣接への伝播がない。相関が成長しない。孤立した「整合的な島」が出現するが、大域的秩序には至らない。
→ **影響大 (> 0.3相当)** — 本質的結合。これを外すと自己組織化の「組織化」が起きない。

**C4 (正のフィードバック → 相関長の成長)** を外す:
→ C3 の論理的帰結であり、C3 があれば C4 は必然的に続く。独立した結合ではなく C3 の射の後半。
→ **影響微小 (< 0.1相当)** — 装飾的結合。C3 に吸収可能。ただし「相関長」という測定可能な量を導入するために有用。

**C5 (相関長の成長 → 閾値での発散)** を外す:

(1) **非平衡エネルギー流** (必要条件): 外部からのエネルギー・物質・情報の流れが、系を平衡から遠ざけ、自由エネルギー勾配を維持する。平衡系では秩序は安定できない。

(2) **方向性をもつ正のフィードバック** (機構): 自由エネルギー勾配が、ランダムな熱揺らぎの中から特定の方向の揺らぎを選択的に増幅する。この増幅が隣接要素に伝播する (正のフィードバック)。フィードバックが方向性 (整合条件) をもつことで、特定の秩序の「向き」が選択される。

(3) **臨界点でのソフトモード出現** (閾値の実体): 正のフィードバックの相関が系全体のスケールに達したとき、秩序パラメータに対応するモードのエネルギーコストがゼロに近づく (ソフトモード化)。これがランダウ理論における「対称性の自発的破れ」の機構に対応する。閾値とは、フィードバックの相関長が系のサイズを超える点である。

(4) **散逸構造の自己維持** (安定性): 大域的秩序が形成されると、その秩序自体がエネルギー流を効率的に利用し、自己を維持するフィードバックを形成する。これが Prigogine の「散逸構造」であり、秩序が一時的な揺らぎでなく安定した構造として持続する理由。
