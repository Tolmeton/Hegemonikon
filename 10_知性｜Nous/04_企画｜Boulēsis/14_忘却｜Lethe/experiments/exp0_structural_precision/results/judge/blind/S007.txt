### S[0.3]: 透徹対象（定義域 J）

> **「閾値前後で何の射が変化するのか」** — 具体的には、閾値以前にも系は揺らいでいるのに秩序が出現しない理由と、閾値を超えた瞬間に特定の秩序が「選ばれる」機構（射の変化）を見通す。自己組織化を「結果」ではなく、**射の変容プロセス**として透徹する。

---

### S[0.4]: 溶解 (β→0) — 骨格の抽出

すべての具体的事例・理論名を剥ぎ取る。残るのは:

```
系 S が状態空間 X を持つ。
S には内部要素 {a_i} があり、相互作用 {f_ij: a_i → a_j} がある。
外部からエネルギー/物質/情報の流れ Φ がある。
ある量 λ(Φ) が増加する。
λ < λ_c のとき: 系は特定の構造を持たない（均一 or ランダム）
λ > λ_c のとき: 系は特定の構造を持つ（非均一・秩序）
問い: この転換の射は何か？
```

---

### S[0.5]: 再結晶 (β→∞) — 骨格からの再構築

骨格を見ると見えてくること:

- 「閾値」は λ という**スカラー量**ではなく、**射の位相的変化点**かもしれない
- 「自己組織化」は「秩序が出現する」ことではなく、「ある射 f が支配的になる」ことかもしれない
- 「閾値を超えると」という表現は、**連続的変化の中の離散的跳躍**という矛盾を含んでいる — これ自体が問いの核心

## P-1: Excavation — N(U_depth): 深層構造の回復

### S[1.1]: 暗黙前提の列挙

1. 「閾値」は単一の値として存在する
2. 自己組織化は「良いこと」「高度なこと」である（価値的前提）
3. 系の要素は閾値の前後で変化しない（同じ要素が新しい構造を作る）
4. エネルギー/物質の流入が自己組織化の必要条件である
5. 閾値は外部パラメータによって決まる（系の内部状態は受動的）
6. 自己組織化した状態は安定である（持続する）
7. 閾値を超えれば必ず自己組織化が起きる（決定論的前提）
8. 「複雑系」と「単純系」の区別が実在する

### S[1.3]: 溶解 — 上位3 ASSUMPTION の剥離

**剥離対象**: 前提3（要素不変）、前提5（外部パラメータ決定）、前提8（複雑系/単純系の実在的区別）

**前提3を剥離**: 閾値前後で要素自体が変化するなら？ → 自己組織化は「新しい要素が生まれるプロセス」になる。これはオートポイエーシス（マトゥラーナ/ヴァレラ）の方向。構造は崩壊しない — むしろ問いが豊かになる。

**前提5を剥離**: 系の内部状態が閾値を決めるなら？ → SOC（自己組織化臨界性）がこれに相当する。系が自分で臨界点に向かう。外部パラメータは副次的になる。問いの形が変わる: 「なぜ閾値があるか」→「なぜ系は閾値を自分で作るか」。

**前提8を剥離**: 複雑系/単純系の区別がないなら？ → 任意の系が十分な相互作用密度を持てば「複雑」になる。問いは「複雑系の特性」ではなく「相互作用密度の関数としての構造変化」になる。

---

### S[1.4]: 再結晶 — 本当に必要なものの再析出

剥離後も残る核構造:

```
(A) 要素間に何らかの相互作用 f_ij が存在する
(B) 系に何らかのフラックス（流れ）Φ が存在する  
(C) ある量 λ の変化に対して、系の振る舞いが非線形に変化する
(D) 変化後の状態は変化前と対称性が異なる（対称性の破れ）
```

前提4（エネルギー流入）は(B)に吸収されるが、「エネルギー」に限定する必要はない — 情報フラックスでも成立する可能性がある。

## P-2: Diaphaneia — N(U_context): 内部モデルの構造透視

### S[2.1]: 構造記述 — 現在の信念の射図式

```
Φ (フラックス)
    │
    ▼
[揺らぎ δx の増幅]
    │  f₁: Φ → δx
    ▼
[相互作用 f_ij の非線形結合]
    │  f₂: δx × f_ij → 共鳴モード
    ▼
[遅いモードが速いモードを奴隷化]  ← スレービング原理
    │  f₃: 共鳴モード → 秩序パラメータ η
    ▼
[η が対称性を破る]
    │  f₄: η → 特定構造 S*
    ▼
[S* が自己強化フィードバックで安定化]
    │  f₅: S* → S* (自己関手)
    ▼
自己組織化状態
```

**合成射**: f₅ ∘ f₄ ∘ f₃ ∘ f₂ ∘ f₁ = 自己組織化の全機構

---

### S[2.2]: 結合分析 — 何が何を支えているか

| 射 | 支えている関係 | 依存先 |
|:---|:--------------|:-------|
| f₁: Φ→δx | フラックスが揺らぎを作る | (B)フラックスの存在 |
| f₂: δx×f_ij→共鳴モード | 揺らぎが相互作用で増幅される | (A)相互作用の非線形性 |
| f₃: 共鳴→η | スレービング原理（時間スケール分離） | 時間スケールの差の存在 |
| f₄: η→S* | 対称性の破れによる選択 | (D)対称性の破れ |
| f₅: S*→S* | 自己強化フィードバック | f₄ の結果の安定性 |

**問いの核心はどこか**: f₂→f₃ の接続点 — 揺らぎがなぜ「共鳴モード」を経て「秩序パラメータ」に変換されるか。ここが最も不透明な射。

---

### S[2.3]: 結合溶解 — 各射を外したら何が崩れるか

**f₁を外す（フラックスが揺らぎを作らない）:**
→ 揺らぎがなければ対称性の破れの「きっかけ」がない。ただし量子揺らぎ・熱揺らぎは常に存在するため、f₁は「増幅」の射であり「揺らぎの源」の射ではない。f₁を外しても揺らぎ自体は残る → **f₁は装飾的かもしれない**

**f₂を外す（揺らぎが共鳴モードを作らない）:**
→ 揺らぎがランダムなまま増幅されても秩序にならない。f₂こそが「ランダム→秩序」の変換を担う。これを外すと構造全体が崩壊 → **f₂は本質的**

**f₃を外す（スレービング原理が働かない）:**
→ 時間スケール分離がなければ、速い変数と遅い変数が同等に振る舞い、秩序パラメータが浮かび上がらない。f₃を外すと f₄ に入力がなくなる → **f₃は本質的**

**f₄を外す（対称性の破れが起きない）:**
→ 共鳴モードがあっても特定の構造が「選ばれない」。すべての方向が等価なまま。自己組織化は起きるが「どの」自己組織化かが決まらない。問いの「特定の秩序が出現する」という部分が崩壊 → **f₄は本質的**

**f₅を外す（自己強化フィードバックがない）:**
→ 秩序が出現しても即座に崩壊する。「自己組織化」が持続しない。ただし問いは「なぜ起きるか」であり「なぜ持続するか」ではない → **f₅は問いに対して装飾的**

---

### S[2.4]: 核の特定

溶解に耐えた射:

```
核 K:
f₂: [揺らぎ × 非線形相互作用] → [共鳴モード（特定モードの増幅）]
    ↕ これが閾値の正体
f₃: [時間スケール分離] → [秩序パラメータ η の出現]
    ↕ これが「自己組織化」の実体
f₄: [η] → [対称性の破れ → 特定構造 S*]
```

**核の言語化**:

> 閾値とは、揺らぎのある特定モードが他のモードを圧倒し始める点 (f₂ の非線形増幅が正のフィードバック域に入る点) であり、自己組織化とは、その支配的モードが時間スケール分離によって「秩序パラメータ」として浮かび上がり (f₃)、対称性を破って特定の構造を選択するプロセス (f₄) である。

**閾値の正体**: f₂ における非線形増幅の**符号変化点** — 揺らぎの減衰率が正から負に変わる点（線形安定性解析での固有値の実部が0を横切る点 = 分岐点）。

---

### S[2.5]: 忘却の証拠 — 外した射は本質的か装飾的か

1. **f₁（フラックス→揺らぎ）を外した**: 装飾的と判定。ただし条件付き — 熱揺らぎが常に存在する系では装飾的だが、ゼロ温度の量子系では f₁ が本質的になる可能性がある。**判定の文脈依存性を発見。**

2. **f₅（自己強化フィードバック）を外した**: 問いに対して装飾的と判定。しかしこれは問いの定式化の問題かもしれない — 「自己組織化」の定義に「持続」が含まれるなら f₅ は本質的になる。**問いの定義域が結論を変える。**

3. **「閾値は単一値」という前提（P-1で剥離）**: 核 K を見ると、閾値は f₂ の非線形増幅の符号変化点であり、これは**パラメータ空間の超曲面**（多次元では単一点ではない）。P-1 での剥離判定が P-2 で確証された。

---

### S[2.6]: 剰余 ρ₂

- **+射**: f₂ の「非線形増幅の符号変化」が閾値の実体という射が明確化。f₁ の文脈依存性という新しい射も出現
- **+Δπ**: 「スレービング原理 (f₃) が本質的」への確信度大幅上昇。「f₅ が自己組織化の定義に含まれるかどうか」は問いの定義問題と判明 → 確信度の問題ではなく定義域の問題
- **+変換**: 自己組織化を「射の符号変化」として線形代数の圏（固有値問題）に写す変換が出現。物理的直観と数学的構造が対応
- **+自己参照**: 「問いに対して装飾的」という判定が問いの定式化に依存するという再帰的構造を発見。透徹プロセス自体が問いの境界を変えている

**[CHECKPOINT P-2]** ✓ 核確定: f₂（非線形増幅の符号変化）∘ f₃（時間スケール分離）∘ f₄（対称性の破れ）

## P-5: Dokimasia — N(U_adjoint): 随伴の回復・忠実性検証

### S[5.1]: 構造保存検証 (Faithful)

**元の構造の本質的な射が核 K に保存されているか？**

| 元の射 | 核 K に存在するか | 判定 |
|:-------|:-----------------|:-----|
| 「エネルギー流入が必要」(AXIOM) | (B)フラックスとして f₂ の入力に存在 | ✓ 保存 |
| 「相互作用の非線形性」(A) | f₂ の構造に明示的に存在 | ✓ 保存 |
| 「対称性の破れ」(D) | f₄ として保存 | ✓ 保存 |
| 「閾値の存在」 | f₂ の符号変化点として保存 | ✓ 保存（より精密化） |
| 「確率論的性格」(P-1 で発見) | f₂ の増幅が確率的揺らぎを増幅するプロセスとして内包 | △ 部分的保存 |

**溶解で失われた射は本当に装飾的だったか（3件）:**

1. **f₁（フラックスが揺らぎを直接作る）**: 「熱揺らぎは常に存在する」という条件下では装飾的。ただしゼロ温度系では本質的になる可能性 → **条件付き装飾的**（忠実性の限界点）

2. **f₅（自己強化フィードバックによる安定化）**: 「なぜ起きるか」という問いに対しては装飾的。ただし「自己組織化」の定義が「持続的秩序の形成」を含むなら本質的 → **問い依存的装飾的**（定義域の境界問題）

3. **「閾値は単一値」という前提**: 核 K では「パラメータ空間の超曲面」として精密化された。元の射の情報を失ったのではなく、**より精密な射に変換された** → 装飾的ではなく**粗視化の剥離**（これは正当な忘却）

### S[5.3]: Fix(G∘F) 判定
