## P-1: 前提掘出 — 深層の前提を掘り起こす

入力: 「閾値＝揺らぎの増幅/減衰の分水嶺」という透徹対象

### ステップ 1.1: 暗黙前提の列挙（7個）

1. **「複雑系には閾値が存在する」** — 閾値は普遍的に存在し、それを超えることで自己組織化が起きる
2. **「自己組織化は秩序の生成である」** — 無秩序→秩序という一方向の変化として捉えている
3. **「閾値は単一の値（スカラー）である」** — パラメータ空間上の1点として閾値を想定している
4. **「外部エネルギーや物質の流入が必要である」** — 散逸構造論に基づく前提
5. **「要素は均質である」** — 要素間の相互作用が対称的・均質的であると暗黙に仮定している
6. **「自己組織化は目的論的でない（設計なし）」** — 目的や意図なしに秩序が生まれるという前提
7. **「閾値以下では系はランダムまたは均一な状態にある」** — 閾値以下の状態を「無秩序」と同一視している

---

| # | 前提 | 分類 | 必要度(0-100) | 根拠 |
|:--|:-----|:-----|:-------------|:-----|
| 1 | 複雑系には閾値が存在する | ASSUMPTION | 70 | 閾値なしに連続的に自己組織化する系も存在しうる（ゆっくりした相転移） |
| 2 | 自己組織化＝秩序の生成 | ASSUMPTION | 40 | 「秩序」の定義に依存。エントロピー的には局所的秩序生成＋全体的エントロピー増大 |
| 3 | 閾値は単一値（スカラー） | ASSUMPTION | 30 | 多次元パラメータ空間では閾値は超曲面になる |
| 4 | 外部エネルギー流入が必要 | AXIOM | 90 | 熱力学第二法則: 孤立系では局所的秩序は維持できない（エントロピー増大則） |
| 5 | 要素は均質 | ASSUMPTION | 25 | 不均質な要素分布でも自己組織化は起きる（むしろ不均質性が駆動することも） |

| 7 | 閾値以下＝無秩序 | ASSUMPTION | 35 | 閾値以下でも局所的な相関・秩序は存在する（相関長が有限なだけ） |

### ステップ 1.3-1.4: 溶解→再結晶

**上位3つのASSUMPTIONを剥離する（必要度が低い＝剥がしやすい順）:**

**前提3（閾値はスカラー）を剥離:**
→ 閾値が超曲面になっても、「揺らぎの増幅/減衰の境界」という構造は保たれる。剥離可能。

**前提5（要素の均質性）を剥離:**
→ 不均質な要素でも「局所的な揺らぎが全体に伝播するか否か」という問いは残る。構造保持。剥離可能。

**前提7（閾値以下＝無秩序）を剥離:**
→ **ここで構造が変容する。** 閾値以下でも相関は存在するなら、「閾値を超える」とは「相関長が発散する（系全体に及ぶ）」ことを意味する。これは元の問いの答えに直結する新構造。

**再結晶後の核:**
```
閾値 = 相関長が系のサイズを超える点
自己組織化 = 局所的相関が大域的相関に転換される過程
```

## P-2: 透徹 — 内部モデルの構造を透視する

入力: P-1の再結晶前提
- AXIOM: 散逸（エネルギー流）なき自己組織化は持続不可能
- 核命題: 閾値＝相関長が系全体に及ぶ点 / 自己組織化＝局所的相関の大域的転換

### ステップ 2.1: 構造記述

現在の理解を構造として明示する:

```
[複雑系の構造]
  ├─ 要素: 多数の相互作用する局所的エージェント
  ├─ 相互作用: 非線形、フィードバックを含む
  ├─ 状態: 各要素は局所的な相関を持つ（相関長 ξ）
  └─ 駆動: 外部からのエネルギー/物質の流入（散逸構造）

[閾値の構造]
  ├─ 閾値以下: ξ < L（系のサイズ）→ 局所的秩序のみ
  ├─ 閾値: ξ → ∞（臨界点）→ すべてのスケールで相関
  └─ 閾値以上: 大域的秩序パラメータが非ゼロになる

[自己組織化のメカニズム]
  ├─ 揺らぎが生じる（熱的/確率的）
  ├─ 閾値以下: 揺らぎは減衰（系が揺らぎを「吸収」する）
  ├─ 閾値: 揺らぎは減衰も増幅もしない（臨界減速）
  └─ 閾値以上: 特定の揺らぎモードが増幅 → パターン形成

[なぜ閾値で転換するか]
  └─ 非線形フィードバックが「正のフィードバックが負のフィードバックを超える点」
     = 系が自分自身の状態を維持・増幅できる臨界点
```

---

### ステップ 2.2: 結合分析

「何が何を支えているか」を列挙する:

| 結合 | 支えるもの | 支えられるもの |
|:-----|:---------|:-------------|
| A | 非線形相互作用 | 正のフィードバックの可能性 |
| B | 正のフィードバック | 揺らぎの増幅 |
| C | 揺らぎの増幅 | 相関長の拡大 |
| D | 相関長の発散 | 大域的秩序パラメータの出現 |
| E | 散逸（エネルギー流） | 系が平衡から遠ざかる状態の維持 |
| F | 平衡からの遠ざかり | 非線形相互作用の活性化 |
| G | 大域的秩序パラメータ | 自己組織化パターン |

**連鎖構造:**
```
散逸(E) → 平衡からの遠ざかり(F) → 非線形相互作用(A)
→ 正のフィードバック(B) → 揺らぎの増幅(C)
→ 相関長の発散(D) → 大域的秩序パラメータ(G)
```

---

### ステップ 2.3: 結合溶解

各結合を1つずつ外す:

**結合A（非線形相互作用 → 正のフィードバック）を外す:**
→ 線形系でも正のフィードバックは存在しうる（例: 線形増幅器）。
→ しかし線形系の正のフィードバックは単純な指数増大のみで、**特定のパターンを選択しない**。
→ 非線形性が「どの揺らぎモードが増幅されるか」の選択性を与える。
→ 結合Aを外すと: 秩序は生まれうるが **パターンの多様性と選択性が失われる**。部分的崩壊。

**結合B（正のフィードバック → 揺らぎの増幅）を外す:**
→ 正のフィードバックがなければ揺らぎはすべて減衰する。
→ **構造崩壊**: 閾値という概念自体が消える。増幅なき系には相転移がない。

**結合C（揺らぎの増幅 → 相関長の拡大）を外す:**
→ 揺らぎが増幅されても空間的に伝播しなければ相関長は拡大しない。
→ これを外すと: 局所的な振動は起きるが大域的なパターンは出現しない。
→ **重要な発見**: 相関長の拡大には「揺らぎの空間的伝播」という別の機構が必要。結合Cは実は2段階（増幅＋伝播）を含む複合結合だった。

**結合D（相関長の発散 → 大域的秩序パラメータ）を外す:**
→ 相関長が有限のままでも局所的な秩序は存在する。
→ しかし「自己組織化」と呼べる大域的なパターンは出現しない。
→ 構造崩壊: 「自己組織化」の定義の核心部分が失われる。

**結合E（散逸 → 平衡からの遠ざかり）を外す:**
→ 散逸なしでは系は熱平衡に向かう。熱平衡では詳細釣り合いが成立し、正のフィードバックが持続的に機能できない。
→ **構造崩壊**: P-1でAXIOMに昇格した前提を確認。散逸は不可欠。

**結合F（平衡からの遠ざかり → 非線形相互作用の活性化）を外す:**
→ 非線形相互作用は平衡状態でも「存在」するが、平衡では揺らぎが対称的に減衰するため、非線形項が有効に機能しない。
→ 平衡からの遠ざかりが「非線形性を有効化する文脈」を提供する。
→ 部分的崩壊: 非線形相互作用は存在するが機能しない。

---

### ステップ 2.4: 核の特定

溶解に耐えた（または崩壊が最も致命的だった）結合:

```
【構造の核】

核1: 散逸（エネルギー流）が系を平衡から遠ざける
     ↓
核2: 平衡からの遠ざかりが非線形フィードバックを有効化する
     ↓
核3: 正のフィードバックが揺らぎを「減衰させるか増幅させるか」の
     分岐点（閾値）を生み出す
     ↓
核4: 閾値では特定の揺らぎモードの増幅 + 空間的伝播が同時に起き、
     相関長が系全体に及ぶ
     ↓
核5: 相関長の発散が大域的秩序パラメータを出現させる
```

**最小核（これ以上削れない):**
> **「散逸によって維持される非平衡状態において、非線形正のフィードバックが揺らぎの増幅と空間伝播を臨界的に可能にする点が閾値であり、そこで相関長が発散して大域的秩序が出現する」**

---

### ステップ 2.5: 忘却の証拠（溶解で失ったもの）

| 失ったもの | 本質的か装飾的か | 判定理由 |
|:---------|:-------------|:---------|
| 「自己組織化＝秩序の生成」という直観的記述 | 装飾的 | 「相関長の発散による大域的秩序パラメータの出現」に置換可能。情報損失なし |
| 「閾値は単一の値（スカラー）」という単純化 | 装飾的 | 多次元パラメータ空間での超曲面に一般化しても核は保たれる |
| 「非線形性だけで自己組織化が説明できる」という信念 | **本質的に誤りだった** | 非線形性は必要条件だが十分条件ではない。散逸＋非線形性の組み合わせが必要。これは「失った」のではなく「誤りを発見した」 |

### ステップ 2.6: 収穫（P-2）

- **新しい関係**: 結合Cの分解で「揺らぎの増幅」と「揺らぎの空間的伝播」が別の機構であることが判明。自己組織化には両者が必要。これは元の問いへの重要な補完
- **確信度の変化**: 「非線形性が自己組織化の十分条件」という暗黙の信念の確信度が大幅低下。「散逸＋非線形性の組み合わせ」という複合条件への確信度が上昇
- **新しい類推**: 「揺らぎの増幅＋空間伝播の同時成立」→ レーザーでの誘導放出（増幅）＋光共振器（空間的選択・伝播）の組み合わせとの対応。また神経系での「局所的な興奮」＋「活動電位の伝播」との対応
- **思考プロセスへの気づき**: 結合を「1本の線」として見ていたが、実際には「複合結合（2段階以上のプロセスを内包する結合）」が存在する。結合溶解は結合を単純化しすぎるリスクがある

## P-5: 忠実性検証 — 構造が保存されたか検証する

入力: P-2の構造の核

### ステップ 5.1: 構造保存検証

**元の問いの本質的関係:**
- 「複雑系」「閾値」「自己組織化」の三者関係
- 「なぜ」という因果/機構の問い
- 「ある閾値を超えると」という不連続性・相転移的変化

**核の中にこれらは保存されているか?**

| 元の関係 | 核の中での対応 | 保存状態 |
|:---------|:-------------|:---------|
| 複雑系（多数の非線形相互作用する要素） | 非線形正のフィードバック＋散逸構造 | ✅ 保存（ただし「多数性」の役割が薄れた） |
| 閾値（不連続な変化点） | 正のフィードバックが揺らぎを増幅に転じる分岐点＋相関長の発散 | ✅ 保存かつ深化 |
| 自己組織化（外部設計なしの秩序出現） | 大域的秩序パラメータの自発的出現 | ✅ 保存 |
| なぜ（因果メカニズム） | 散逸→非平衡→非線形フィードバック有効化→揺らぎ増幅＋伝播→相関長発散 | ✅ 保存かつ深化 |

**「多数性」の役割が薄れた点について:**
→ 多数の要素が必要なのは「相関長が発散する余地（系のサイズ）を確保するため」と「統計的な揺らぎが確率的に生じるため」。核に暗黙に含まれているが、明示されていない。これは軽微な情報損失。

### ステップ 5.3: 安定性検証

**結論を蒸留して再確認:**

> 複雑系が閾値を超えると自己組織化するのは:
> 1. 散逸（エネルギー/物質の流れ）が系を熱平衡から遠ざけ
> 2. その非平衡状態が非線形正のフィードバックを有効化し
> 3. あるパラメータ値（閾値）で、特定の揺らぎモードが「減衰」から「増幅＋空間伝播」に転じ
> 4. 相関長が系全体に及ぶことで、大域的な秩序パラメータが自発的に出現するから

**もう一度見直す: 何か変わるか?**

→ 「なぜ閾値が存在するか」という問いへの答えが: **「正のフィードバックと負のフィードバックの競合が、特定のパラメータ値でバランスを崩す（分岐する）から」** という形でさらに凝縮できる。

→ 変化あり: 「分岐」という概念が核心語として浮上。しかし既存の構造と矛盾しない。強化的変化。

**安定性スコア: 15/100**（低い = 安定）
→ 到達判定。P-2への差し戻し不要。
