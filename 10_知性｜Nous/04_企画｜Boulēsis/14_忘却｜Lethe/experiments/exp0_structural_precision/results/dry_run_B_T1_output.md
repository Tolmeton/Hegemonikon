## P-1: Excavation（前提掘出）

### S[1.1]: 暗黙前提の列挙

1. **P-A**: 自己組織化は「閾値を超えると起きる」— 段階的ではなく不連続に起きる
2. **P-B**: 系には「無秩序状態」と「秩序状態」という2つの安定状態がある
3. **P-C**: 閾値はパラメータ空間の特定の点として定義できる
4. **P-D**: 自己組織化には「自由エネルギーの最小化」や「エントロピー生成」が関与している
5. **P-E**: 要素間の相互作用が自己組織化を「引き起こす」（因果的に）
6. **P-F**: 「自己」組織化 — 外部設計者なしに秩序が生まれる
7. **P-G**: 閾値は普遍的（系の詳細によらない臨界指数が存在する）

### S[1.2]: AXIOM / ASSUMPTION 分類

| 前提 | 分類 | 必要度 | 根拠 |
|------|------|--------|------|
| P-A: 不連続な相転移 | **ASSUMPTION** | 60 | 連続的相転移（2次転移）も存在する。「不連続」は特殊ケース |
| P-B: 2安定状態の存在 | **ASSUMPTION** | 55 | 多安定・準安定・カオスなど多様な相が存在する |
| P-C: 閾値の定義可能性 | **ASSUMPTION** | 70 | 有限系では閾値が滲む（finite-size effect）。厳密な点は熱力学極限のみ |
| P-D: 熱力学的関与 | **ASSUMPTION** | 50 | 情報論的・ゲーム論的自己組織化は熱力学と独立に定式化可能 |
| P-E: 相互作用の因果性 | **AXIOM** | 95 | 相互作用なしの自己組織化は定義矛盾。除去不可 |
| P-F: 外部設計者の不在 | **AXIOM** | 90 | 「自己」の定義に含まれる。除去すると問い自体が消える |
| P-G: 普遍性（臨界指数） | **ASSUMPTION** | 65 | 普遍性クラスは存在するが、全ての系に適用されるわけではない |

### S[1.3]: 溶解 — 上位3つの ASSUMPTION を剥離

**P-A（不連続性）を剥離:**
→ 構造保持。「段階的な自己組織化」でも問いは成立する。P-A は装飾的。

**P-C（閾値の定義可能性）を剥離:**
→ 構造変容。「閾値」という概念が消えると、問いは「なぜ系の振る舞いが変化するのか」に変形される。
→ **これは重要**: 閾値は現象の記述であり、説明の基盤ではないかもしれない。

**P-D（熱力学的関与）を剥離:**
→ 構造保持。情報論的に定式化しても自己組織化は説明可能。P-D は特定の圏での記述に過ぎない。

### S[1.4]: 再結晶 — 本当に必要なものだけ

再結晶後の前提セット:
- **必須**: P-E（相互作用の因果性）+ P-F（外部設計者の不在）
- **条件付き必須**: 「振る舞いの変化が存在する」（P-C の弱化版）
- **剥離確定**: P-A（不連続性）、P-D（熱力学的枠組み）、P-G（普遍性の強主張）

## P-2: Diaphaneia（透徹）

### S[2.1]: 構造記述 — 現在の理解

「複雑系が閾値を超えると自己組織化する」という信念の構造:

```
[多数の要素] × [局所的相互作用] × [非線形性]
    ↓ (パラメータ増加)
[臨界点 = 閾値]
    ↓ (超えると)
[大域的秩序 = 自己組織化]
```

この構造には暗黙の因果連鎖が埋め込まれている:
「相互作用 → 閾値 → 秩序」という直線的因果

### S[2.2]: 結合分析 — 何が何を支えているか

結合点の列挙:

| 結合 | A | → | B | 役割 |
|------|---|---|---|------|
| C1 | 局所相互作用 | → | 閾値の存在 | 相互作用が閾値を「生む」という前提 |
| C2 | 閾値の存在 | → | 秩序の出現 | 閾値が秩序の「原因」という前提 |
| C3 | 非線形性 | → | 不連続な変化 | 非線形性が閾値的変化を「説明」するという前提 |
| C4 | 秩序 | → | 自己組織化 | 秩序と自己組織化を同一視する前提 |
| C5 | パラメータ増加 | → | 相互作用密度増加 | 外部パラメータと内部相互作用の同一視 |

### S[2.3]: 結合溶解 — 各結合点を外す

**C1（相互作用 → 閾値）を外す:**
→ 閾値は相互作用から生まれるのではなく、*系の位相空間の幾何学的特徴*（鞍点、分岐点）として独立に存在するかもしれない。
→ 構造変容: 「相互作用が閾値を生む」ではなく「相互作用が位相空間の幾何を決定し、その幾何に閾値が埋め込まれている」

**C2（閾値 → 秩序）を外す:**
→ 閾値は秩序の原因ではなく、*秩序相と無秩序相の境界の記述*に過ぎない。
→ 構造変容: 問いは「なぜ境界を超えると変わるのか」から「なぜ2つの相が共存できるのか」へ反転する。

**C3（非線形性 → 不連続変化）を外す:**
→ 線形系でも閾値的変化は起きる（例: パーコレーション）。非線形性は十分条件ではあるが必要条件ではない。
→ 構造保持（C3は装飾的）

**C4（秩序 = 自己組織化）を外す:**
→ 秩序は静的な状態。自己組織化は動的なプロセス。同一視は誤り。
→ 構造変容: 自己組織化は「秩序状態への到達」ではなく「秩序を維持するフラックス（流れ）の確立」かもしれない。

**C5（外部パラメータ = 内部相互作用密度）を外す:**
→ 温度・濃度などの外部パラメータは相互作用の*強度*を変えるが、*構造*は変えない場合がある。
→ 構造変容: 閾値は相互作用の強度ではなく、相互作用の*接続パターン（トポロジー）*の変化で決まる可能性。

### S[2.4]: 核の特定 —
