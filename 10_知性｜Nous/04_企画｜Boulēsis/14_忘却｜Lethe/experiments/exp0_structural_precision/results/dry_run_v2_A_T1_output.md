## P-1: 前提掘出 — 深層の前提を掘り起こす

### ステップ 1.1: 暗黙前提を列挙する

1. **「閾値」は単一の点として存在する** — 閾値は一つの臨界値であり、その前後で振る舞いが変わる
2. **「自己組織化」は秩序の増加である** — 自己組織化 = エントロピーの局所的減少 = 構造の出現
3. **「複雑系」は多数の要素と相互作用があれば成立する** — 要素数と相互作用の多さが複雑系を定義する
4. **閾値の前後で「同じシステム」が存在し続ける** — 閾値を越えても系の同一性は保たれる
5. **自己組織化は「外部設計なし」に起きる** — エージェントや設計者がいないことが自己組織化の定義に含まれる
6. **閾値と自己組織化は因果関係にある** — 閾値を超えることが自己組織化の「原因」である
7. **自己組織化は安定した構造を生む** — 出現した秩序は持続する（一時的でない）
8. **「複雑系」と「単純系」は本質的に異なる** — 複雑系に特有の現象として自己組織化を扱っている

## P-2: 透徹 — 内部モデルの構造を透視する

### ステップ 2.1: 構造記述

**現在の理解（P-1 を経た後）:**

```
[深層変数: 相互作用密度 ρ、エネルギー流強度 Φ]
         ↓
[ρ と Φ の積が臨界値を超える]
         ↓ （同時に）
[閾値の出現]    [自己組織化の出現]
         ↓              ↓
[状態空間の縮小: 可能状態数の急激な減少]
         ↓
[安定なマクロ構造（秩序）の出現]
```

自分が信じていること:
1. 閾値と自己組織化は同一の深層変数から派生する（因果ではなく共変）
2. 自己組織化の本質は「状態空間の圧縮」である
3. この圧縮は、ミクロな自由度がマクロな拘束に「吸収」されることで起きる
4. 「なぜ閾値でそれが起きるか」は、非線形系では相互作用が**乗算的**に増大するため、ある点で「ミクロ自由度の数」を「マクロ拘束の数」が超える

---

### ステップ 2.2: 結合分析

「何が何を支えているか」を列挙する。

```
結合 α: [相互作用密度 ρ] → [正フィードバックループの形成可能性]
結合 β: [正フィードバック] → [揺らぎの選択的増幅]
結合 γ: [揺らぎの選択的増幅] → [特定モードへの状態集中]
結合 δ: [状態集中] → [状態空間の縮小]
結合 ε: [エネルギー流 Φ] → [散逸による非平衡状態の維持]
結合 ζ: [非平衡状態の維持] → [正フィードバックが維持可能]
結合 η: [状態空間の縮小] → [マクロ秩序の出現]（= 自己組織化）
結合 θ: [ρ と Φ の積] → [臨界点の位置]（= 閾値）
```

**支持構造の要約:**
- ρ（相互作用密度）と Φ（エネルギー流）が基底
- 正フィードバックが媒介
- 状態空間の縮小が結果
- 閾値はこの縮小が「急激に」起きる点

---

### ステップ 2.3: 結合溶解

各結合点を 1 つずつ外す。

**結合 α を外す（ρ → 正フィードバック）:**
→ 相互作用密度が高くても正フィードバックが形成されないとしたら？
→ 例: 全要素が同じ方向に揃っている場合、相互作用は「増幅」ではなく「飽和」を生む
→ 崩れるもの: 「相互作用密度が高ければ自己組織化する」という直接的主張
→ **残るもの: 正フィードバックは相互作用密度の必要条件ではなく、その「非線形性」が必要**

**結合 β を外す（正フィードバック → 揺らぎの選択的増幅）:**
→ 正フィードバックがあっても揺らぎが選択的に増幅されないとしたら？
→ 例: 全揺らぎが均等に増幅される場合、「特定モード」への収束は起きない → カオスになる
→ 崩れるもの: 「正フィードバック＝自己組織化」という等式
→ **残るもの: 「選択性」が自己組織化に必須。選択性はどこから来るか？ → 対称性の破れ**

**結合 γ を外す（揺らぎの選択的増幅 → 特定モードへの状態集中）:**
→ 揺らぎが選択的に増幅されても、特定モードに集中しないとしたら？
→ 例: 複数の安定モードが同等に存在する場合、系は複数モード間を遷移し続ける
→ 崩れるもの: 「増幅 → 一つの秩序」という単純な連鎖
→ **残るもの: 「一つのモードへの収束」には非線形的な安定化（負フィードバックとの組合せ）が必要**

**結合 ε を外す（Φ → 非平衡状態の維持）:**
→ エネルギー流がなくても自己組織化するとしたら？
→ 例: 平衡系でも相転移（磁化、結晶化）は起きる
→ 崩れるもの: 「散逸構造論が自己組織化の唯一の説明」という主張
→ **残るもの: 自己組織化には平衡型と非平衡型の2種類がある。閾値の性質が異なる**

**結合 θ を外す（ρ×Φ → 臨界点の位置）:**
→ 閾値が ρ と Φ の積で決まらないとしたら？
→ 例: ネットワーク構造（トポロジー）が閾値の位置を決定的に変える（スケールフリーネットワークは閾値が消滅する）
→ 崩れるもの: 「相互作用密度とエネルギー流が閾値を決める」という単純な主張
→ **残るもの: 閾値の位置はシステムのトポロジーにも依存する。閾値は「密度」だけでなく「構造」の関数**

---

### ステップ 2.4: 核の特定

溶解に耐えた部分 = 本質的に不可分な構造の核:

```
【核 I】: 「対称性の破れ」が自己組織化の必要条件
  → 正フィードバックだけでは不十分。「どの揺らぎが選ばれるか」という選択性が必要
  → 選択性は対称性の破れから来る

【核 II】: 「非線形性」が閾値の不可避性を生む
  → 線形系には閾値が出現しない（連続的変化のみ）
  → 非線形系では相互作用が乗算的に増大し、ある点で「正フィードバックが負フィードバックを超える」
  → これが閾値の数学的必然性

【核 III】: 「状態空間の縮小」が自己組織化の本質
  → 自己組織化 = ミクロ自由度数 > マクロ拘束数 の状態から、ミクロ自由度数 ≤ マクロ拘束数 の状態への転換
  → 言い換え: 「何でもありうる」状態から「これしかありえない」状態への圧縮

【核 IV】: 自己組織化の「自己」は「系+環境の境界」に依存する
  → 平衡型（結晶化）と非平衡型（散逸構造）で「自己」の境界が異なる
  → 共通するのは「核 I〜III」であり、「散逸」は非平衡型の特殊条件
```

**統合的核:**

> **非線形系において、相互作用の乗算的増大がある点で「対称性の破れを選択的に増幅する正フィードバック」を可能にする。その点が閾値であり、その結果として状態空間が圧縮される。この圧縮が自己組織化の正体である。閾値と自己組織化は別々の現象ではなく、同一の非線形臨界現象の「位置」と「結果」である。**

### ステップ 2.5: 忘却の証拠

溶解で失ったものは本質的か装飾的か？

1. **「散逸構造論が自己組織化の唯一の説明」** → 装飾的。平衡型自己組織化（結晶化）が反例として存在する。散逸は非平衡型の特殊条件であり、核ではない。

2. **「閾値は単一の臨界点」** → 装飾的（理想化）。実際にはクロスオーバー領域があり、スケールフリーネットワークでは閾値が消滅する。「単一点」は熱力学的極限（要素数→∞）での近似。
