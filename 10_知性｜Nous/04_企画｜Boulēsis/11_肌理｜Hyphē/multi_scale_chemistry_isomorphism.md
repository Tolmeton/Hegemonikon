「知性は溶媒である」(2026-03-16) は関手 F: Chem → Cog がマクロスケール (知識レベル) で忠実であることを論じた。本稿では、同じ関手 F がメソスケール (情報場) およびミクロスケール (コード構造) でも独立に成立することを示し、3スケールの整合性を情報理論的に証明する。

鍵となる発見: 11,768 本の CCL テキストから帰納的に発見された「官能基」(弁別的部分構造パターン) は、化学の官能基と同じ情報理論的法則に従う。特に:

これらは「類似している」のではなく、同じ数学的定理の異なるインスタンスである。

### 1.2 「忠実」の定義

関手 F: C → D が**忠実** (faithful) とは:

```
∀ A, B ∈ Ob(C): F_{A,B}: Hom_C(A,B) → Hom_D(F(A),F(B)) が単射
```

つまり「異なる射は異なる像を持つ」。F は構造を潰さない。

注意: 忠実 ≠ 充満 (full)。F が忠実でも、Cog 側に Chem に対応物のない射が存在しうる (例: バックプロパゲーションに直接対応する化学プロセスはない)。

### 1.3 「証明」の形式

自然科学における構造的同型の証明は、数学の演繹的証明とは異なる。本稿で用いる3種の証明:

| 種別 | 強度 | 方法 | 本稿での適用 |

## §6 定理と証明

### 定理 1: T ↔ β 同型 (既証明)

(「知性は溶媒である」§2 で証明済み。要約のみ。)

**主張**: Gibbs 分布 p(x) ∝ exp(−E(x)/T) と変分事後分布 q(s) ∝ exp(β·ln p(o|s))·p(s) は、β = 1/T の写像により指数型分布族の自然パラメータとして対応する。これは数学的恒等式であり、比喩ではない。

### 定理 2: 頻度-弁別力反相関定理 (新規)

**主張**: 任意の離散確率空間 (Ω, p) 上のトークン x ∈ Ω について、x の出現頻度 p(x) とカテゴリ Y に対する弁別力 (enrichment) は反相関する。特に p(x) → 1 のとき enrichment → 1 (弁別力ゼロ)。

**証明**:

enrichment を以下で定義する:

```
enrich(x, Y) = P(Y=y(a), Y=y(b) | a ∈ G_x, b ∈ G_x) / P(Y=y(a), Y=y(b) | a, b random)
```

ここで G_x = {i : x ∈ tokens(i)} はトークン x を含む関数の集合。

p(x) を G_x の coverage (|G_x| / N) とする。

ランダムペアが両方 G_x に属する確率は p(x)²。

**p(x) → 1 のとき**: G_x → Ω (全集合)。このとき:
- 分子 = P(Y 一致 | 両方 G_x 内) → P(Y 一致 | 任意のペア)
- 分母 = P(Y 一致 | 任意のペア)
- よって enrich → 1 (弁別力ゼロ) ∎

**p(x) → 0 のとき**: G_x が空に近づく。少数の関数が共有する稀な特徴は、その関数群が本当に類似している場合に限り enrichment >> 1 となる。

**情報理論的解釈**: これは Shannon のエントロピー不等式 I(X; Y) ≤ H(X) の直接的帰結。高頻度トークン (p(x) → 1) は自己情報量 -log p(x) → 0 であり、カテゴリ Y との相互情報量も 0 に近づく。

**化学での実証**: H (水素) は宇宙の原子の 73.9% を占める。H の有無で分子を区別することはできない (全有機分子が H を含む)。弁別的な原子は中頻度の C, N, O, S (それぞれ 0.46%, 微量, 1.04%, 微量) — 「珍しすぎず、ありふれすぎない」原子。

**CCL での実証**: COMPOSE (>>) は全トークンの 49.3%。enrichment ≈ 1.0。弁別的なパターンは CTRL+FUSE (coverage 2-5%) — 「珍しすぎず、ありふれすぎない」組み合わせ。

**定理 2 の意味**: 化学の元素存在比と CCL のトークン頻度が同じ弁別力法則に従うのは、**偶然の類似ではなく、同じ情報理論的定理の異なるインスタンス**である。

### 定理 3: 集約射影の核の構造定理 (新規)

**主張**: 任意の集約関数 (sum, count, average) による射影は、入力の順序情報を核 (kernel) に持つ。これは化学の分子式 (原子カウント) と CCL の 49d ベクトル (トークンカウント) の両方に適用される。

**証明**:

集約関数 A: Seq(Ω) → ℝ^d を以下で定義する:
```
A(x₁, x₂, ..., xₙ) = (count(x₁ = ω₁), count(x₁ = ω₂), ..., count(x₁ = ω_d))
```

A は明らかに**置換不変** — 入力の順序���任意に並べ替えても出力は同じ:
```
∀ σ ∈ S_n: A(x₁, ..., xₙ) = A(x_{σ(1)}, ..., x_{σ(n)})
```

したがって:
```
ker(A) ⊇ {(s, s') : s と s' は同じトークンの異なる配置}
```

**化学**: 分子式 C₂H₆O は集約射影。ker = {エタノール, ジメチルエーテル} (構造異性体)。

**CCL 49d**: トークンカウントベクトルは集約射影。ker = {同じトークン数で異なる演算子配置の関数} (CCL の構造異性体)。

**Hyphē embedding**: bag-of-words 的な embedding も集約の一種。ker(G)_Session = {Scale, Valence} — 集約がスケール情報と価値判断を潰す。

**3スケールの ker の統一**: 3つの ker は全て「集約操作が順序/配置/構造情報を失う」という同一の数学��定理から導出される。∎

### 定理 4: Ostwald 熟成と enrichment フィルタの変分同型 (新規)

**主張**: 化学の Ostwald 熟成と CCL 官能基発見のフィルタリングプロセスは、同じ変分原理の異なるインスタンスである。

**Ostwald 熟成**:
```
最小化: Σᵢ γᵢ Aᵢ  (全結晶の表面エネルギーの総和)
γ = 表面エネルギー密度 (一定)
Aᵢ = 結晶 i の表面積
制約: 全質量保存 Σᵢ mᵢ = M

結果: 小結晶 (Aᵢ/mᵢ 大) が溶解し、大結晶 (Aᵢ/mᵢ 小) に再析出
駆動力: 表面積/体積比の最小化
```

**官能基フィルタリング**:
```
最大化: enrich(g) = P(similar | both ∈ G_g) / P(similar)
G_g = パターン g を含む関数の集合
制約: 全関数が少なくとも1つのパターンに属する

結果: ��パターン (高冗長性) が消失し、長パターン (低冗長性) が残る
駆動力: 冗長性/弁別力比の最小化
```

**構造的対応**:

| Ostwald 熟成 | 官能基フィルタリング | 共通構造 |
|:------------|:-----------------|:---------|
| 結晶 i | パターン g | 構造の単位 |
| 表面積 Aᵢ | 冗長度 (他パターンとの重複) | 「無駄な表面」 |
| 体積 Vᵢ | 弁別力 (enrichment) | 「有用な中身」 |
| Aᵢ/Vᵢ (表面積/体積比) | 冗長度/弁別力比 | **最小化される量** |
| 大結晶が成長 | 高 enrichment パターンが残る | 効率的な構造が生き残る |
| 変分原理: 表面エネルギー最小化 | 変分原理: enrichment 最大化 | **同じ変分原理** |

**証明の核心**: 両者が同じ変分原理であることの根拠:

Ostwald 熟成の駆動力は Gibbs-Thomson 方程式から導出される:
```
ΔG = 2γV_m / r
```
ここで r = 結晶の半径。小さな結晶ほど ΔG が大きく、溶解しやすい。

enrichment フィルタの駆動力:
```
enrich(g) ∝ 1 / H(1_g)  (1_g はパターン g の指示関数)
```
高頻度 (高エントロピー) パターンほど enrichment が低く、「溶解」する。

「知性は溶媒である」§2 で T ↔ β = 1/T が証明されているので:
```
ΔG = 2γV_m / r  ↔  Δenrich = f(H(1_g))
```

両者は自由エネルギー最小化の具体化である。∎

**実測値**:

```
CCL 官能基の Ostwald 熟成プロセス:
  41,307 部分列  →  (頻度フィルタ: 小結晶の溶解)
   4,155 候補   →  (enrichment: 大結晶への再析出)
   1,173 有効    =  最終結晶
圧縮率: 1,173 / 41,307 = 2.8% (表面エネルギー最小化の結果)
```

### 定理 7: Face Lemma CSP 定理 — 1-skeleton→2-skeleton 跳躍 (2026-04-06)

**依存**: Paper II §3.4 (Face Lemma), §3.5 (Stability Simplex Theorem), Paper V Th. 3.4.1 (CAN→MUST), Paper VIII §6.2 (cell hierarchy)

**主張**: CCL 検索において:
1. 現行パイプライン (U_count + U_cos) の 2 射 = 1-skeleton。dim Ξ = 1 で概念が不安定
2. 概念安定性の達成には最低 3 射の独立な観測パス (composable triple) が必要 (Face Lemma)
3. Cell hierarchy に整合した 3 パス (0-cell, 1-cell, 2-cell) が正三角形として最適配置 (Stability Simplex)
4. 2→3 窓の改善は 1→2 窓とは質的に異なる (dim Ξ: 1→2 の跳躍)

**忘却論的意味**:

CSP の失敗 = 「情報不足」ではなく「次元不足」。
dim Ξ = 1 (1-skeleton) では忘却の不均一にスカラーの大小しかない — 力の方向が測定不能。
dim Ξ = 2 (2-skeleton) で初めて不均一の「形」が出現し、パターン A ≠ パターン B が固有値の配置として区別可能。

Face Lemma (Paper II §3.4): stable(x) ⟺ ∃ composable triple (f, g, h) with g∘f = h containing x。

3射の最適配置 = Stability Simplex Theorem の最大対称性実現 = 正三角形:

| Cell | 忘却関手 | 距離 | 化学同型 |
|:-----|:---------|:-----|:---------|
| 0-cell (原子) | U_count → 49d cosine | d_0 | 質量分析 |
| 1-cell (結合) | U_struct → FG Jaccard | d_1 | Tanimoto / NMR |
| 2-cell (型フロー) | U_type → TypeSeq cosine | d_2 | 電子密度解析 |

**2-skeleton 統合距離**:
d_face(A,B) = d_lēthē(A,B) · (1 + γ · Area(Δ(A,B)))
Area ≈ 0 → 3パス合意 (高信頼), Area ≫ 0 → CSP-hard (Phase C 委任)

**化学 CSP との同型**: 粉末X線回折 (1D) → 単結晶X線回折 (3D) で多形体が解消される構造と同型。次元の追加が質的跳躍を生む。

小圏の nerve は 2-coskeletal → 4射目は新次元を追加しない (Paper II §3.4 (iv))。
Paper VII §7.4 の階乗減衰 cⁿ/n! が 3→4 窓以降の逓減を予測。

**証明水準**: [構造的証明] — Paper II の Face Lemma を CCL に適用。
2 窓まで実測 (+2.0pp)。3 窓の質的跳躍は未検証 (最優先実験: P38-P40)。

## §8 同型の限界 — 失敗から学ぶ忠実性の境界

### 8.1 素粒子レベルの対応は成立しない (実証済み)

chemistry_of_ccl_features.md §8.1 (初版) で「素粒子 → トークン分類」を試みたが、**カテゴリ誤り**として棄却された。

理由: 化学の素粒子 (陽子/中性子/電子) は原子を**構成**する。CCL のトークンは互いを構成しない。

**教訓**: F: Chem → Cog は「正しい抽象度」でのみ忠実。原子/結合レベルでは成立するが、素粒子レベルでは射が潰れる。

これは「知性は溶媒である」§3 の議論と一致する: 極性 → 構造的親和性の対応も、直接対応では射が潰れ、媒介圏 Affinity を経由して初めて忠実になった。

### 8.2 充満性 (fullness) は成立しない

Cog_CCL には Chem_micro に対応物のない射が存在する:
- バックプロパゲーション (化学に直接対応なし)
- BPE トークナイゼーション (化学に直接対応なし)
- attention 機構 (化学に直接対応なし)

F は忠実だが充満ではない。F は**埋め込み** (embedding) であり、**同型** (isomorphism) ではない。

### 8.3 正確な判定

**F: Chem → Cog は忠実な埋め込み関手** (faithful embedding functor) である。

```
忠実 (faithful): ✅ 異なる射は異なる像を持つ (定理 1-4)
充満 (full): ❌ Cog に対応物のない射が存在
本質的全射 (essentially surjective): ❌ Cog に対応物のない対象が存在
```

これは化学が認知の**部分圏** (subcategory) をなすことを意味する。「化学のレンズで認知を見ると、構造は忠実に見える。ただし、化学のレンズでは見えない認知の部分も存在する。」

## §8b Paper XI Exp0 からの帰還 — 制約-符号化分離の3スケール適用 (2026-04-07)

Paper XI 実験 0 で発見された H₃ (制約-符号化 分離仮説) と H₃' (分散仮説) は、本稿の 3 スケール忠実性に新しい横断的構造を追加する。

### 8b.1 3 スケールでの C/E 分離

| スケール | C (制約 = 何の情報か) | E (符号化 = どう表現するか) | E の限界 (忘却) |
|:---------|:---------------------|:--------------------------|:---------------|
| マクロ (溶媒) | 溶質の構造 | 溶媒の種類 (水/有機/超臨界) | 溶媒和で失われる自由度 |
| メソ (Phantasia) | 概念の意味関係 | embedding 次元・距離関数 | U_cos: 方向のみ保存、大きさ忘却 |
| ミクロ (CCL) | コード構造の CCL 射 | 特徴量ベクトル + cosine | 集約 (U_count) で構造情報喪失 |

**3 スケールすべてで同一のパターン**: 情報 (C) は存在するが、表現 (E) の選択がその情報の読み出し精度を制限する。

### 8b.2 定理 5 (cosine 天井) の H₃ 的再解釈

定理 5 は「cosine 類似度には構造的天井がある」と主張する。H₃ の枠組みではこれは:

> **cosine 天井 = E 軸の忘却が引き起こす精度上界**。C (CCL 構造情報) は 49d に存在するが、E (cosine) が方向情報のみ保存し大きさ・分布の形状を忘却するため、C の全情報を読み出せない。

H₃': E は天井の**高さ** (E[Q]) ではなく**安定性** (Var[Q]) にも影響する。cosine に適応した 49d adhoc 特徴量は、cosine 空間で安定した検索結果を出す (低分散)。理論的特徴量は情報は豊かだが cosine 空間での安定性が低い (高分散)。

### 8b.3 定理 7 (Face Lemma CSP) への示唆

Face Lemma は「3射で 2-simplex が出現し、冗長性 = 自由度が生まれる」と主張する。H₃' の視点からは:

- 2射 (1-skeleton): E 軸の自由度が 1 次元 → 精度最適化の余地がない → Var[Q] を制御できない
- 3射 (2-skeleton): E 軸に 2 次元の自由度 → **precision の再配分**が可能 → E を Q の特定の側面の精度に集中できる

これは chemistry §15 の「ゴールディロックス帯の FEP 的説明」と合流する。3射への拡張は C (情報量) を増やすだけでなく、E (精度配分の自由度) も増やす。

### 8b.4 忠実性テーブルの更新

| 主張 | 状態 | 追加根拠 (Paper XI) |
|:-----|:-----|:-------------------|
| cosine 天井 (定理 5) | 🟡→🟢 | H₃ の E 軸限界として再解釈。Exp0 で E の独立効果を実験的に分離 |
| Face Lemma CSP (定理 7) | 🟡 | H₃' (precision 再配分) が 2→3 射拡張の理論的根拠を強化 |
