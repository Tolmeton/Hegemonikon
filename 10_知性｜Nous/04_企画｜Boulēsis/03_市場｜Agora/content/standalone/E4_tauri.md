
「デスクトップアプリを作りたいけれど、Electronは重すぎるし、ネイティブ言語は学習コストが高い……」

そんなジレンマに陥ったことはありませんか？ 私もその一人でした。Web技術で手軽に作りたい、でもPCのメモリを占有するのは嫌だ。そんなわがままな要望を叶えるために、最新の軽量フレームワーク「Tauri」と生成AIを組み合わせて、自分専用のデスクトップアプリを開発してみました。

驚くべきことに、コードの大部分は私が書いていません。今回は、AIと共に作り上げた「爆速・軽量デスクトップアプリ」の開発記録と、そこで直面した壁について共有します。

### 技術スタック：なぜTauriなのか

今回採用した構成は以下の通りです。

*   **フロントエンド:** React + Vite (UI構築)
*   **バックエンド:** Python (FastAPI)
*   **アプリ化:** Tauri (Rust製フレームワーク)

Tauriの最大の特徴は、ElectronのようにChromiumを丸ごと抱え込むのではなく、OS標準のWebviewを利用する点です。これにより、配布サイズとメモリ使用量が劇的に削減されます。

**表1：ElectronとTauriの比較（筆者環境での実測値）**

| 項目 | Electron (一般的なアプリ) | Tauri (今回作成したアプリ) | 改善率 |
| :--- | :--- | :--- | :--- |
| **インストーラーサイズ** | 約 120 MB | 約 8 MB | **約 1/15** |
| **起動時メモリ使用量** | 約 250 MB | 約 25 MB | **約 1/10** |
| **起動速度** | もっさり (2-3秒) | 爆速 (0.5秒以下) | **体感5倍** |

この数字を見ただけで、Tauriを選ばない理由がないほどです。しかし、ここには「Rust」という難解な言語の壁があります。そこでAIの出番です。

### AIによるコード生成率：ほぼ「全自動」

Rustはメモリ安全性に優れた言語ですが、初学者がいきなり書くにはハードルが高いのが現実です。しかし、今回はClaude 3.5 SonnetやGPT-4oを駆使することで、驚異的なコード生成率を叩き出しました。

*   **Rust (Tauri設定周り):** **80% AI**
    *   ウィンドウ制御やPythonプロセスの立ち上げなど、定型的ながら複雑な処理はすべてAI任せ。
*   **React (フロントエンド):** **90% AI**
    *   コンポーネントの作成、State管理、API通信部分。
*   **Python (バックエンド):** **95% AI**
    *   FastAPIのルーティング、ビジネスロジック。

私がやったことといえば、各パーツを繋ぎ合わせる「配管工」のような作業と、エラーが出たときのログをAIに投げ返すことだけでした。

### 表示崩れとの格闘：AIの弱点

順風満帆に見えた開発ですが、唯一苦労したのが「UIの表示崩れ」です。

AIは論理的なコード（RustやPythonのロジック）を書くのは得意ですが、視覚的な調整（CSSの微調整やレイアウト）は苦手とする傾向があります。「ボタンを右端に寄せて」と頼んでも、flexboxの指定が微妙に間違っていたり、ウィンドウサイズを変えると崩れたり。

結局、CSSに関しては自分でChromeの開発者ツールを開き、微調整を行う必要がありました。AIは「動くもの」は作れますが、「美しいもの」を作るにはまだ人間の感性によるフィードバックが不可欠だと感じた瞬間です。

### 読者が試せること：TauriのHello World

まずは、あなたの環境でTauriがいかに軽量かを体感してみてください。以下のステップは、Node.jsとRustがインストールされていれば5分で完了します。

1.  **プロジェクト作成**
    ターミナルで以下を実行します。
    ```bash
    npm create tauri-app@latest
    ```
2.  **設定**
    対話形式で進みます。「Project name」を決め、「Web assets」にViteなどを選択します。
3.  **起動**
    ```bash
    cd <プロジェクト名>
    npm install
    npm run tauri dev
    ```

これだけで、驚くほど軽量なデスクトップアプリが起動します。ここに普段使っているWebアプリのコードを移植するだけで、自分専用ツールの完成です。AIに「このHTMLをTauriで動くようにして」と頼めば、Rustの設定ファイルまで書いてくれますよ。
