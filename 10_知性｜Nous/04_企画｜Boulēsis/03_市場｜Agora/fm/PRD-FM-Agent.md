## 1. Summary

FM-Agent は、VLM (Vision-Language Model) ベースの GUI 自動化エージェントである。

## 3. Background

*Source: `/noe.prd`*

FileMaker Pro は 1987 年から続くレガシープラットフォームで、GUI 以外の操作インターフェースを持たない。HGK プロジェクトでは Forge エンジンで XML 生成を自動化したが、「配備」「検証」は手動のまま残っている。

2026-01 に AT-SPI (Linux アクセシビリティ) による自動化を検討したが、Wine 上の FM は AT-SPI 非対応であることが判明。Bytebot (VLM ベース) パスに方向転換した。2026-03 に Windows VM の自動構築 (QEMU/KVM + unattended install) を着手。

│  Forge   │────▶│ FM-Agent │────▶│ FM Pro (Win VM)  │

                     │
         ┌───────────┴───────────┐
         │                       │
    ┌────▼────┐            ┌─────▼─────┐
