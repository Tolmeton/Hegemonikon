## 1. 公理

### A0: 記述は信念分布の不可逆圧縮である

頭の中にある理解——曖昧で、豊かで、文脈に満ちた信念の全体——を、
文字という蝋に刻印する。刻印した瞬間に、何かが消える。必ず消える。

**Rate-Distortion 定式化**:

```
R(D) = min_{p(ℓ|m)} I(M; L)   s.t.  E[w(m) · d(m, ℓ)] ≤ D
```

- **M** = 信念分布（確率分布、統計的多様体上の点）
- **L** = 記号列（言語）
- **f: M → L** = 記述写像（不可逆圧縮）
- **d(m, ℓ)** = 歪み関数 — (m, ℓ) ペアに作用するコスト
- **w(m)** = 顕著性（salience）重み — m にのみ作用する注意配分
- **β** = Rate-Distortion トレードオフのラグランジュ乗数

> **A0 は標準 Rate-Distortion (Cover & Thomas) と同一ではない。**
> 標準 R-D は X = X̂（同一アルファベット）がデフォルト。
> A0 は「信念(連続・主観) → 言語(離散・共有)」を明示的に規定し、M ≠ L は公理に内在する。

### FEP との関係

```
              FEP（変分自由エネルギー最小化）
              ╱                         ╲
       認知 (HGK v4.1)              記述 (Týpos v7.0)
       予測誤差の最小化             圧縮損失の最小化
       1公理 → 7座標 → 24動詞      1公理 → 7基底 → 24記述行為
```

Týpos は HGK の姉妹ではなく**娘**。FEP の記述行為への別の帰結。

**HGK との同型性**: 体系核 (1 + 7 + 24 = 32) の三層構造、構成距離配分 (1+3+3)、
生成規則 (生成子 × 6修飾 × 4極 = 24) が HGK v4.1 と**構造的同型 (Structural Isomorphism)**。
概念（内容）の同一性ではなく、構造（数・配分・生成規則）の対応。

## 2. 基底体系 — A0 からの構成距離

> **成分数は結果として出るべきであり、目標であってはならない。** — Creator

### 基底の資格条件

A0 から導出される次元は、以下の条件を満たすとき「基底」と呼ぶ:

1. **導出可能性**: A0 からの追加仮定数 d が有限
2. **独立性**: 他の基底の組合せで生成できない
3. **相補的対立 (Opposition)**: 不可分の二極。一方を定義すれば他方が自動的に定まる

### 7つの基底 (1 + 3 + 3)

A0 の Rate-Distortion 式からの構成距離 d で配置する:

| d | 基底 | 対立 (Opposition) | 数式成分 | 追加仮定 | HGK 対応座標 |
|:--|:-----|:------------------|:---------|:---------|:------------|
| 0 | **Endpoint** | source ↔ target | M, L | 0: f: M→L に内在 | Flow (I↔A) |
| 1 | **Reason** | arché ↔ telos | d(m, ℓ) | 1: 圧縮に目的がある | Value (E↔P) |
| 1 | **Resolution** | precise ↔ compressed | β | 1: R-D トレードオフ | Function (Expl↔Expt) |
| 1 | **Salience** | focused ↔ diffuse | w(m) | 1: 全信念が等しく重要ではない | Precision (C↔U) |
| 2 | **Context** | local ↔ global | 階層的 M, L | 2: R-D + 階層的構文 | Scale (Mi↔Ma) |
| 2 | **Order** | object ↔ meta | L の自己参照 | 2: R-D + Tarski | Valence (+↔-) |
| 2 | **Modality** | actual ↔ possible | 代替 p(ℓ|m) | 2: R-D + Kripke | Temporality (Past↔Future) |

> **Image (旧 d=0 基底) は A0 そのものであり基底ではない。**
> HGK で FEP が座標ではないのと同じ。Endpoint が f: M→L の domain/codomain として d=0 に座る。

### d=1 の3基底はラグランジアンの3つの独立パラメータに完全対応

```
L = I(M; L) + β · E[w(m) · d(m, ℓ)]
         ↑           ↑      ↑       ↑
      (暗黙)    Resolution Salience Reason
```

| パラメータ | 作用域 | 基底 | 独立性の根拠 |
|:----------|:-------|:-----|:------------|
| d(m, ℓ) | (m, ℓ) ペア | Reason | 歪みの種類（意味的・統語的・音韻的等） |
| β | スカラー | Resolution | 圧縮強度（Rate-Distortion トレードオフ） |
| w(m) | m のみ | Salience | ドメイン空間の測度（注意配分） |

> **Salience の独立性**: w(m)·d(m,ℓ) = d'(m,ℓ) として Reason に吸収可能。
> しかし w(m) は m にのみ作用（ドメイン測度）、d(m,ℓ) は (m,ℓ) ペアに作用（距離関数）。
> 関数型（作用域）が異なる。HGK の Precision (π=V[ε]⁻¹) も q(s) に吸収可能だが独立座標。
> **吸収「できる」≠ 吸収「すべき」** — 構造的分離原理。

### d=2: 公理 + ドメイン固有理論

d=2 は A0 に加え、ドメイン固有（言語学・論理学・様相論理学）の追加仮定を必要とする:

| TYPOS d=2 | ドメイン理論 | HGK d=2 対応 | HGK のドメイン理論 |
|:----------|:-----------|:------------|:----------------|
| Context | 言語学（階層的構文） | Scale | 階層的生成モデル |
| Order | Tarski（メタ言語） | Valence | Seth 2013（内受容予測） |
| Modality | Kripke（可能世界） | Temporality | Pezzulo 2022（時間的深度） |

> **注**: d=2 の HGK-TYPOS 対応は**構成的対応 (Constitutive Correspondence)** であり、
> 概念の同一性ではない。「同じ構成距離にある」= 同じ構造的位置にある。

### 基底から除外されたもの

| 候補 | 除外理由 |
|:-----|:--------|
| Image | A0 そのもの。公理であり基底ではない (v7.0 で確定) |
| Valence (+↔-) | TYPOS の基底ではなく TYPOS-HGK インターフェース上の次元 |
| Temporality | Reason の 志向(未来)/経緯(過去) に概ね吸収される |

> **Valence/Temporality が住む場所**: 「記述を使う認知主体」が持ち込む次元。
> 記述の構造的基底ではなく、記述を認知と接続するインターフェース。

## 5. 保存原理 — 歪みコストによる優先順位

A0 (不可逆圧縮) + Rate-Distortion 理論:

```
R(D) = min I(M; L)  subject to  E[w(m) · d(m, ℓ)] ≤ D
```

歪みコストの高い基底を先に保存する:

| 基底 | 歪みコスト | 理由 |
|:-----|:----------|:-----|
| **Endpoint** | 極大 | 射そのもの。省略不可 |
| **Reason** | 高 | 圧縮の目的なしに射を解釈できない |
| **Resolution** | 中〜高 | 圧縮度は文体・ジャンルから推測可能 |
| **Salience** | 中 | 文脈から注意配分が推測可能な場合がある |
| **Context** | 低 | デフォルト圏で代替可 |
| **Order** | 最低 | デフォルト object-level で代替 |
| **Modality** | 最低 | デフォルト actual で代替 |
