## 1. 公理

### A0: 記述は信念分布の不可逆圧縮である

頭の中にある理解——曖昧で、豊かで、文脈に満ちた信念の全体——を、
文字という蝋に刻印する。刻印した瞬間に、何かが消える。必ず消える。

- 信念 = 確率分布 = 統計的多様体 M 上の点
- 記述 = M から記号列 L への写像 f: M → L
- 不可逆 = f は忠実関手ではない (Kolmogorov)

FEP の記述版として:

```
F[記述] = D_KL(q‖p) − E_q[log p(y|x)]
```

- q(x) = 読み手が記述から構築する事後信念
- p(x) = 書き手が伝えたい事前信念
- y = テキスト（観測）
- F = 記述の自由エネルギー = (冗長性) − (忠実性)

**最適な記述は F を最小化する。**

### FEP との関係

```
              FEP（変分自由エネルギー最小化）
              ╱                         ╲
       認知 (HGK)                   記述 (Týpos)
       予測誤差の最小化             圧縮損失の最小化
       1公理 → 6座標 → 24定理      1公理 → 5基底 → 24成分
```

Týpos は HGK の姉妹ではなく娘。FEP の別の帰結。

## 2. 基底体系 — A0 からの導出距離

> **v5 までの10成分は「数え上げ」だった。v6 では基底から成分を導出する。**
> **成分数は結果として出るべきであり、目標であってはならない。** — Creator

### 基底の資格条件

A0 から導出される次元は、以下の4条件を満たすとき「基底」と呼ぶ:

1. **導出可能性**: A0 からの追加仮定数 d が有限
2. **独立性**: 他の基底の組合せで生成できない
3. **相補的対立 (Opposition)**: 光と闇のような不可分の二極。一方を定義すれば他方が自動的に定まる
4. **記述の本質的側面**: これなしには記述を完全に特徴づけられない

### 5つの基底 (1 + 2 + 2)

A0 からの導出距離 d で配置する:

| d | 基底 | Opposition | 追加仮定 |
|:--|:-----|:----------|:---------|
| 0 | **Image** (暗黙) | 信念 ↔ 記号 | なし。A0 の定義 f: M→L に内在 |
| 1 | **Endpoint** | 始域 ↔ 終域 | 射には domain と codomain がある |
| 1 | **Reason** | 志向 ↔ 経緯 | 射には「何のために」と「何に基づいて」がある |
| 2 | **Context** | 局所 ↔ 大域 | 射の有効範囲にスコープがある |
| 2 | **Order** | 対象 ↔ メタ | 記述は記述を対象にできる (L ⊂ M) |

### HGK 座標との対応

| Typos 基底 | HGK 座標 | d | 対応の根拠 |
|:-----------|:---------|:--|:----------|
| Image | Flow (I↔A) | 0 | 公理の両辺がそのまま Opposition |
| Endpoint | Value (E↔P) | 1 | 射の方向 = 価値の方向 |
| Reason | Function (Expl↔Expt) | 1 | 目的↔根拠 = 探索↔活用 |
| Context | Scale (Mi↔Ma) | 2 | 局所↔大域 = ミクロ↔マクロ |
| Order | Precision (C↔U)? | 2 | **暫定対応。要検証** |

> **注**: HGK には6座標あるが Typos は5基底。Valence (+↔-) に対応する Typos 基底は存在しない。
> 記述には身体的情動方向がない — Týpos は行為の理論であって感情の理論ではない。
> **Precision (C↔U)** との対応は暫定的。Order の Object↔Meta は確実↔不確実ではない。

### 各基底の外部検証

10のフレームワークによる交差検証 (Desktop Claude, 2026-02-20):

| 基底 | 支持数 | 決定的フレームワーク |
|:-----|:-------|:-------------------|
| Image | 9/10 | 全フレームワークで普遍的確認 |
| Endpoint | 9/10 | SFL interpersonal, RSA S₁/L₁ |
| Reason | 4/10 | **Speech Act Theory** (Searle の direction of fit) |
| Context | 5/10 | SFL textual, Relevance Theory |
| Order | 2/10 | Jakobson metalingual, 圏論 2-morphism |

#### Reason の独立性: Speech Act Theory からの決定的支持

- **Arche** (経緯) ≈ Word-to-World fit — 世界に合わせて言葉を調整
- **Telos** (志向) ≈ World-to-Word fit — 言葉に合わせて世界を変える

SFL 等の「言語=体系」フレームワークでは Reason は interpersonal に吸収可能。
しかし Speech Act Theory の「言語=行為」フレームワークでは独立次元。
**A0 が記述を「能動的圧縮」として定義する以上、Reason は正当化される。**

#### Order の導出: 記述の自己適用

A0: f: M → L。ここで:

- **仮定1**: L ⊂ M — 記述 (記号列) は信念の対象になりうる
- **仮定2**: f は L に対しても適用可能 — 記述を記述できる

2つの追加仮定 → d = 2。

Tarski の不定義性定理 (1933): 形式的に整合な体系は自身の真理述語を定義できない。
→ Object/Meta の区別は論理的に不可避。

> **日常の記述では Object↔Meta は暗黙に平坦化される (モナドの μ)。
> しかし自己参照的記述 (引用、注釈、メタアナリシス) では μ が破綻し、
> Order が独立基底として顕在化する。**

#### 棄却された候補

| 候補 | 棄却理由 |
|:-----|:--------|
| Quality (精度↔基準) | 相補的対立ではない (カテゴリ錯誤) |
| Resolution (精確↔概略) | Image の測度 (Rate-Distortion の rate 側) |
| Form/Manner | Opposition 不安定。パラメータ β として扱う |
| Valence (肯定↔否定) | 記述に身体的情動方向なし |

#### Form/Manner: 基底ではなくパラメータ

RSA, Relevance Theory, Grice の Manner maxim が収束的に識別する次元。
しかし Rate-Distortion の rate 側 = Image の圧縮強度。
**精度重み β = 全基底を横断するパラメータ。** 次元ではなく温度。

> **HGK では Precision は座標 (d=2)。Typos では座標に昇格しない。
> この非対称性は「娘は母と同型ではない」ことの具体的な現れ。**

## 4. 深度レベルシステム — 全成分を常に使う必要はない

> **毎回 /ax を発動する妥当性がないのと同じ。** — Creator

24成分は常に全て活性化する必要はない。記述の厳密性に応じて活性化する基底を選択する:

| Level | 活性基底 | 活性成分数 | 用途 |
|:------|:---------|:----------|:-----|
| **L0** | Image (暗黙) | 0 (暗黙) | 日常会話、メモ |
| **L1** | + Endpoint, Reason | 4 (E×R) | メール、チャット |
| **L2** | + Context | 12 (E×R + E×C + R×C) | レポート、企画書 |
| **L3** | + Order | 24 (全成分) | 学術論文、法律文書、仕様書 |

> **HGK の L0-L3 と構造的に対応する。**
> 使用頻度の不均一性は基底の存在を否定しない。
> 日常的に A-series 定理を使わないからといって A-series が不要ではないように、
> ANNOTATION や METHOD が日常記述で空虚でも、学術記述では不可欠。

A0 (不可逆圧縮) + Rate-Distortion 理論:

```
R(D) = min I(X;Y)  subject to  E[d(x,y)] ≤ D
```

歪みコストの高い基底を先に保存する:

| 基底 | 歪みコスト | 理由 |
|:-----|:----------|:-----|
| **Image** | 極大 | 射そのもの。省略不可 |
| **Endpoint** | 高 | 始域・終域なしに射を解釈できない |
| **Reason** | 中 | 文脈から推測可能な場合がある |
| **Context** | 低 | デフォルト圏で代替可 |
| **Order** | 最低 | デフォルト object-level で代替 |

**最小の記述 = Image のみ。** (事実の陳述)
**完全な記述 = 全5基底 × 全テンソル積成分。** (学術論文)

## 6. パラメータ β — 圧縮の温度

基底とは別に、射 f: M→L には圧縮の強度を制御するパラメータがある:

```
β ∈ [0, ∞)  — 精度重み付け
β → 0: 極限圧縮（文字数最小、情報損失最大）
β → ∞: 忠実再現（冗長、情報損失最小）
```

- β は全基底を横断して作用する
- RSA の cost function C(u)、Relevance Theory の processing effort、Grice の Manner に対応
- **HGK では Precision が座標 (d=2)。Typos では β はパラメータであり基底ではない**
- この非対称性: 認知には「確実さの度合い」が独立次元だが、記述には形式的な「確実さ」はない

## 7. 反証条件

### 近接反証（実験可能）

1. **成分インスタンス化テスト**: 24成分のうち、どの記述ジャンルでもインスタンス化できない成分があれば、
   その成分を生むペアのテンソル積を再検討
2. **基底独立性テスト**: ある基底を除去しても24成分が実質的に変わらなければ、
   その基底は他の基底に吸収されている
3. **深度レベルテスト**: L1 の記述に L3 成分を追加して品質が変わらなければ、
   深度レベルの区分を修正

### 遠距離反証（理論的）

1. 5基底で網羅できない情報消失が見つかれば、基底を追加
2. テンソル積で生成できない成分が発見された場合
3. Order がモナドの μ で完全に吸収可能であることが証明された場合 → 4基底に戻す
4. LLM が暗黙文脈を完全に推測できるようになったら、Týpos は不要になる
