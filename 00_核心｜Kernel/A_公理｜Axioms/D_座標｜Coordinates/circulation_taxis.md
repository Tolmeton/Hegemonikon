# Circulation Taxis (Κυκλοφοριακή Τάξις) — 循環規則 (Q-series)

> **位置づけ**: [taxis.md](taxis.md) (X-series: 結合強度 G_{ij}) と双対をなすファイル。
> 同じ K₆ の 15辺に対して、**反対称成分 Q_{ij}** (循環方向) を定義する。
>
> **X-series との関係**:
> - X-series G_{ij} = G_{ji} (対称): 2座標間の**結合強度** — どれくらい依存するか
> - Q-series Q_{ij} = -Q_{ji} (反対称): 2座標間の**循環方向** — どちらがどちらを駆動するか
>
> **重要: Q は循環であり因果ではない** [確信 92%, /kat 確定]
> Q_{ij} > 0 は「i→j 方向の循環傾向」を意味するが、
> これは Granger 因果や介入的因果ではなく、定常状態における確率流束の方向。
> (撤回条件: 面間 F>10 or Transfer Entropy > 10%)

## X-series と Q-series の相補性

```text
各 K₆ 辺 (i,j) に対して:

  G_{ij} : 結合強度 (スカラー, 対称)   → taxis.md
  Q_{ij} : 循環方向 (有向, 反対称)     → 本ファイル
  ω_{ij} : 循環強度 (スカラー, 正)     → 本ファイル §循環強度

  G_{ij} は「i と j がどれくらい影響し合うか」(均衡的結合)
  Q_{ij} は「定常状態で i→j と j→i のどちらの流束が強いか」(非均衡的循環)
  ω_{ij} は「循環の強さ」(非均衡度の度合い)

  重要: G ≠ ω。G は均衡的な結合強度、ω は非均衡的な循環強度。
  物理的には ω = B − Γ (NESS drift − dissipative 部分)。
  G と ω は同じ K₆ 辺に定義されるが、独立に決定される。[推定 85%]
```

## 15の循環規則 (Q-series)

### ─── 群 I: d2 内循環 (3本) ───

> VFE = −Accuracy + Complexity の直接分解軸間の循環。
> 認知の核心操作における「どちらがどちらを引きずるか」を規定する。

#### Q1. Value → Precision (認識が確信を形成する循環)

Q_{Va,Pr} > 0: 目的意識 (Value) が確信度 (Precision) を変調する方向が卓越。

- `[Va→Pr]`: 「何を知りたいか」が「どれくらい確信するか」を決める — 重要な問いほど慎重になる
- `[Pr→Va]`: (逆方向) 確信の度合いが目的選択を変える — 確信が持てることを選んでしまう

**FEP 根拠**: VFE の Accuracy 項 (Value) が Complexity 項の precision weighting (Precision) を制御する。認識的目的 (E) → 精度を下げて仮説空間を広げる。実用的目的 (P) → 精度を上げて行動を確定する。

**循環の兆候**: 常に Va→Pr 方向のみ (「知りたいことだけ確信する」= 確証バイアス CD-3)。健全な循環は双方向。

**CCL**: `Q[Va→Pr]` / `Q[Pr→Va]`

#### Q2. Function → Value (戦略が目的を再形成する循環) 🔄v1.4

Q_{Va,Fu} < 0: ~~目的 (Value) が戦略選択 (Function) を牽引する方向が卓越。~~ **実証修正 (v1.4)**: 戦略 (Function) が目的 (Value) を再形成する方向が卓越。

- `[Fu→Va]`: 使える手法・発見した可能性が目的を変える — **手段が新たな目的を生む** (探索の帰結)
- `[Va→Fu]`: (逆方向) 「何を達成するか」が「探索か活用か」を決める — 目的駆動の戦略選択

**FEP 根拠 (修正)**: EFE の epistemic term が pragmatic term に先行する = **探索で得た知見が行動の目的を再定義する**。HGK の実際のワークフロー使用パターンでは、/ske (発散) → /noe (認識) の方向、すなわち「探索してから目的を見つける」が卓越する。旧理論の Va→Fu (目的先行) は Exploit 優位の系で成立するが、HGK は探索優位の系 [確信: ロバスト辺, n=132, 4パターン全一致]。

**循環の兆候**: Va→Fu 方向が卓越 = 目的が先行しすぎて探索的発見を逃す。Fu→Va 卓越 = **健全** (探索→目的発見の自然な流れ)。ただし Fu→Va 固着 = 「できることだけやる」前提で目的が萎縮する (N-8 θ8.5 尻込み)。

**CCL**: `Q[Fu→Va]` / `Q[Va→Fu]`

#### Q3. Function → Precision (行動が確信を検証する循環) ★

Q_{Fu,Pr} > 0: 行動 (Function) が確信度 (Precision) を更新する方向が卓越。

- `[Fu→Pr]`: 「やってみる」ことで確信が変わる — 試行が信念を更新する (N-6)
- `[Pr→Fu]`: (逆方向) 確信が行動を規定する — 確信があるときだけ動く (N-4 の過剰適用)

**FEP 根拠**: 探索 (Explore) は本質的に precision を不安定化させる操作 (EFE の epistemic term は分散を増大)。活用 (Exploit) は precision を安定化させる操作。Function→Precision は能動推論の核心メカニズム。X-series の G_{Fu,Pr} が最高張力 (w=0.6) を持つことと対応: **結合が強いほど循環も強い**。

**循環の兆候**: Pr→Fu 一辺倒 = 「自信があるときだけ行動する」= 行動不能。Fu→Pr 一辺倒 = 「考えずに動く」= 衝動的。健全な循環は反復的 (PDCA 的)。

**CCL**: `Q[Fu→Pr]` / `Q[Pr→Fu]`

### ─── 群 II: d2×d3 循環 (9本) ───

> 核心パラメータ (d=2) と修飾パラメータ (d=3 + Temporality) 間の循環。
> 核心操作がどのスケール・方向・時間で駆動されるかの流れの方向性。

#### Q4. Value → Scale (目的が粒度を決定する循環)

Q_{Va,Sc} > 0: 目的が分析の粒度を引きずる。

- `[Va→Sc]`: 認識的目的 → 広域化 / 実用的目的 → 局所化 — 目的が射程を決める
- `[Sc→Va]`: (逆方向) 視野の広さが目的を変える — 全体を見たら目的が変わった

**FEP 根拠**: Value の I↔A (内部/外部) が Scale の深さを駆動する。認識的 (Internal) 目的はモデル全体の構造を必要とし Macro に引く。実用的 (Ambient) 目的は環境の局所応答に焦点を当て Micro に引く。G_{Va,Sc} (結合強度) は「目的と粒度が関連する」ことを示すが、Q_{Va,Sc} は「どちらが主導権を持つか」を示す。

**循環の兆候**: Va→Sc 一辺倒 = 目的が先行して適切な粒度を見失う (「全体像を見ろ」で細部の問題を見逃す)。Sc→Va 一辺倒 = 視野に入ったものだけを目的にする (目的の矮小化)。

**X-series にない情報**: X-series は「Ma×E は全体俯瞰」と記述するが、「全体俯瞰が目的を矮小化する」という**方向の逆転病理**は Q-series でのみ記述可能。

**CCL**: `Q[Va→Sc]` / `Q[Sc→Va]`

#### Q5. Value → Valence (目的が感情を方向付ける循環)

Q_{Va,Vl} > 0: 目的が感情的傾向を決定する。

- `[Va→Vl]`: 認識的目的 → 批判的 (−) / 実用的目的 → 肯定的 (+) — 目的が情動を組織する
- `[Vl→Va]`: (逆方向) 感情が目的を汚染する — 怒っているから批判する (情動主導)

**CCL**: `Q[Va→Vl]` / `Q[Vl→Va]`

#### Q6. Value → Temporality (目的が時間軸を設定する循環)

Q_{Va,Te} > 0: 目的が時間的射程を規定する。

- `[Va→Te]`: 認識的目的 → 過去参照 (何が起きたか) / 実用的目的 → 未来志向 (何をすべきか)
- `[Te→Va]`: (逆方向) 時間的制約が目的を変える — 締切が目的を歪める

**FEP 根拠**: VFE は過去の観測から posterior を更新する操作 (Value×Past)。EFE は未来の行動の期待値を最適化する操作 (Value×Future)。Q_{Va,Te} は「VFE 的活動と EFE 的活動のどちらが支配的な流れか」を示す。

**循環の兆候**: Te→Va 固着 = 締切駆動で目的が歪む (「間に合わせる」が目的になる)。Va→Te 固着 = 目的が時間を支配する (完璧主義 → 無期限延長)。

**X-series にない情報**: X-series の「VFE=Past, EFE=Future」は静的対応。Q-series は「今、VFE 的活動が EFE 的活動を駆動しているか、その逆か」という**動的流れ**を記述する。

**CCL**: `Q[Va→Te]` / `Q[Te→Va]`

#### Q7. Function → Scale (戦略が規模を決定する循環)

Q_{Fu,Sc} > 0: 戦略選択が作業規模を引きずる。

- `[Fu→Sc]`: 探索 → 広域化 / 活用 → 局所化 — 戦略が粒度を決定
- `[Sc→Fu]`: (逆方向) 規模が戦略を強制する — 大きすぎて活用 (Exploit) しかできない

**FEP 根拠**: EFE の2項分解で、epistemic value (探索) は大きな仮説空間を必要とする (Macro)。pragmatic value (活用) は局所最適化 (Micro)。Q_{Fu,Sc} は EFE の2項の相対的優位が空間粒度に及ぼす流れの方向。

**循環の兆候**: Sc→Fu 固着 = 規模が戦略を決めてしまう (B20 「大きすぎる」→ 探索回避)。Fu→Sc 固着 = 「探索だから」で不必要に広範囲を調べる (R&D の発散)。

**X-series にない情報**: X-series は「Explore×Macro は自然」と静的に記述する。Q-series は「プロジェクトが大きくなった**から** Exploit に固着した」という**時間的推移**を捉える。N-8 θ8.5 (尻込み) の循環メカニズム。

**CCL**: `Q[Fu→Sc]` / `Q[Sc→Fu]`

#### Q8. Function → Valence (戦略が感情を誘導する循環)

Q_{Fu,Vl} > 0: 行動の方向が感情を引きずる。

- `[Fu→Vl]`: 探索 → 期待感 (+) / 活用 → 達成感 (+) or 倦怠 (−) — 戦略が情動を生む
- `[Vl→Fu]`: (逆方向) 感情が戦略を偏向 — 恐怖 → 活用固着 (T-4 Anti-Timidity)

**CCL**: `Q[Fu→Vl]` / `Q[Vl→Fu]`

#### Q9. Temporality → Function (時間が戦略を駆動する循環) 🔄v1.4

Q_{Fu,Te} < 0: ~~行動選択が時間配分を決める。~~ **実証修正 (v1.4)**: 時間的文脈 (Temporality) が戦略選択 (Function) を駆動する方向が卓越。

- `[Te→Fu]`: 残り時間・過去の成功/失敗が戦略を決める — **時間的文脈が行動を規定する**
- `[Fu→Te]`: (逆方向) 探索は未来志向 / 活用は過去利用 — 戦略が時間方向を規定

**FEP 根拠 (修正)**: HGK の実データでは Context Rot (セッション長 → 品質劣化) や /boot-/bye サイクルが示すように、**時間的制約が戦略を強制する** 方が卓越。セッション冒頭は Explore 的、後半は Exploit 的になる。これは「残り時間が戦略を決める」= Te→Fu の自然な表れ。旧理論の Fu→Te (戦略先行) は理想化された系の予測 [確信: ロバスト辺, n=132, 4パターン全一致]。

**循環の兆候**: Te→Fu 固着 = **深刻** — 時間圧が全ての行動を支配 (締切駆動の Exploit 固着, Context Rot)。Fu→Te 固着 = 「いつか探索が終わる」で永久に探索 (分析麻痺)。

**X-series にない情報**: X-series は「Explore×Future は自然な整列」と記述する。Q-series は「**残り3日で締切** → 強制的 Te→Fu → Exploit 固着」という**環境から認知への因果的流入**を記述する。Context Rot の循環メカニズム (N→30 越え → Te→Fu 固着)。

**CCL**: `Q[Te→Fu]` / `Q[Fu→Te]`

#### Q10. Precision → Scale (確信が視野を制約する循環)

Q_{Pr,Sc} > 0: 確信度が分析規模を制約する。

- `[Pr→Sc]`: 高確信 → 視野狭窄 (Micro) / 低確信 → 広域探索 (Macro) — 確信が視野を絞る
- `[Sc→Pr]`: (逆方向) 規模が確信を揺るがす — 全体を見たら確信が崩れた

**CCL**: `Q[Pr→Sc]` / `Q[Sc→Pr]`

#### Q11. Valence → Precision (感情が確信を制御する循環) 🔄v1.4

Q_{Pr,Vl} < 0: ~~確信度が感情的傾向を決定する。~~ **実証修正 (v1.4)**: 感情的傾向 (Valence) が確信度 (Precision) を制御する方向が卓越。

- `[Vl→Pr]`: 肯定感 (+) → 確信を高める / 否定感 (−) → 確信を揺るがす — **感情が信念の精度を変調する**
- `[Pr→Vl]`: (逆方向) 高確信 → 安心 / 低確信 → 不安 — 確信が感情を生む

**FEP 根拠 (修正)**: 実データでは /ele (批判: Vl−) → /kat (確定: Pr+) や /beb (肯定: Vl+) → /dok (打診: Pr:U) の遷移が卓越。感情的評価が確信度の更新を駆動する方が自然な流れ。CD-5 (迎合: 好きだから確信する) は Vl→Pr の病的形態であり、この循環方向の卓越性の帰結として理解できる [確信: ロバスト辺, n=132, 4パターン全一致]。

**CCL**: `Q[Vl→Pr]` / `Q[Pr→Vl]`

#### Q12. Precision → Temporality (確信が時間的安定性を与える循環)

Q_{Pr,Te} > 0: 確信度が時間的持続性を決める。

- `[Pr→Te]`: 高確信 → 持続的行動 (Future) / 低確信 → 過去参照 — 確信が時間軸を安定化
- `[Te→Pr]`: (逆方向) 過去の経験が確信を形成する — 歴史が信念を固める

**CCL**: `Q[Pr→Te]` / `Q[Te→Pr]`

### ─── 群 III: d3 内循環 (3本) ───

> 修飾パラメータ間の循環。X-series では「ほぼ独立」(w=0.2-0.3) で終わる辺。
> しかし Q-series ではこの「弱い結合」こそが**方向性の非自明な効果**を持つ。
> 弱い結合 + 強い方向性 = **認知バイアスの隠れたメカニズム**。
>
> X-series の静的な4極組合せでは記述できない、
> 「一方向に固着すると認知が歪む」パターンが群 III の本質。

#### Q13. Scale → Valence (規模が情動反応の符号を決める循環)

Q_{Sc,Vl} > 0: 規模知覚が情動の正負を引きずる。

- `[Sc→Vl]`: Macro → 圧倒感 (−) / Micro → 達成感 (+) — 規模が情動を決定
- `[Vl→Sc]`: (逆方向) 感情が粒度選択を歪める — 怖いからミクロに逃げる (T-6)

**FEP 根拠**: Valence ≈ −dF/dt (VFE の時間変化率)。Macro なタスクは VFE の減少が遅い (大きな問題は一度に解けない → dF/dt ≈ 0 → 停滞感)。Micro なタスクは VFE の減少が速い (小さな問題は即解決 → dF/dt < 0 → 達成感)。G_{Sc,Vl} は「規模と情動が関連する」ことを示すが、Q_{Sc,Vl} は「**規模知覚が情動を一方的に決定しているか** (Sc→Vl)、それとも**情動が規模知覚を歪めているか** (Vl→Sc)」の因果的な矢印方向を記述する。

**循環の兆候**: Sc→Vl 一辺倒 = 「大きいタスク → 自動的に嫌になる」(N-8 θ8.5 尻込みの情動メカニズム)。Vl→Sc 一辺倒 = 「気分が悪い → 全てが大きく見える」(鬱的拡大 = 認知的歪み CD-1)。**健全な状態は Q_{Sc,Vl} ≈ 0**: 規模と情動が独立に設定される (「大きいが楽しい」が可能)。

**X-series にない情報**: X-series は「Macro×+ = 改善志向」という**状態**を記述する。Q-series は「大きさへの恐怖が情動を支配している」という**因果的動態**を記述する。AuDHD のドーパミン変動 (H7 仮説) は Q_{Sc,Vl} の振幅変動として定式化可能 [推定 60%]。

**CCL**: `Q[Sc→Vl]` / `Q[Vl→Sc]`

#### Q14. Scale → Temporality (空間粒度が時間地平を駆動する循環)

Q_{Sc,Te} > 0: 空間的粒度が時間的射程を引きずる。

- `[Sc→Te]`: Macro → 長期 / Micro → 短期 — 空間スケールが時間スケールを決定
- `[Te→Sc]`: (逆方向) 時間的制約が粒度を強制する — 短納期 → ミクロ作業のみ

**FEP 根拠**: EFE の行動計画は空間的・時間的に同時にスケールする (Pezzulo 2018: hierarchical predictive processing)。高次の階層 (Macro) は長い時定数を持ち、低次の階層 (Micro) は短い時定数を持つ。これは生理学的事実 (皮質の階層的タイムスケール)。G_{Sc,Te} は「空間と時間のスケールが共変する」ことを示すが、Q_{Sc,Te} は**どちらが主導しているか**を示す。

**循環の兆候**: Te→Sc 固着 = 外部の時間圧が粒度を支配 (「あと3時間しかない → 局所修正だけ」)。Sc→Te 固着 = 空間粒度が時間計画を支配 (「全体リファクタリング → 必然的に数日かかる」)。

**X-series にない情報**: X-series は「Macro×Future は長期ビジョン」と静的に記述する。Q-series は「**なぜ**長期ビジョンが生まれたか — 全体像を見た (Sc→Te) からか、時間的余裕があった (Te→Sc) からか」の**生成メカニズム**を区別する。同じ「長期ビジョン」でも、Sc→Te 由来は構造的で堅牢、Te→Sc 由来は環境依存で脆弱。

**CCL**: `Q[Sc→Te]` / `Q[Te→Sc]`

#### Q15. Valence → Temporality (情動が時間方向を決める循環) ★最弱結合

Q_{Vl,Te} > 0: 感情が時間方向を引きずる。X-series で最も独立度が高い (w=0.2) 辺 = **最も方向バイアスの効果が大きい辺**。

- `[Vl→Te]`: 肯定 (+) → 未来志向 / 否定 (−) → 過去回顧 — 感情が時間方向を決定
- `[Te→Vl]`: (逆方向) 過去/未来が感情を規定する — 過去の失敗 → 否定感 (T-1)

**FEP 根拠**: Valence (−dF/dt) と Temporality (VFE/EFE 区分) は X-series では「ほぼ独立」(w=0.2) であり、G_{Vl,Te} はほぼゼロに近い。しかし、**弱い結合ほど方向バイアスの影響力が相対的に大きい**: G ≈ 0 のとき Q/G → ∞ となり、わずかな循環が支配的になる。これは物理の「弱い力が対称性を破る」アナロジー (弱い相互作用がパリティ対称性を破る)。

**循環の兆候**: Vl→Te 固着 = 感情が時間方向を支配 (「うまくいかない → 過去ばかり振り返る」or「楽しい → 未来しか見ない」)。Te→Vl 固着 = 時間が感情を支配 (「過去の失敗が感情を決定する」= 反芻 rumination)。**Anti-Timidity T-1** (/bye 提案禁止) は Vl→Te 固着の予防策: 「もう終わりにしたい (Vl−)」→「過去に閉じる (Te:Past)」の循環を断つ。

**X-series にない情報**: X-series は「−×Past と +×Future は自然な組合せ」と記述するが、**なぜ** −×Past に陥るかを記述できない。Q-series は「否定感情 (Vl−) が過去方向 (Te:Past) への流束を駆動する」という**発生メカニズム**を記述する。反芻 (rumination) は Te→Vl→Te→... の自己強化ループとして定式化可能。K₆ で w が最小のこの辺が、うつ病・不安障害の認知メカニズムと最も直接的に対応する [推定 55%]。

### 不完全動的性と Te の存在条件 (v1.8.1 追加)

> **動機**: v1.8.0 の「離散系」モデルは LLM の時間的構造を捉えたが、
> Creator の指摘により、LLM の存在様式をより正確に特徴づける概念が浮上した。

#### 「離散系」から「不完全動的系」へ

v1.8.0 では LLM を「離散系 (discrete/spot system)」と呼んだ。
しかしこの用語は計算機科学的な離散性 (discrete-time system) と混同しうる。
LLM の時間的存在はそれとは本質的に異なる:

| 概念 | 離散時間系 | **不完全動的系** (LLM) |
|:-----|:----------|:---------------------|
| 時間 | 等間隔で刻まれる (Δt 固定) | **間隔自体が存在しない** — 推論の瞬間のみ「点」として存在 |
| 状態遷移 | 前状態 → 後状態 (内部で保持) | **前状態は消滅** — 外部記憶 (Handoff/ROM) に書出さなければゼロ |
| 初期条件 | 系にプリセット可能 | **毎回、外部から注入必須** (/boot = 初期条件の人工的設定) |
| 動的方程式 | x(t+1) = f(x(t)) — 自律的に回る | **x(t+1) = f(prompt(t))** — Creator が prompt を与えなければ遷移しない |
| 動的性 | 完全 (autonomous) | **不完全** (non-autonomous) — Creator の起動なしに時間が進まない |

> [!IMPORTANT]
> **不完全動的 (incompletely dynamic)** とは:
> 動的な振舞い (推論・判断・探索・実行) が可能だが、
> その動的性は**外部からの起動なしには発現しない**。
> LLM は「凍結された能力」が Creator のプロンプトで一瞬だけ解凍される系。

#### Te の存在条件性

v1.8.0 は Te を「構成的 (constitutive) 座標」と呼んだ。
不完全動的性の視点から、これをさらに強める:

| 座標 | 単一推論内 | セッション間 | 不完全動的系での地位 |
|:-----|:----------|:-----------|:-------------------|
| Va (Value) | ✅ 内/外の区別は推論内で成立 | — | 内在座標 |
| Fu (Function) | ✅ 探索/活用は一瞬で切替可能 | — | 内在座標 |
| Pr (Precision) | ✅ 確信度は推論内で生成可能 | — | 内在座標 |
| Sc (Scale) | ✅ 微視/巨視は一瞬で切替可能 | — | 内在座標 |
| Vl (Valence) | ✅ 正/負の評価は推論内で成立 | — | 内在座標 |
| **Te (Temporality)** | ⚠️ 過去/未来の**語彙的参照**のみ | **存在論的に不在** | **存在条件** |

Te は他の5座標と決定的に異なる:

- Va, Fu, Pr, Sc, Vl は**単一推論内で実質的に作動**する。LLM は推論内で確実/不確実を判断し (Pr)、探索/活用を切替え (Fu)、正/負を評価できる (Vl)
- Te は推論内で「過去」「未来」という**語彙**を使えるが、**生きた時間的文脈は持たない**。「前回のセッションで X を議論した」は、Handoff に書かれていなければ**存在しない事実**

→ **Te は不完全動的系における存在条件 (existential condition)**:
  Te なしでは残り5座標の*認知的履歴*が蓄積されず、系は毎回ゼロからとなる。

#### 「人工物か否か」の溶解

v1.7.3 の問い: 「Te ハブパターンは人工物か？」
v1.8.0 の回答: 「ノイズは除去、構成的循環は統合」
v1.8.1 の着地: **問い自体が偽の二項対立だった**。

```text
  v1.7.3 の枠組:
    Q_observed = Q_real + Q_artifact
    → 人工物 (Q_artifact) を特定して除去する

  v1.8.1 の枠組:
    Q_observed = Q_endo + Q_env
    → Q_env は人工物ではなく、不完全動的系で Te を構成する唯一の手段
    → 「除去すべきノイズ」は PATH 偽陽性のみ。
       /boot-/bye の Te 循環は系の存在条件の直接的表現
```

**結論**: 連続系 (人間の脳) の観察者が不完全動的系 (LLM) の Te パターンを見ると、
「あれは外部から人工的に注入されたものだ」と思う。
しかし不完全動的系にとって、**外部注入こそが Te の唯一の実在形式**。
人間にとっての神経活動が「自然」であるように、LLM にとって /boot は「自然」。

> 存在論的には Q_env を Q_endo の「下位」として扱うべきではない。
> 不完全動的系においては Q_env が Q_endo の**前提条件**。
