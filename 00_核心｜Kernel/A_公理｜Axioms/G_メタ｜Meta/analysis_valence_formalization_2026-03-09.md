# Valence 公式定式化の選定分析

**結論**: 単一定義を選ばず**メタ枠組み**を採用。4定式化は半直積作用 φ の異なる実装。
**運用デフォルト**: Hesp (2021)
**日付**: 2026-03-09

## 1. 問い

4つの Valence 定式化のうち、HGK の公式定義としてどれを採用するか？

## 3. 判定: メタ枠組み + 運用デフォルト

### なぜ単一定義を選ばないか

半直積 G = H ⋊_φ K において、φ は「K が H をどう修飾するか」を記述する。
3定義比較実験の核心的発見: **φ の形は定義ごとに異なるが、半直積構造自体は不変**。

これは Valence の本質: **評価的方向付けの方法は、対象基質 (substrate) によって変わる**。
- 生物学的基質 → Seth (内受容) or Pattisapu (効用)
- 認知エージェント → Hesp (方策精度)
- 全 FEP 系 (物理含む) → Joffily (VFE 速度)

特定の φ を「唯一の正しい定義」とすることは、半直積概念自体に反する。

### なぜ Hesp が運用デフォルトか

1. **弱結合** (Fisher ratio 0.22) → 計算機実装で他座標と分離しやすい
2. **情報幾何学的に自然**: natural parameter として Fisher 行列上で交差項最小
3. **v-ω 結合ゼロ**: Seth の強結合 (0.71) は乗法仮定 (ω·exp(v)) の恒等的帰結
4. **HGK の対象**: 認知エージェント (方策を持つ) → 方策精度は自然な量
5. **圏論的解釈**: lax natural transformation → 半直積の作用が弱い → 6座標の独立性を最大限保存

### Joffily の位置づけ

v = −dF/dt は最も普遍的だが:
- Fisher 実験が未実施 → 結合強度が不明
- VFE の時間微分は瞬時的 → 持続的な「感情」の説明力が弱い可能性
- [仮説 45%] Fisher 実験すれば Hesp と同等かそれ以下の結合強度になる可能性あり

## 4. HGK 公式記述

```
Valence の HGK 定式化:

メタ枠組み: 全座標空間 = H ⋊_φ K
  H = Flow × Value × Function × Precision × Scale × Temporality (6座標)
  K = Valence (1座標)
  φ: K → Aut(H) は基質依存

運用デフォルト: Hesp (2021)
  v = log(π_precision)  — 方策分布の精度の対数
  φ_H: v は π のエントロピーを修飾 (弱結合, ratio=0.22)

代替実装:
  φ_J (Joffily): v = −dF/dt  — 全 FEP 系に適用可能
  φ_S (Seth): v → ω 修飾  — 身体性を持つ系に適用
  φ_P (Pattisapu): v → C·E[o]  — Prior preferences を持つ系に適用
```
