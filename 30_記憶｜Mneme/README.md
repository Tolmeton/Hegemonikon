# 30_記憶｜Mneme — 経験記録系

> **FEP 演繹**: Mneme はベイズ更新の「結果格納庫」。
> 観測を経て確定した信念 Q(s|o) と、未処理の生観測 o を扱う帯。

---

## 分類基準: POMDP 変数マッピング

**唯一の問い: 「このファイルは Q(s|o) か o か？」**

| 帯 | POMDP 変数 | 判定基準 |
|:---|:-----------|:---------|
| `00_信念｜Beliefs` | **P(s)** — prior 信念 | Mneme 内の固定 prior。Doxa など変わらない信念 |
| `01_記録｜Records` | **Q(s|o)** — 事後信念 | 観測後に確定した記録全般 |
| `02_索引｜Index` | **P(o\|s)** — 尤度モデル | 記録を検索・照合するメカニズム |
| `03_素材｜Materials` | **o** — 生観測 | 未処理の外部入力。消化 (/eat) 前の状態 |
| `04_知識｜Gnosis` | **Q(s\|o)** — 確定知識 | 論文・文献など精査済みの学術知識 |
| `05_状態｜State` | **(ランタイム)** | POMDP の外側。実行中のシステム状態 |

---

## 01_記録 の内部構造

| サブ帯 | 内容 | Q(s\|o) の種別 |
|:-------|:-----|:--------------|
| `a_引継｜handoff` | セッション引き継ぎ文書 | **推移確率 T** — 状態遷移の記録 |
| `b_対話_sessions` | 対話ログ | **観測履歴** |
| `c_ROM｜rom` | コンテキスト蒸留ファイル | **圧縮された後験** |
| `c_成果_artifacts` | WF 成果物・生成文書 | **確定した出力** |
| `d_成果物｜artifacts` | セッション別出力・成果物 | **確定した出力** |
| `e_ログ_logs` | システムログ | **観測の痕跡** |
| `f_実行痕跡_traces` | 実行テープ (tape_*.jsonl) | **行動の痕跡** |
| `z_旧構造_legacy` | 廃止された旧構造 | **Q_{t-1}** |

---

## 境界ケースの判定

| ケース | 判定 |
|:-------|:-----|
| Handoff → 01_記録 か 04_知識？ | Q(s\|o): 観測後の記録 → **01_記録** |
| 論文 → 03_素材 か 04_知識？ | 消化前=o → **03_素材** / 消化後=Q(s\|o) → **04_知識** |
| Doxa → 00_信念 か 01_記録？ | P(s): 変わらない prior → **00_信念** |
| ランタイム状態 → どこ？ | POMDP 外 → **05_状態** |
