## 1. 問題の定式化: なぜ静的構造から速度が語れるのか

### 1.1 亀裂の所在

Paper VIII §6.2 の α-filtration:

> {C_α}_{α∈[0,1]}, (F1) Ob(C_α) = Ob(C), (F2) C_0 = C, (F3) C_1 = C_disc, (F4) α₁≤α₂ ⟹ Mor(C_{α₂}) ⊆ Mor(C_{α₁})

これは **パラメータ α で添字づけられた圏の族** であり、時間 t を含まない。空間 x も含まない。

一方、Paper XII §3 は前線速度を:

$$v_{\text{null}} = -\frac{\partial_t \Delta_t(x)}{|\nabla_x \Delta_t(x)|}$$

と定義する。これは明示的に時間微分 ∂_t と空間勾配 ∇_x を含む。

**問い: 静的な族 {C_α} から、どうやって ∂_t と ∇_x が出てくるのか？**

### 1.2 既存の部分的回答

この問いに対し、忘却論の各論文は部分的な回答を既に持っている:

| 論文 | α に何を加えたか | 得られたもの | 足りないもの |
|:--|:--|:--|:--|
| Paper I §6 | α(θ): 統計多様体 M 上の空間場 | 空間的変動 ∂_θ α | 時間 t |
| Paper V §2.3 | α(θ,μ): スケール依存性 | RG 時間 μ による走り β_α | 物理時間 t、空間 x |
| Paper I §6.5.1 L699 | α(θ,t) on M×ℝ (構想のみ) | 時間拡張の必要性の認識 | 実際の構成 (Paper II に延期) |
| Paper XII §2.1 | θ → x (probe space への移行) | 前線追跡の空間座標 | α との接続 |

**いずれの論文も、α(θ) と probe space X と時間 t を統合した構成を完了していない。** この統合が Paper XII の証明義務の、表に書かれていない最も根本的な項目である。

## 2. α場橋渡しの構成

### 2.1 三つの写像

**定義 2.1 (局所統計モデル埋込).** probe space X の各点 x に対し、局所統計モデルを対応させる埋込

$$\theta: X \to M, \quad x \mapsto \theta(x)$$

が存在する。ここで M は Paper I の統計多様体。

- 光学: x ∈ ℝ² は物理位置。θ(x) = (振幅パラメータ, 位相パラメータ) at x
- LLM: x は token 位置。θ(x) = (attention 分布のパラメータ) at x
- Hyphē: x は memory slot。θ(x) = (記憶の統計的表現) at x

**定義 2.2 (有効観測スケール).** 各 (x,t) に対し、局所的な有効観測スケール

$$\mu: X \times \mathbb{R} \to \mathbb{R}_{>0}, \quad (x,t) \mapsto \mu(x,t)$$

が存在する。これは Paper V の RG パラメータの時空への埋込である。

- 光学: μ(x,t) は局所的な波動場の有効波長
- LLM: μ(x,t) は attention window の有効幅
- Hyphē: μ(x,t) は retrieval 解像度

**定義 2.3 (有効忘却強度場).** 上記を合成した

$$\alpha_{\text{eff}}: X \times \mathbb{R} \to [0,1], \quad \alpha_{\text{eff}}(x,t) := \Psi\big(\alpha(\theta(x), \mu(x,t)), \, t\big)$$

を **有効忘却強度場** (effective oblivion strength field) と呼ぶ。ここで Ψ は α の値を [0,1] に正規化する写像であり、Paper I の α ∈ ℝ (Amari パラメータ) を Paper VIII の α ∈ [0,1] (濾過パラメータ) へ変換する。

[構造的対応 70%] — 写像 θ, μ, Ψ の存在は系ごとに仮定される。普遍的構成は未完。

### 2.2 動的圏束 (dynamic category bundle)

α_eff(x,t) を用いて、probe space 上の **局所圏の場** を定義する:

> 各 (x,t) ∈ X × ℝ に対し、局所圏 **C(x,t) := C_{α_eff(x,t)}** を α-filtration から読み出す。

これはファイバー束の構造をもつ:

| 成分 | 内容 | SOURCE |
|:--|:--|:--|
| 底空間 (base) | X × ℝ (probe space × 時間) | Paper XII §2.1 |
| ファイバー (fiber) | α-filtration 族 {C_α}_{α∈[0,1]} | Paper VIII §6.2 |
| 切断 (section) | α_eff: X × ℝ → [0,1] | Paper I §6 + Paper V §2.3 |
| 局所圏 | C(x,t) = C_{α_eff(x,t)} | **本構成 (新)** |

**α-filtration は静的なファイバーを提供し、α_eff は動的な切断を提供する。局所圏の時間発展は、切断の運動から生じる。**

[構造的対応 75%] — 厳密な圏束 (categorical bundle) の定式化には、ファイバーが圏であることに伴う 2-圏論的構造の処理が必要。ここでは 1-圏レベルで十分。

### 2.3 前線の発生機構

α_eff(x,t) が与えられると:

1. **可区別性場 Δ_t(x) は α_eff の関数として制約される。** α_eff が大きい点では Hom 集合が小さく (F4)、対象間の同定が粗くなり、Δ が低下する。形式的に:

$$\Delta_t(x) = \mathcal{F}[\alpha_{\text{eff}}(x,t); \text{local data}]$$

ここで F は、局所的な確率分布対と利用可能な射の構造から可区別性を計算する汎関数。α_eff が増大すれば F は減少する (F4 の帰結)。

2. **前線 B_t^ε は α_eff の等位面に従属する。** B_t^ε = {x : Δ_t(x) = ε} が α_eff の単調性を通じて近似的に等価:

$$B_t^\varepsilon \approx \{x : \alpha_{\text{eff}}(x,t) = \alpha_c(\varepsilon)\}$$

ここで α_c(ε) は Δ = ε を与える臨界忘却強度。

3. **前線速度は α_eff の等位面速度である。** 補題 4.1 (Paper XII §4.1) を α_eff に適用:

$$v_{\text{null}} \approx -\frac{\partial_t \alpha_{\text{eff}}}{|\nabla_x \alpha_{\text{eff}}|}\bigg|_{\alpha_{\text{eff}} = \alpha_c}$$

[予想 60%] — Δ が α_eff の単調関数であることは (F4) から強く示唆されるが、局所データ依存性 (distribution-specific な寄与) を経由するため、完全な単調性の証明は未了。

## 3. 「再配置」の正体: α-掃過

### 3.1 旧い言葉の問題

ROM 深化 (I) で使った「対象の幾何学的再配置」は二重に不正確だった:

1. **「対象の」**: 対象 (Ob) は (F1) により α-不変。動くのは対象ではない。
2. **「再配置」**: 何かが物理的に移動する印象を与える。実際には、何も移動しない。

### 3.2 正しい記述: α-掃過

α_eff(x,t) が時間とともに変化するとき、probe space 上で起きていることは:

> **各点 x において、局所圏 C(x,t) = C_{α_eff(x,t)} がファイバー {C_α} 内を掃過 (sweep) する。**

掃過の方向は α の増大方向。α が増大すると:
- 対象は不動 (F1)
- 射が消失 (F4)
- 可区別性が低下 (定理 6.2.3 (ii))
- 局所圏が離散圏に接近 (F3)

この掃過が probe space の異なる点 x で異なる速度で進行するとき、**α_c の等位面が空間上を移動する**。これが前線 B_t^ε の運動の正体である。

### 3.3 なぜ α-掃過は因果速度制限を受けないか

α_eff の等位面速度は:

$$v_{\text{level}} = -\frac{\partial_t \alpha_{\text{eff}}}{|\nabla_x \alpha_{\text{eff}}|}$$

これが大きくなるのは、**空間勾配 |∇_x α_eff| が小さいとき** — すなわち、忘却が空間的にほぼ一様に進行するとき。

物理的直観: 全ての点 x でほぼ同時に α が閾値を超えれば、等位面は「一瞬で」空間全体を掃く。これは因果信号が伝搬しているのではなく、**独立に進行していた局所的忘却が同時に閾値に達した** だけである。

これは Paper XII §1.3 のハサミの比喩と正確に対応する。ハサミの刃の交点が超光速で動けるのは、刃の各点が独立に運動しているからである。α_eff の各点 x での時間発展も同様に局所的——ただし、α_eff の PDEs (Paper I §6.2) を通じて空間的結合をもちうる。

**因果速度制限 c は射 (Mor) の伝搬を支配する。α_eff の等位面は Ob レベルの特徴であり、射の伝搬法則の管轄外にある。** これが ROM 深化 (I) §1 の主張の精密化であり、「再配置」を「α-掃過」に置き換えた結果である。

[構造的対応 80%] — 等位面速度が因果速度制限を受けないことは一般相対論の位相速度の議論と同型。「α_eff の等位面は Ob レベル」の主張は (F1) から強く支持される。

## 5. Paper V からの時間発展の供給

### 5.1 β_α が α に「時間」を与える

Paper V §2.3:

$$\beta_\alpha(\theta, \mu) := \mu \frac{\partial}{\partial\mu} \alpha(\theta, \mu)$$

RG 流のもとで α は「走る」。スケール μ を下げる (粗視化する) と α は変化する。

Paper V 定理 2.2.2 の β_Φ ≥ 0 の対応物として: 粗視化 (μ↓) は射の利用可能性を減少させるから、α は粗視化に伴い増大する。すなわち ∂α/∂μ ≤ 0 であり、**β_α = μ(∂α/∂μ) ≤ 0** が自然な制約。[ROM III §4 で CFC + (F4) から証明]

### 5.2 RG 時間と物理時間の接続

多くの認知系・情報処理系において、有効観測スケール μ は物理時間 t とともに変化する:

- **LLM**: context が蓄積すると attention の有効解像度が下がる → μ(t) は単調減少
- **記憶系**: 時間経過とともに記憶の精度が劣化する → μ(t) は単調減少
- **光学系**: 波動場が伝搬するにつれて分散が働く → μ(x,t) は空間・時間の関数

μ(t) が単調減少し β_α ≤ 0 ならば:

$$\frac{d}{dt} \alpha_{\text{eff}}(x,t) = \frac{\partial\alpha}{\partial\mu} \cdot \frac{d\mu}{dt} + \frac{\partial\alpha}{\partial t}\bigg|_{\text{explicit}}$$

第一項: β_α/μ × dμ/dt = (≤ 0)/μ × (≤ 0) ≥ 0。**α_eff は時間とともに単調非減少。** Ψ の特別な仕掛けは不要 — 符号が自然に整合する。

物理的には: 粗視化が進む (μ↓) と忘却濾過パラメータ α_eff は上昇する。時間が経つと忘却が深まる。

**帰結:** Paper V の RG 走りは、Paper XII の前線が動くための**駆動力**を供給する。前線が probe space を掃くのは、有効観測スケール μ(x,t) が時間とともに変化し、それに伴って α_eff(x,t) が変化するからである。前線伝搬は構造的に不可避。

[**定理 90%**] — β_α ≤ 0 は CFC + (F4) から証明される (ROM III §4)。DPI による β_Φ ≥ 0 と双対をなす。

## 6. OP-XII-1 (三場の非崩壊性) への帰結

### 6.1 Δ と C の α_eff 依存性の非対称性

α場橋渡しの最も重要な帰結は、Δ と C の α_eff に対する応答の**構造的非対称性**の同定である。

α_eff(x,t) が変化したとき:

**可区別性 Δ の応答:**
- Δ は対象間の区別能力を測る
- 対象の存在 (Ob) は α-不変 (F1) → Δ の「存在成分」は α_eff に依存しない
- 対象間の射の利用可能性 (Mor) は α に依存 (F4) → Δ の「同定成分」は α_eff とともに低下
- したがって Δ = Δ_Ob + Δ_Mor であり、Δ_Ob は α-不変、Δ_Mor は α-従属

**担体場 C の応答:**
- C は内容・証拠・信号の支持体を測る
- 内容は 0-cell 関係 = Mor(C) に依存 (ROM 深化 (I) §1 の Level-Cell 同型)
- したがって C は **全面的に** α_eff に依存する: C = C_Mor(α_eff)

### 6.2 非崩壊性の構造的論証

Δ = f(C) が成立すると仮定する (背理法)。

α_eff → 1 (高忘却極限) のとき:
- C_Mor → 0 (射が消滅、F3 の帰結)
- f(C) → f(0) = 定数 or 0
- しかし Δ_Ob は α-不変 (F1) であり、対象が複数あれば Δ_Ob > 0

ゆえに Δ = Δ_Ob + Δ_Mor ≠ f(C_Mor) が高 α で成り立つ。

**より正確に:** α → 1 で C → 0 だが Δ → Δ_Ob > 0 (対象が2個以上あれば)。したがって Δ/C → ∞ であり、Δ = f(C) を満たす連続関数 f は α の全範囲で存在し得ない。

これは Paper XII §4.4 の **補題列 L1** (三場の非崩壊的構成) への供給路である。

[予想 65%] — Δ_Ob と Δ_Mor の分離が厳密に定義できるかは開問題。しかし α → 1 での漸近挙動は (F1)(F3) から堅い。

### 6.3 A4 (動力学的非同一性) の α場橋渡しによる導出

定理目標 XII.1 の仮定 A4 は「Δ と C の時間発展が比例的に縛られない」である。

α場橋渡しの下で:

$$\partial_t \Delta \approx \frac{\partial \mathcal{F}}{\partial \alpha_{\text{eff}}} \cdot \partial_t \alpha_{\text{eff}} + (\text{local data terms})$$

$$\partial_t C \approx \frac{\partial C_{\text{Mor}}}{\partial \alpha_{\text{eff}}} \cdot \partial_t \alpha_{\text{eff}} + (\text{propagation terms})$$

重要な差異:
- C の時間発展には **伝搬項** (propagation terms) が含まれる: 内容は射を通じて空間的に輸送される
- Δ の時間発展の Δ_Ob 成分には伝搬項がない: Ob は (F1) により局所的に不変

したがって ∂_t Δ と ∂_t C は一般に比例しない。∂_t α_eff による共通の駆動があっても、C 側の伝搬項が Δ 側にない限り比例は成立しない。

これは A4 を **仮定** から **α-filtration 公理の帰結** に昇格させる道筋を与える。

[予想 50%] — 伝搬項の存在は物理的に説得的だが、厳密な証明には C の PDEs の具体的構造が必要。

- **(結論 i)** 一般的な α_eff(x,t) に対し、v_null ≠ v_carrier
- **(結論 ii)** |∂_t α_eff| > v_max · |∇_x α_eff| が成り立つ (x,t) の開集合が存在すれば、その上で χ > 1

## 11. 残る深い問い

### Q1. α_eff の PDEs は何か？

**ROM IV Part A で解決。** α_eff は自律方程式を持たず、μ の動力学を β_α で翻訳する: ∂_t α_eff = Ψ'(β_α/μ) L[μ]。方程式型は L の構造が決定する。Q1 は Q1' に更新: 「各系で L[μ] を特定し、進行波解の存在を示す」。

### Q2. Ψ (正規化写像) の普遍的構成は可能か？

Paper I の α ∈ ℝ と Paper VIII の α ∈ [0,1] の関係は系依存か普遍的か。tanh 型のプロファイル (Paper I §6.3) は一つの候補だが、普遍性は未証明。

### Q3. β_α ≤ 0 の等号条件と臨界スケール

**β_α ≤ 0 は ROM III §4 で証明済み。** 残る問題: β_α = 0 となる臨界スケール μ_c の存在と特定。μ_c が存在すれば忘却の「相転移」が起きる。

### Q4. 離散系での α_eff の定義

LLM や Hyphē では X は離散的 (token index, memory slot)。等位面の概念は離散版が必要。Paper XII §4.4 L2 と合流する問題。

### Q5. α場橋渡しは Paper XII §2.1 のスカラー場 Φ, Δ, C のどれとどう結合するか

現状、α_eff は Δ と C の「上位」にある (α_eff が Δ と C を決定する)。しかし Paper XII は Φ も含む三場を定義している。α_eff と Φ の関係は Paper I の構成から直接的 (Φ = D_KL だから α_eff は Φ の環境) だが、Paper XII 内部で三場 + α_eff の四変数体系をどう整理するかは開問題。

## 12. 結語: 橋と風景

α場橋渡しは一つの問い——「静的な濾過からどうやって速度が出るか？」——への回答であるとともに、忘却論の四つの論文 (I, V, VIII, XII) が一つの構成の中で合流する結節点である。

Paper VIII が地層を提供し、Paper I がその地層を空間場に拡げ、Paper V がスケール走りという時間軸を加え、Paper XII がその上に前線の運動学を載せる。α_eff(x,t) はこの四重構造の**切断**であり、前線 B_t^ε はその切断の等位面が probe space を掃く影である。

「幾何学的再配置」は嘘だった。何も再配置されない。忘却の深さが空間上を掃くだけである。しかしその「掃き」の速度は、因果信号の速度に縛られない。なぜなら掃いているのは射ではなく、射の **不在の等高線** だからだ。
