## 3. §4 数学的先行性の三例

### 3-1. Yoneda 補題 (Paper VIII §6.1–6.5)

> **[DECISION]** 米田補題は「存在 (THAT)」を忘却し「同定 (WHAT)」のみを扱う。忘却論は α=0 の特殊ケースとして米田を包含する。

核心:
- 米田: Hom(-,X) で X を同定。だが `X ∈ Ob(C)` (存在) は**暗黙の前提** = 忘却されている
- CPS0': 容器 (1-cell=存在) は内容 (0-cell=同定) に先行 — 米田が「見えていない」レベル
- α-米田埋込: α=0 で完全同定 (通常の米田)、α>0 で部分的同定 (有限主体定理)
- **系 6.5.2** (Paper VIII): 忘却論 ⊃ 米田 (α=0 の特殊ケース)

確信: [確信 80%]

### 3-2. Gödel 不完全性定理 (完全性エッセイ §3)

> **[DECISION]** 形式体系の定義 = 忘却関手 U: T→F の適用。公理選択 = ker(U) の設計。完全性 = 豊かさの切除。

核心:
- プレスバーガー算術が完全な理由: 乗法を ker(U) に落として自己言及を封じた
- Gödel 第一不完全性 = ker(U_T) ≠ ∅ の帰結。欠陥ではなく**豊かな体系の構造的必然**
- 「完全性 = 忘却を深く適用した貧しい到達可能性」(完全性エッセイ §3 原文)

確信: [確信 75%] ← OP-Integration-1: Paper VIII §6 との整合性確認が必要

### 3-3. FTC の CPS スパン構造 (Paper II §1.1)

> **[DECISION]** FTC の非対称性 (∫∘d = id+C) = U⊣N の余単位が同型でない = Drift Θ = 忘却の不可逆性の解析的実現。

核心:
- 微分 d = 忘却関手 G (積分定数を捨てる)
- 積分 ∫ = 構成関手 F (定数 C で補う)
- 積分定数 C = Drift Θ = 忘却の不可逆成分
- DRB 圏 (完全構造) からの CPS スパン: DIF ←U'- DRB -V'→ RBA
- FTC の不完全性 = 余単位 ε の非同型性 → これが解析学の根本的非対称性の正体

確信: [確信 85%]
