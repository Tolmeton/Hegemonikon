### η trace (本質洞察) {#sec_03a_eta}

### 修正と検証 {#sec_03b_fix}

> **[DECISION]** 多層防御を採用:

| 層 | 設定先 | 状態 |
|:---|:-------|:-----|
| User settings | `~/.config/Code/User/settings.json` | ❌ VS Code が Reload 時に上書き (キャッシュ問題) |
| WS settings | `~/oikos/.vscode/settings.json` | ✅ `true` で安定 — User 設定をオーバーライド |

> **[DISCOVERY]** VS Code User settings は `sed` 直接編集しても Reload 時にメモリキャッシュから上書きされる。GUI (Settings Editor) 経由 or WS 設定で上書き必須。
