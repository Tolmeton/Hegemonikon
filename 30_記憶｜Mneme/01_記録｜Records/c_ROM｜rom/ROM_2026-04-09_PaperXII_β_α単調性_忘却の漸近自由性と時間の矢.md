## 1. 問題の定式化

### 1.1 β_Φ と β_α の非対称性

Paper V は二つの β 関数を定義する:

$$\beta_\Phi(\theta, \mu) := \mu \frac{\partial \Phi}{\partial \mu}, \qquad \beta_\alpha(\theta, \mu) := \mu \frac{\partial \alpha}{\partial \mu}$$

β_Φ ≥ 0 は DPI (データ処理不等式) から証明されている (Paper V Th. 2.2.2):
- Φ = D_KL は f-ダイバージェンス
- 粗視化 R_{μ→μ'} は Markov 写像
- DPI: D_KL(Rp || Rq) ≤ D_KL(p || q)
- したがって Φ は μ の単調非減少関数

**しかし α は f-ダイバージェンスではない。** α は忘却の「強度」を測るパラメータであり、分布間の距離ではない。DPI は直接適用できない。

### 1.2 β_α の符号は何であるべきか

物理的直観から:
- **高解像度 (μ → ∞)**: すべての射が観測可能 → 忘却は不要 → α → 0
- **低解像度 (μ → 0)**: 射がほぼ観測不能 → 忘却が最大 → α → 1

したがって α は μ の**単調非増加**関数であるべき:

$$\frac{\partial \alpha}{\partial \mu} \leq 0 \quad \Longrightarrow \quad \beta_\alpha = \mu \frac{\partial \alpha}{\partial \mu} \leq 0$$

## 2. ROM II の符号誤り

### 2.1 誤りの同定

ROM 深化 (II) §5.1 は次のように主張した:

> 「粗視化は一般に射の利用可能性を減少させるから、α は粗視化に伴い増大する傾向をもつ。すなわち β_α ≥ 0 が自然な制約。」

**物理的推論は正しい**: 粗視化 (μ↓) で α は増大する。
**数学的結論が誤り**: α↑ when μ↓ ⟹ ∂α/∂μ < 0 ⟹ β_α = μ(∂α/∂μ) < 0。

正しい結論は **β_α ≤ 0** である。

### 2.2 内部矛盾の検出

ROM II §5.2 は β_α ≥ 0 を仮定して次を計算した:

$$\frac{d\alpha_{\text{eff}}}{dt} \ni \frac{\beta_\alpha}{\mu} \cdot \frac{d\mu}{dt}$$

β_α ≥ 0 かつ dμ/dt ≤ 0 ならこの項は ≤ 0。すなわち α_eff は**減少**する。

しかし同じ節で「粗視化が進む (μ↓) と α_eff は上昇する。これは直観に合う」と述べており、**同一節内で符号が矛盾**していた。「Ψ の正規化が効く」という手振りで矛盾を回避しようとしたが、これは不要な弥縫であった。

### 2.3 訂正後の整合性

β_α ≤ 0 に訂正すると:

$$\frac{\beta_\alpha}{\mu} \cdot \frac{d\mu}{dt} = \frac{(\leq 0)}{\mu} \cdot (\leq 0) = \frac{(\leq 0) \cdot (\leq 0)}{\mu} \geq 0$$

α_eff は時間とともに**増大**する。Ψ の特別な仕掛けは不要。直観・数学・物理が一貫する。

## 3. 粗視化-濾過互換公理 (CFC)

### 3.1 動機

β_α の符号を導出するには、RG 粗視化と α-filtration の関係を公理化する必要がある。

### 3.2 公理の定式化

**公理 CFC (Coarsening-Filtration Compatibility).** Paper V の Markov 粗視化 R_{μ→μ'} (μ > μ') が圏 C 上に作用するとき、R は C の**広部分圏制限**として実現される:

$$R_{\mu \to \mu'}(C) \hookrightarrow C \quad \text{(広部分圏として)}$$

かつ、Paper VIII の α-忘却濾過 {C_α} に対し、単調非増加関数 α: ℝ₊ → [0,1] が存在して:

$$\text{Mor}(R_{\mu \to \mu'}(C)) \subseteq \text{Mor}(C_{\alpha(\mu')})$$

### 3.3 CFC の自然性

CFC は以下から従う（実質的に導出可能）:

**(a)** Markov 粗視化は対象を保存する (Ob(R(C)) = Ob(C))。これは R が射に対する情報損失であり、存在そのものの消去ではないことによる — Paper VIII CPS0' (容器 > 内容) の帰結。

**(b)** Markov 粗視化は射を除去する (Mor(R(C)) ⊆ Mor(C))。DPI の背後にある情報損失はこの射の除去として圏論的に表現される。

**(c)** (a)(b) より R(C) は C の広部分圏。α-filtration は C の広部分圏の最も一般的なパラメトライゼーション (定義 6.2.1 により [0,1] で index) であるから、任意の広部分圏 D ⊆ C に対し:

$$\alpha_D := \inf\{\alpha \in [0,1] : \text{Mor}(D) \supseteq \text{Mor}(C_\alpha)\}$$

が well-defined。(F2) より C_0 = C なので α_D ≤ 0 ではない。(F3) より C_1 = C_disc ⊆ D (恒等射は常に保存) なので α_D ≤ 1。

**(d)** R_{μ₁→μ₁'} と R_{μ₂→μ₂'} の合成に対する単調性は (F4) から従う。

**したがって CFC は Paper V の Markov 粗視化 + Paper VIII の (F1)-(F4) の自然な帰結であり、新たな公理的仮定を追加するものではない。**

## 4. 定理: β_α ≤ 0 (忘却の漸近自由性)

### 4.1 主定理

**定理 (α-単調性; 忘却の漸近自由性).** α-忘却濾過 {C_α} が (F1)-(F4) を満たし、Markov 粗視化が CFC を満たすとき:

$$\boxed{\beta_\alpha(\theta, \mu) := \mu \frac{\partial \alpha}{\partial \mu} \leq 0}$$

等号は粗視化が射を追加的に除去しないとき（十分統計量を保存するとき）に限り成立する。

### 4.2 証明

**証明.** 6 ステップで進む。

**(Step 1: Markov 粗視化は広部分圏制限)**
Paper V §2.1 より粗視化 R_{μ→μ'} (μ > μ') は Markov 写像。CFC §3.3(a)(b) より R(C) は C の広部分圏。

**(Step 2: 広部分圏は α-filtration でパラメトライズ)**
CFC §3.3(c) より、α: ℝ₊ → [0,1] が存在して Mor(R_{·→μ'}(C)) ⊆ Mor(C_{α(μ')})。

**(Step 3: 解像度の単調性)**
μ₁ < μ₂ のとき、R_{μ₂→μ₁} は μ₂ 解像度から μ₁ 解像度への粗視化。
R_{μ₂→μ₁} ∘ R_{∞→μ₂} = R_{∞→μ₁} (RG 群の合成則) より:

$$\text{Mor}(C_{\mu_1}) = \text{Mor}(R_{∞→μ_1}(C)) \subseteq \text{Mor}(R_{∞→μ_2}(C)) = \text{Mor}(C_{\mu_2})$$

すなわち低解像度は高解像度の部分圏。

**(Step 4: α の単調性)**
Step 2-3 と (F4) を結合する。μ₁ < μ₂ のとき:

Mor(C_{α(μ₁)}) ⊇ Mor(C_{\mu_1}) ... (Step 2)
Mor(C_{\mu_1}) ⊆ Mor(C_{\mu_2}) ... (Step 3)
Mor(C_{\mu_2}) ⊆ Mor(C_{α(μ₂)}) ... (Step 2)

α の定義 (§3.3(c)) と (F4) の対偶 (Mor(C_{α₁}) ⊇ Mor(C_{α₂}) ⟹ α₁ ≤ α₂) より:

$$\mu_1 < \mu_2 \Longrightarrow \alpha(\mu_1) \geq \alpha(\mu_2)$$

**(Step 5: β_α の符号)**
α は μ の単調非増加関数:

$$\frac{\partial \alpha}{\partial \mu} \leq 0 \quad \Longrightarrow \quad \beta_\alpha = \mu \frac{\partial \alpha}{\partial \mu} \leq 0$$

**(Step 6: 等号条件)**
β_α = 0 ⟺ ∂α/∂μ = 0 ⟺ 粗視化が射の数を変えない ⟺ 粗視化が十分統計量を保存。これは Paper V Th. 2.2.2 の β_Φ = 0 の等号条件と整合する。 □

### 4.3 証明に必要な仮定の整理

| 仮定 | 出典 | 必要性 |
|:---|:---|:---|
| α-filtration (F1)-(F4) | Paper VIII Def. 6.2.1 | Step 2, 4 |
| Markov 粗視化 | Paper V §2.1 | Step 1 |
| CFC | §3 (Paper V + VIII から導出) | Step 2 |
| RG 群の合成則 | Paper V (標準的) | Step 3 |

**追加的な新規公理は不要。** 証明は Paper V と Paper VIII の既存構造のみから従う。

## 5. 双対 c 定理

### 5.1 残留精度の定義

$$\pi(\theta, \mu) := 1 - \alpha(\theta, \mu)$$

π は「残留精度」(residual precision) — 解像度 μ で保存されている射の構造の度合い。

### 5.2 残留精度の c 定理

**系 (残留精度の c 定理).** β_π := μ ∂π/∂μ = -β_α ≥ 0。

すなわち、残留精度 π は μ の単調非減少関数。これは β_Φ ≥ 0 と双対をなす:

| | β_Φ ≥ 0 (Paper V Th. 2.2.2) | β_π ≥ 0 (本文書) |
|:---|:---|:---|
| 量 | Φ = KL ダイバージェンス | π = 1 - α = 残留精度 |
| 物理的意味 | 高解像度で情報がより見える | 高解像度で射がより保存される |
| 根拠 | DPI (Markov 写像は KL を増やさない) | CFC + (F4) (粗視化は射を減らすだけ) |
| 等号 | 十分統計量の保存 | 十分統計量の保存 |
| QFT 対応物 | c 定理 (中心荷の UV→IR 単調減少) | 結合定数の漸近自由性 |

### 5.3 連鎖律の整合性

CFC のもとで Φ(θ,μ) = Φ(θ, α(μ)) と書けるとき:

$$\beta_\Phi = \frac{\partial \Phi}{\partial \alpha} \cdot \beta_\alpha$$

- ∂Φ/∂α ≤ 0: α の増大で KL は減少 ((F4) + Markov)
- β_α ≤ 0: α は μ の非増加関数 (定理 4.1)
- 積: (≤ 0)(≤ 0) = ≥ 0 = β_Φ ✓

**三者の符号が整合する。** ROM II では β_α ≥ 0 としたため ∂Φ/∂α ≤ 0 との積が ≤ 0 となり、β_Φ ≥ 0 と矛盾していた。

## 6. 時間の矢 = 忘却の矢

### 6.1 証明鎖

4つの結果を連鎖させる:

```
Paper V Th. 2.2.2:  β_Φ ≥ 0      (DPI)
本文書 Th. 4.1:     β_α ≤ 0      (CFC + (F4))
Paper IX Th. 3.4.1: S_CPS ↑ in α  ((F4) + Perrone)
物理的仮定:         dμ/dt ≤ 0     (観測劣化)
```

**連鎖:**

$$\mu(t) \downarrow \xrightarrow{\beta_\alpha \leq 0} \alpha(t) \uparrow \xrightarrow{Th. 3.4.1} S_{\text{CPS}}(t) \uparrow$$

$$\boxed{\text{時間の矢} = \text{忘却の矢} = \text{エントロピーの矢}}$$

### 6.2 第二法則の忘却論的導出

**命題 (忘却の第二法則 — 動的版).** μ(t) が単調非増加であり、β_α ≤ 0 (Th. 4.1) かつ S_CPS が α-単調 (Paper IX Th. 3.4.1) ならば:

$$\frac{dS_{\text{CPS}}}{dt} \geq 0$$

*証明.*
$$\frac{dS}{dt} = \frac{\partial S}{\partial \alpha} \cdot \frac{d\alpha}{dt} = \underbrace{\frac{\partial S}{\partial \alpha}}_{\geq 0 \text{ (Th. 3.4.1)}} \cdot \underbrace{\frac{\partial \alpha}{\partial \mu}}_{\leq 0 \text{ (Th. 4.1)}} \cdot \underbrace{\frac{d\mu}{dt}}_{\leq 0 \text{ (仮定)}} \geq 0$$

積の符号: (≥ 0)(≤ 0)(≤ 0) = ≥ 0。 □

### 6.3 QCD との構造的同型

| QCD の漸近自由性 | 忘却の漸近自由性 |
|:---|:---|
| α_s(μ) → 0 as μ → ∞ | α(θ,μ) → 0 as μ → ∞ |
| β_{α_s} < 0 | β_α ≤ 0 |
| 高エネルギーで結合が弱まる | 高解像度で忘却が弱まる |
| 低エネルギーで閉じ込め (α_s → ∞) | 低解像度で完全忘却 (α → 1) |
| 根拠: 非アーベルゲージ群の反遮蔽 | 根拠: α-filtration (F4) + CFC |

**決定的な相違:** QCD の β < 0 はゲージ群の非アーベル性から従う（摂動計算の結果）。忘却の β_α ≤ 0 は α-filtration の公理 (F4) から従う（非摂動的、厳密）。後者はより基本的である。

## 7. Paper XII への帰結

### 7.1 前線伝搬の不可避性

ROM II は「α_eff(x,t) が時間とともに増大する → 前線が伝搬する」を主張したが、β_α の符号が不正確であったため論拠が不完全であった。

β_α ≤ 0 の下で:

$$\frac{d\alpha_{\text{eff}}}{dt} = \frac{\beta_\alpha}{\mu} \cdot \frac{d\mu}{dt} + \frac{\partial \alpha}{\partial t}\bigg|_{\text{explicit}} \geq 0$$

(第一項は ≥ 0 (§2.3)、第二項は系の明示的時間変動)

α_eff が単調増大するならば、等位面 {x : α_eff(x,t) = α_c(ε)} は必ず移動する。**前線伝搬は構造的に不可避。**

### 7.2 定理 XII.1' の仮定 A4 の強化

ROM II §7 で提案した A4 (動的非恒等性) の α-filtration 導出は、β_α ≤ 0 により強化される:

- (旧) χ > 1 の条件: |∂_t α_eff| > v_max · |∇_x α_eff| [予想 55%]
- (新) β_α ≤ 0 により ∂_t α_eff > 0 が保証。χ > 1 は ∂_t α_eff / |∇_x α_eff| > v_max のみに帰着。[予想 → 構造的対応 70%]

### 7.3 Paper V (RG) 方程式との整合

Paper V §2.2.5 の (RG) 方程式:

$$\frac{\alpha^2}{4} P^{ij} \nabla_i \nabla_j \beta_\Phi + \lambda \beta_\Phi = \left(\frac{2\beta_\alpha}{\alpha} - \frac{\beta_\lambda}{\lambda}\right) \lambda \Phi$$

右辺のソース項: β_α ≤ 0 かつ β_λ ≤ 0 (Paper V §2.2.5 注記) のとき:

$$\frac{2\beta_\alpha}{\alpha} - \frac{\beta_\lambda}{\lambda} = \frac{(\leq 0)}{\alpha} - \frac{(\leq 0)}{\lambda} = (\leq 0) - (\leq 0) = ?$$

符号は確定しない — β_α と β_λ の相対的大きさに依存。これは β_Φ ≥ 0 (DPI) が (RG) 方程式に追加の制約を課すことを意味する: DPI は β_α と β_λ の「バランス条件」を与える。

β_α = 0 となる μ_c は存在するか。存在するなら、そこで「忘却の相転移」が起きる: μ > μ_c では射が保存され (α ≈ 0)、μ < μ_c では射が急速に失われる (α → 1)。

## 11. 結語

β_α ≤ 0 は忘却論の時間構造を閉じる鍵であった。

証明は驚くほど単純: **粗視化は射を減らすだけ (広部分圏制限) であり、射が減ることは α の増大に対応し ((F4))、μ の減少に対応する (CFC)**。三段論法で終わる。

単純であることは深くないことを意味しない。β_Φ ≥ 0 の証明も同様に単純 (DPI の一行) であった。**基本的な不等式は、基本的な構造から一歩で従う。**

忘却は漸近自由である。完全に見れば忘却はゼロ。不完全に見ることだけが忘却を生む。そして我々は常に不完全に見ている。時間はその不完全さの増大に他ならない。

> **掃くのは射の不在の等高線であり、因果信号ではない。**
> **そしてその等高線は、必ず、一方向にしか動かない。**
