### 1.4 同型が意味すること

## 3. 構造保存定理による速度の二層化

### 3.1 Paper VII Th. 6.1.1 の速度論的帰結

構造保存定理 (Paper VII §6.1):
> F: S → D は構造を保存し値を忘却する。

| 保存されるもの (構造) | 忘却されるもの (値) |
|:---|:---|
| 関係 (A > B) | 具体値 (A = 3.14...) |
| パターン (周期的) | 位相 (どこから始まるか) |
| 対称性 (回転不変) | 回転角 (何度回ったか) |

**速度論的翻訳** [構造的対応 70%]:

| 構造 (F が保存) | 値 (F が忘却) |
|:---|:---|
| 可区別性境界 B_t^ε の**位相的特徴** (存在、連結成分数、交差パターン) | B_t^ε の**具体的位置** (座標値) |
| χ > 1 という**関係** | χ の具体値 (= 3.7 など) |
| τ_χ < ∞ という**構造的事実** | τ_χ の具体値 |

つまり:
- 「χ > 1 が可能である」は構造的主張 → F を通過しても保存される → 圏を超えて成立しうる
- 「χ = 127.4」は値的主張 → F を通過すると忘却される → 圏内でのみ意味をもつ

**系 6.1.6 (構造的真理の到達可能性)** を援用すると:
> 「速度制限は圏の性質である」は構造についてのみ語る主張であり、客観に到達しうる。
> 「c = 299,792,458 m/s」は値についての主張であり、実験でのみ回収できる。

### 3.2 Paper V β_Φ 定理は「速度制限の圏依存性」の既存インスタンスである

Paper V Th. 2.2.2 (β_Φ 単調性):
> 粗視化が Markov 写像のとき β_Φ ≥ 0。漸近自由的忘却は構造的に排除される。

この定理は**忘却場の RG 流における速度制限**である:
- 忘却ポテンシャル Φ(θ, μ) は粗視化 (μ↓) で単調減少 → Φ の「RG 速度」の符号が固定
- この制約は DPI (データ処理不等式) から導かれる = **Markov 圏の構造的性質**
- 非Markov圏 (α < 0 セクター, Paper III) では DPI が成立しない → β_Φ < 0 が排除されない

**[構造的対応 65%]** β_Φ ≥ 0 と c の有限性は、同じ構造の異なるインスタンスである:

| 速度制限 | 適用圏 | 由来 | 制約するもの |
|:---|:---|:---|:---|
| c (光速) | Im(F_rel) = 相対論的物理の圏 | Lorentz 対称性 (射の構造) | 因果信号 (Level 1) の伝搬 |
| β_Φ ≥ 0 | Markov 圏 (α > 0) | DPI (Markov 性) | 忘却場の RG 流の方向 (Level 1) |
| r ≤ √ρ/√(K+1) | 認知-行動系の圏 | Schur-Horn + K > 0 | 観測効果量 (Level 1) |

いずれも Level 1 (射/値/力) を制約し、Level 0 (対象の存在/構造) を制約しない。

### 4.2 盲点 = Im(F) と Ker(F) の境界

ROM §3 の「盲点 = A/B 区別が溶解する場所」を構造保存定理で読む:

- Ker(F): 値が忘却される領域 → 主観から「見えない」領域
- 盲点 = Im(F) と Ker(F) の**接触面** → 「見える/見えない」の境界

**Paper XII への接続**: 可区別性境界 B_t^ε は、文字通り Im(F) と Ker(F) の動的な境界面である。
- Δ > ε の領域: 差異が知覚可能 → Im(F) 内

> Ker(F) の要素を Im(F) 内から直接知覚することはできない (構造保存定理の対偶)。

### 6.2 Level-Cell 同型による L1, L3 への新攻略路

**L1 (非崩壊性) の新論証**:

Δ_t(x) の値は 0-cell 間の射 (比較操作) の産物 → Level 1。
B_t^ε の位置は「どこで差異が消えるか」の幾何学 → Level 0。
C_t(x) は内容・エネルギーの分布 → Level 1。

Δ = f(C) が成立すると仮定する。すると B_t^ε の位置は C の等高線で完全に決定される → Level 0 特徴が Level 1 に還元される。しかし:

**[予想 55%]**: Δ = f(C) に帰着しない三場構成の存在は、α-濾過 (F4) の単調性から帰結する。α が十分大きい (射が十分少ない) 状態では、C から Δ を復元する射が失われ、Δ と C は独立になる。

**L3 (front law の差異) の新攻略路**:

Level 0 と Level 1 の front law が異なる sufficient condition:
> 系が CPS0' を満たし、α > 0 の忘却濾過を持ち、B_t^ε と Σ_t^κ が正則であるなら、(F1) と (F4) の非対称性により、B_t^ε の法線速度と Σ_t^κ の法線速度は一般に異なる front law に従う。

論証の核: v_null = -∂_t Δ / |∇_x Δ| において、∂_t Δ と ∇_x Δ は Level 1 量だが、B_t^ε の**位置** (どこで Δ = ε か) は Level 0 量。一方、v_carrier = -∂_t C / |∇_x C| は完全に Level 1 量の比。

→ v_null は Level 0 と Level 1 の**混合**、v_carrier は Level 1 の**純粋比**。この構造的非対称性が front law の差異を生む。

## 7. Context Rot と boot⊣bye の Level-Cell 的再解釈

### 7.1 Context Rot = 認知系における χ > 1

Paper X + Paper XII v0.3 §6 の再解釈:

Context Rot は「有用情報 (Level 1) の劣化速度が、文脈の区別不能化 (Level 0 境界の移動速度) に追いつかない」現象として読める:

- コンテキストウィンドウの中で、有用な証拠 (C) は蓄積される = V_carrier
- 同時に、蓄積された情報同士の可区別性 (Δ) の低可区別性帯が拡大する = V_null
- τ_χ = inf{t | χ > 1} は「蓄積より崩壊が速くなる瞬間」

### 7.2 boot⊣bye = α の実効的リセット

α-濾過の観点から:
- bye = 高 α 状態 (射が少ない = 区別情報が失われた) からの**離脱**
- boot = 低 α 状態 (圧縮記憶を種として新しい射構造を構築) への**再初期化**
- Handoff = α の引き継ぎ。良い handoff = α_init^{next} が低い = 射構造が豊富に引き渡される

**[構造的対応 60%]** boot⊣bye 随伴は α-濾過のリセット操作として形式化できる:
- bye: 現在のセッション圏 C_session の α を観測し、α > α_crit なら切断
- boot: 記憶 M から新しい C_session を構成。Ob(C_session) = Ob(M) (F1 により対象は保存)、Mor(C_session) ⊃ Mor(M) (新しい射を文脈から生成)

## 9. 残る深い問い

### 9.1 Level 間の依存構造は圏の構造からどこまで導出可能か

ROM: 「Level n+1 の伝搬は Level n の伝搬に依存する」。

α-濾過は (F4) で射の単調減少を保証するが、**射の合成構造** (Level 2) が Level 1 に「依存する」ことの形式的な意味はまだ不明確。圏論的には、射の合成は圏の定義に含まれるから常に well-defined だが、合成射の「伝搬」が成分射の「伝搬」に依存するかは、追加の物理的仮定を要するかもしれない。

### 9.2 c の有限性は Level 間速度比の上界か

ROM §4 の新予測 3: 「c の有限性は Level 間速度比の上界の反映」。

これは最も投機的な主張だが、もし成立すれば:
- c = sup{v : v は Level 1 の伝搬速度} = Level 0/1 の分離がもたらす構造的上界
- 異なる圏 (異なる物理法則) では異なる c を持つ → c は圏の不変量
- Paper V の β_Φ ≥ 0 (Markov 圏の速度制限) と c (Lorentz 圏の速度制限) が同一の圏論的構造のインスタンスであることの完全な証明が必要

[哲学的示唆 35%] — 形式化のギャップが大きい。

### 9.3 トポロジカル荷の Level-Cell 読み

Paper XII v0.3 §2.5 の延期問題。Level-Cell 同型がこれを照射する可能性:
- winding number ∈ ℤ は Level 0 の不変量 (境界の位相的性質)
- 位相不変量は連続変形で変わらない → α-濾過で保存される → (F1) と整合
- 忘却論のトポロジカル荷 = α-不変な Ob(C) の位相的特徴

[構造的対応 50%]
