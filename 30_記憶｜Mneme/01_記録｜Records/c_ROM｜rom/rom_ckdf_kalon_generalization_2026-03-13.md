## 核心的発見 (3つ)

### 1. FEP は場の1つにすぎない

「場の数学的構造から座標を検出する」アプローチは FEP 固有ではない。
PCA, Fourier, スペクトル分解, Helmholtz 分解, HGK 座標導出は全て同じ構造:

> **構造化された空間の内在的方向を解析する = P (多項式時間)**

NP 困難になるのは「最適分割 (Search)」を求めるときのみ。
FEP は「場」の一つ。HGK が FEP からやったことは PCA が共分散行列からやることと構造的に同型。

### 2. Kalon ⊃ Optimization

Kalon は最適化問題を特殊ケースとして包含する:
- 最適化: argmin f(x) — コスト関数 f が必要
- Kalon: Fix(G∘F) — 随伴 F⊣G が必要 (コスト関数不要)

全ての最適化は Kalon に埋め込めるが、逆は成り立たない。
例: CCL の ~ (自然変換) は「何を最適化しているか」が定義できないが Fix(G∘F) は存在する。

Kalon ≈ 「最適化の圏の terminal object」。普遍性。

### 3. Kalon▽ (狭義) と Kalon△ (広義) の区別

| | Kalon▽ (狭義) | Kalon△ (広義) |
|:--|:-------------|:-------------|
| 定義 | Fix(G∘F) in 全空間 Ω | Fix(G∘F) in MB(A) |
| 計算量 | NP | P |
| 一意性 | 一意 | 非一意 (A 依存) |
| 到達性 | 不可能 | 可能 |
| 主体依存 | 客観 | 主観 |

**混同が引き起こす3つの誤り:**
- Type 1: △を▽と誤認 → 過信 (最も危険。LLM が特に犯す)
- Type 2: ▽の不可能性から虚無 → △の実用的価値を軽視
- Type 3: △の非一意性から相対主義 → MB 内では一意 (Knaster-Tarski)

Creator の言葉: 「答えは常にそこにある (▽)。ただそれを知覚できていないだけ (我々が到達するのは△)」

## CKDF (Categorical Kalon Detection Framework) 骨格

```
L0: 場 (任意の完備束 C)
L1: 随伴 (ガロア接続 F⊣G, 非退化)
L2: 座標検出 (場の固有分解 → d 本の座標) ... P
L3: Kalon△ (G∘F 反復 → 局所不動点) ... P
L∞: Kalon▽ (全空間の不動点 → 到達不能) ... NP
普遍性: フレームワーク自体が Fix(G∘F) (M1)
```

FEP はこのフレームワークの instantiation:
C = VFE 空間, F⊣G = Helmholtz, d = 6+1

## 注意点・疑念

1. **P の条件**: 全ての不動点が P ではない (Brouwer = PPAD-complete)。
   P になる十分条件は「ガロア接続 + 完備束」。一般の圏では保証なし。
2. **Kalon ⊃ Optimization の厳密化**: 直感的には明らかだが worked examples + 証明が必要。
3. **座標数の決定原理**: なぜ有限個の座標が出るか。場の対称群の次元に関連する仮説。
