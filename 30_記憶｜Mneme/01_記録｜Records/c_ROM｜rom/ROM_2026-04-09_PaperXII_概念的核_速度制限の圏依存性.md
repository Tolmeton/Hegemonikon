## 2. 暗闇 = CPS スパンのヌル点

**構造**: 光の複素場 ψ = u + iv は CPS スパン:
```
u の視座 ←U_A── ψ (客観) ──U_B→ v の視座
```

- u と v は MECE な射影（互いに排他的 + 網羅的）
- 暗闇 = U_A(ψ)=0 **かつ** U_B(ψ)=0 = **全ての視座が同時にゼロ**
- u/v の分解は恣意的（位相回転で変わる）だが、暗闇の点は不変

**CPS0' との対応**: 暗闇 = 純粋な容器 (THAT without WHAT)。空間的存在はあるが、場の内容はゼロ。

**Paper II との接続**: 暗闇 = 相補性の崩壊点。通常「A か B の少なくとも一方は見える」という保証が破れた場所。

## 3. 盲点の論理

**Creator の定式化**:
- 暗闇は「観測できたら暗闇ではない」——定義上、観測を逃れるもの
- 「語りえぬもの」(Wittgenstein)
- 読み A（客観的不在）と読み B（主観的潰え）の区別自体が溶解する
- 盲点 = A/B の区別が観測的に意味をなさない場所
- 主観-客観の境界が定義不能になる場所

**Paper VII (知覚は忘却) との接続**: 暗闇 = 知覚的忘却が完全なものになった場所。忘却の余地すらない場所。

## 4. 速度制限は圏の性質である ★核心★

**展開の経路**:

(a) 前段: 「盲点は情報ではないから光速制限に縛られない」→ **粗すぎる。盲点にもメタ的情報はある** (Creator の異議)

(c) 深化 (Creator): 「速度制限」は**圏の性質**であり、重力加速度のような宇宙定数と同じ位置づけ。「その圏でのみ真」。

**Level 階層による整理 (Creator 修正 21:05)**:

値 = 状態の状態 = 構造の不均一性 = 力 (Paper I: F = ∇Φ)

| Level | 何であるか | 例 | 速度制限 |
|:---|:---|:---|:---|
| Level 0 | **状態** (構造そのもの) | 場 Φ の存在、ゼロ点の位置 | **c に縛られない** |
| Level 1 | **状態の状態** (構造の不均一 = 値 = 力) | 勾配 ∇Φ, 因果信号, エネルギー | c (相対論) |

伝搬の依存則: **Level n+1 の伝搬は Level n の伝搬に依存する。** ∇Φ を伝搬するには Φ がまず伝搬していなければならない。

## 5. 「外の Bob」と圏間通信

**Creator の問い**: 暗闇を「メッセージとして読む」には受け手 (Bob) がどの圏にいるかが決定的。

**展開**:
- Im(F_rel) 内の Bob: 暗闇の移動が Alice の操作の帰結かどうか区別できない → メッセージにならない
- Im(F_rel) 外の Bob (1-cell を直接知覚する存在): 構造的変化を読み取れるかもしれない → 超光速メッセージになりうる
- 「外の Bob が存在するか」は F_rel の中からは判定不能 (構造保存定理の帰結)

**Creator の洞察** ★最重要★:
> **Alice と Bob を 0-cell としたとき、1-cell の圏では必然的に「暗闇の運動が Alice の操作の結果であること」を知っている（いなければならない）**

→ 1-cell は 0-cell 間の**関係**を含む。関係を知っているのが 1-cell の定義。

**圏間通信の構造保存定理**:
> 相互の圏は**構造でのみ**やりとりできる。値は通信できない。
> 構造のみが、主観 (0-cell) から他の圏 (独立した他の 0-cell や 1-cell) とをつなぐ。
> 値も「1-cell 下の構造」として分解可能。
