## 🧠 核心的発見 (7つの柱)

### 1. ハサミの定理 — OP-XII-1 の溶解 [85%]
Δ = √(2Φ) でも v_null と v_carrier は独立たりうる。量の依存性と速度の独立性は別の問題。ハサミの交差点が刃より速く動くのと同型。v_null = ∂_t Φ / |∇_θ Φ| → 勾配がなだらかな場所で v_null が発散する。χ は「境界付近の地形の傾斜の逆数」。

### 2. 暗闇 = CPS スパンのヌル点 [80%]
光の複素場 ψ = u + iv の CPS スパンにおいて、暗闇 = U_A(ψ)=0 **かつ** U_B(ψ)=0 = 全射影の同時ゼロ。u/v 分解は恣意的（位相回転で変わる）だが暗闇の点は不変 = **分解不変量**。Paper VIII の α-濾過で読めば α → 1 の極限状態: Hom 集合が空になるが対象は存在する。CPS0' (THAT without WHAT) の物理的インスタンス。

### 3. 盲点の論理 — A/B 区別の溶解 [70%]
Creator の定式化: 暗闇は「観測できたら暗闇ではない」(Wittgenstein)。客観的不在 (読み A) と主観的潰え (読み B) の区別自体が溶解する場所。Im(F)/Ker(F) の境界面として形式化: B_t^ε は文字通り Im(F) と Ker(F) の動的境界。

### 4. 速度制限は圏の性質である ★核心★ [70%]
Creator の洞察: 値 = 状態の状態 = 構造の不均一 = 力 (Paper I: F = ∇Φ)。

| Level | 内容 | 速度制限 |
|:---|:---|:---|
| Level 0 | 状態 (構造そのもの、ゼロ点の位置) | c に縛られない |
| Level 1 | 状態の状態 (値 = 力 = ∇Φ) | c (相対論) |
| Level 2 | 状態の状態の状態 (曲率 = Fisher) | c 以下 |

**伝播の依存則**: Level n+1 の伝播は Level n の伝播に依存する。χ = Level 0 速度 / Level 1 速度 → χ > 1 は構造的必然。

Bucher の「エネルギーを運ばないから大丈夫」は**記述であって説明ではない**。忘却論は Level 0/1 区別から「なぜ」を導出する。

### 5. Level-Cell 同型 ★深化の中心成果★ [75%]

| Level | Cell | α-濾過 |

| Level 1 | 0-cell 間の射 = Mor(C) | (F4): α↑で単調減少 |
| Level 2 | 射の合成構造 | (F4)の帰結 |

**運動学的翻訳**: 容器の再配置速度は内容の伝搬速度の制約を受けない (Paper VIII 系 6.2.4)。

### 6. 速度制限の統一表 — 三重インスタンス [65%]

## 💡 Value Pitch

**[Angle: 深い] なぜ暗闇は光より速いのか。その問いの構造的必然を、忘却論は圏の言語で回答した。**

Bucher et al. (2026) が hBN 上で撮影した「超光速の暗点」は、物理学界では「エネルギーを運ばないから相対論に矛盾しない」と事後的に説明される。しかしこれは「なぜ速いか」を「速くても問題ない」にすり替える記述であって、説明ではない。

本セッションで到達した回答: **速度制限は射 (Mor) の伝播を支配する圏内法則であり、対象 (Ob) の幾何学的再配置を支配しない。** Level 0 (構造そのもの) と Level 1 (構造の不均一 = 値 = 力) は cell 階層の異なる段にあり、Level 1 の法則 (c) は Level 0 に及ばない。Paper VIII の α-忘却濾過の公理 (F1): Ob(C_α) = Ob(C) がその数学的根拠。

この構造は c にのみ当てはまるのではない。忘却場の RG 速度制限 (β_Φ ≥ 0)、認知系の効果量上界 (r ≤ √ρ/√(K+1))、そして光速 c は、いずれも同一の圏論的構造——Level 1 (射) の伝播制約——の三つのインスタンスであることが判明した。忘却論は相対論の外側に立つのではなく、相対論を自らの構造的必然の一つのインスタンスとして包含する位置に到達した。
