### 4. 新概念: CPS 交差曲率 F^×

E-1 修正から自然に導出された概念。忘却曲率 F の TM = H ⊕ E_A ⊕ E_B 分解における E_A ∧ E_B 成分:

$$F^\times(v, w) = \frac{\alpha}{2}(d\Phi(\pi_A v) \cdot T(\pi_B w) - d\Phi(\pi_A w) \cdot T(\pi_B v))$$

**物理的意味**: A の盲点方向の忘却と B の盲点方向の感度の反対称的交差。「交差的無知」(cross-ignorance) — A が見逃した変動が B の盲点方向の感度と交差することで初めて出現する曲率。

**構造的意義**: 全曲率 F = F^× + F^{other} と分解される。F^{other} は M の内在的幾何から生じ、CPS 構造がなくても非零。Face Lemma は F^× を特性づける定理であり、Paper I の方向性定理 (F ≠ 0 ⟺ dΦ ∈ ker(T)) は全曲率 F についての定理。両者の役割分担が明瞭になった。

## 技術メモ

### サブモジュール修復 (3層修正パターン)
gitサブモジュールのパス変更には3箇所の修正が必要:
1. `.gitmodules` のパス更新
2. `.git/modules/<path>/` ディレクトリの移動
3. `.git/modules/<path>/config` 内の `worktree` パス更新
`.gitmodules` だけ変えると `fatal: not a git repository` エラー。worktree だけ変えると逆方向の参照が壊れる。**3つ全て揃って初めて動く。**

### CPS 交差曲率 F^× の定義
- 射影演算子 π_A: TM → E_A, π_B: TM → E_B を使用
- F^× は F の6成分分解 (HH, HA, HB, AA, AB, BB) のうち AB 成分
- r_A = 0 ⟹ π_A = 0 ⟹ F^× = 0 (自明)
- 退化ケース: T|_{E_B} = 0 のとき F^× = 0 (B の盲点が統計的に「無害」)

### t-分布の Fisher 計量クロス項
- g_{μν} = E[∂_μℓ · ∂_νℓ] = 0 (∂_μℓ が (x-μ) に関して奇関数 → 奇関数の期待値 = 0)
- g_{σν} = E[∂_σℓ · ∂_νℓ] ≠ 0 (∂_σℓ は偶関数 → 偶×偶 = 偶 → 非零)
- よって g_AB ≠ 0 は g_{σν} ≠ 0 を通じて成立

### α-Δd 対応の補正項
- 旧: α_∞ = Δd + O(Hol(∇^E)) — 数学的に不正確 (Hol は群、スカラーではない)
- 新: α_∞ = Δd + O(‖R^E‖) — R^E は ∇^E の曲率テンソル、ノルムは Fisher 計量による
- Ambrose-Singer: Hol(∇^E) ≠ {1} ⟹ R^E ≠ 0 (逆は一般に偽)
