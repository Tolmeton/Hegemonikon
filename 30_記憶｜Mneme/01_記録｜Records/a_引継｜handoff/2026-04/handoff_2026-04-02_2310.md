## 技術メモ

### MCP SDK anyio 制約 (重要)
`streamablehttp_client` の `__aexit__` を別の async task から呼ぶと RuntimeError (anyio cancel scope mismatch)。そのため Hub は **短命セッション設計** (call_tool ごとに新規接続) が意図的。hub_mcp_server.py L86-88 のコメント参照。これは「バグ」ではなく「制約下の設計」。

### Hub アーキテクチャ
- Hub (port 9700): Starlette ASGI + StreamableHTTPSessionManager per backend route
- `stateless=True`: 各リクエストが新規セッション → ログの "Terminating session: None" は正常
- 13 backends on ports 9700-9712 + api on 9696

### Phantazein Boot の MCP_SERVERS
SSOT は `hub_config.py`。phantazein_mcp_server.py の MCP_SERVERS はヘルスチェック用。名前が hub_config.py と一致しないとフィルタが壊れる。
