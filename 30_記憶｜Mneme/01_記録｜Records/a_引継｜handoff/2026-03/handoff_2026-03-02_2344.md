## ✅ 成果 — 何をしたか

### 1. NVIDIA ドライバ復旧

| 手順 | 内容 |
|:-----|:-----|
| 診断 | `vainfo` → Intel H.264のみ対応。`lsmod` → nvidia 未ロード |
| 根本原因 | `linux-headers-6.12.73` 未インストール → DKMS ビルド失敗 |
| 修正 | `sudo apt install linux-headers-$(uname -r)` → `sudo dkms autoinstall` → `sudo modprobe nvidia` |
| 検証 | `nvidia-smi` ✅ GTX 1050 Ti (550.163.01) |
| 再発防止 | `linux-headers-amd64` メタパッケージ導入、`nvidia-persistenced` 有効化、`/etc/modules-load.d/nvidia-drm.conf` 設定 |

### 2. RustDesk H.265 NVENC 設定

- `RustDesk_default.toml`: `codec-preference = 'h265'`, `av1-test = 'Y'`
- クライアント PC (Intel N100, Intel UHD Gen 12.2) は H.265 HW デコード対応 → H.265 が最適

### 3. サービスヘルスチェック横展開 (5→0 失敗ユニット)
