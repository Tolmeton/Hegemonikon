## 📐 意思決定ログ

| 決定 | 選択肢 | 理由 |
|:-----|:-------|:-----|
| effective_depth 公式 | ~~乗算式 `d×(2-p)`~~ → **加算式 `d+(1-2p)`** | 乗算式は precision=1.0 で depth を据え置く (反直感的)。加算式は precision をシフト量として扱い FEP の gain 解釈と一致: 高precision=exploit=-1段、低precision=explore=+1段 |
| グローバルカウンタ | ~~`_node_counter` モジュール変数~~ → **`itertools.count()` ローカル** | スレッド安全性。並列呼出時の ID 衝突を構造的に不可能にする |
| Parallel BT | ~~MIXED 固定~~ → **Sequence 同様 STATIC 昇格** | 全子ノードが STATIC なら親も STATIC。Sequence と同一パターンにすべき |
| IR エラーログ | ~~`debug`~~ → **`warning` + `ir_error` フィールド** | debug だと本番で見逃す。リグレッション検出のため warning + 結果への明示が必要 |
| `Any` → `ASTNode` | 3箇所 (`ast_node`, `_convert_node`, `_classify_binding_time`) | 型安全性。ASTNode は Union 型として ccl_ast.py L367 で定義済み、L24 で import 済み |
