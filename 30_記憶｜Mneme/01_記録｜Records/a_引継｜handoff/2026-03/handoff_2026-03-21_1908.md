**総合判定: H1 反証** — CoT は ρ の統計的有意な増加を生まない。

### 注目すべきパターン

- **深層 (L17-31) で一貫した正方向 Δρ**: C1/C2 とも深層で +0.02〜+0.057 の改善が見られるが、全て p > 0.12
- **Cohen d は中〜大効果量 (0.5-1.0)**:  サンプルサイズ (n=35) の限界で統計的有意性に到達しない
- **Layer 0 がベスト**: 全条件で Layer 0 (embedding 層) が最高 ρ。CodeBERT と同様のパターン
- **CoT の「方向性」は一貫**: 増加の傾向はあるが、効果が小さすぎて検出力不足

### CodeBERT との比較

| CodeLlama | 0.197 | +0.058 | +0.069 | なし |

CodeLlama の方が Δρ は大きい (約2倍) が、いずれも統計的に非有意。

## 🧠 法則化

### L1: CPU で LLM 7B を回すのは非現実的
CodeLlama 7B (4bit) は CPU 上で 50分/ペアかかる。GPU (T4) では 10秒/ペア。CPU offloading は実験に使えない。4bit 量子化でも 7B モデルは GPU 必須。

### L2: nohup + PYTHONUNBUFFERED=1 は長時間実験の必須パターン
セッション間でコマンドが kill されるため、nohup は必須。さらに Python のバッファリングでログが見えなくなる問題は `PYTHONUNBUFFERED=1` + `python3 -u` で解決。

### L3: rsync でモデルキャッシュ転送は実用的
13GB のモデルキャッシュを Tailscale 経由で rsync すると ~9分。HuggingFace からの再ダウンロードより確実 (認証不要で済む)。
