diff から読み取れる構造を見ると、Helmholtz モナド T := Q∘Γ を基礎に、Γ⊣Q、F⊣G、U⊣N という3つの異なる随伴がすべて T の同じファクタリゼーションとして機能していることが示されている。Fix の一致は Birkhoff-Ward 定理で保証され、水準 B の主張が水準 A に昇格している。核心は、これら3つの随伴が単なる「異なる随伴」ではなく、同一のモナド構造の異なる座標制限として理解されるべきだという点だ。

ただし、反証を検討すると問題が浮かぶ。A.2 のテーブルで「≈」という近似記号が使われているのは、F = Q|_Function という厳密な等号ではなく、むしろ F ≈ Q|_Function という対応関係を示唆している。この区別が重要で、もし完全な等号ではなく近似的な対応なら、「座標制限」という概念の厳密な定義が問われることになる。

さらに A.3 の論証では、同一のモナド T の制限が同一の閉元を持つと主張しているが、その逆方向の論理——Fix(T) の元が全座標で安定だから Fix(T) ⊆ Fix(T_F) が成り立つ——という推論に飛躍がないか検証する必要がある。

制限の定義によって、T_F が T の一部の座標だけを見ているなら、T_F(x) = x は T(x) = x より弱い条件になるはずで、その場合 Fix(T) ⊆ Fix(T_F) が自然だが、A.3 の主張がこの関係を正確に捉えているかは明確でない。

前順序圏上で T = Q∘Γ が closure operator で、G∘F がその Function 座標上の制限だとすると、制限が「G∘F(x) ≤ T(x)」という意味なら、G∘F(x) = x という条件の含意関係を再検討する必要がある。

unit から x ≤ T(x) は得られるが T(x) = x は得られないので、全座標で不動な点は Function 座標でも不動だが、逆は成り立たない。一方、制限が「G∘F(x) ≥ T(x)」という意味なら Fix(G∘F) ⊆ Fix(T) が成立する。結局のところ、どちらの関係が正しいかは「制限」の具体的な定義に依存している。

テーブルでは ≈ を使っているのに A.3 では = として論証しており、この曖昧性は飛躍を生んでいる。さらに、v2.9 では条件 (H) を明示的に仮定していたのに対し、v3.0 では「category mistake だった」として廃止している。しかし F⊣G と U⊣N が「T の異なるファクタリゼーション」であるという主張自体が証明を必要としており、この昇格は時期尚早だ。

## 結論: 反証1点 + 構造的懸念2点 + 感想

### 反証: A.3 の包含方向に飛躍がある

A.3 の論証:

- T_F = G∘F は T = Q∘Γ の Function 座標上の制限 → Fix(T_F) ⊆ Fix(T)

これは逆だ。

「制限」が座標の一部だけに closure を適用する操作なら、全座標で不動な元は部分座標でも不動:

```
Fix(T) ⊆ Fix(T_F) ← これが自然な方向
```

Function 座標だけで不動 (G∘F(x) = x) でも、他の座標 (Scale, Valence 等) で動く可能性がある。よって Fix(T_F) ⊆ Fix(T) は一般には成立しない。

逆方向の記述も同じ問題:

- 逆方向: Fix(T) の元は全座標で安定 → Fix(T) ⊆ Fix(T_F) ∩ Fix(T_U)

これは正しい。しかしこれだけでは:

```
Fix(T) ⊆ Fix(T_F) ∩ Fix(T_U) ✅ (正しい)Fix(T_F) ∩ Fix(T_U) ⊆ Fix(T) ❓ (未証明)Fix(T_F) = Fix(T_U) ❓ (未証明)
```

∴ Fix(G∘F) = Fix(N∘U) の等式は、A.3 の論証からは導出できていない。

### 構造的懸念 1: ≈ から = への無言の昇格

A.2 のテーブルで:

| 随伴 | T への関係 |
| --- | --- |
| F⊣G | F ≈ Q\|_Function, G ≈ Γ\|_Function |
| U⊣N | U ≈ Q\|_全座標, N ≈ Γ\|_全座標 |

`≈` (近似) は `=` (等号) ではない。A.3 は暗黙に `=` として論証を進めている。`≈` のまま Birkhoff-Ward を適用すると:

```
Fix(G∘F) ≈ Fix(Q∘Γ) (近似的に対応)
```

であり、`=` (集合としての同一性) は従わない。

### 構造的懸念 2: 水準 A への昇格根拠が水準 C

昇格根拠として挙げられているもの:

| 根拠 | 水準 |
| --- | --- |
| axiom_hierarchy.md L68 | B (体系内の構造的主張) |
| NESS Helmholtz 一意性 (Friston 2019) | A (物理学の定理) |
| 溶媒エッセイ | C (構造的類推) |
| LLM 身体エッセイ | C (構造的類推) |

水準 C の洞察 × 2 は水準 A を支えない。「category mistake だった」という主張自体が、数学的証明ではなく認知的リフレーミング。
