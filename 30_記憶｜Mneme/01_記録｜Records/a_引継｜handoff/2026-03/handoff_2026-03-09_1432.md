# 🤝 Handoff: [2026-03-09 14:32] MCP Streamable HTTP Migration

## 💎 Value Pitch (成果の意義)
**IDE の呪縛からの解放と、超高速・安定接続の獲得。**
MCPサーバー群が local IDE process の制約から解放され、バックエンドの独立した永続サービス (Systemd + Streamable HTTP) へと生れ変わりました。これにより Antigravity 再起動時の待機時間は消滅し、コードベース内の各サーバーの重複した HTTP 起動ロジックも `mcp_base.py` への一本化によって完全に浄化されました。

## 🎯 成果 (What we achieved)
- 全10台の MCP サーバーを Streamable HTTP トランスポートへ完全移行し、Systemd で永続化しました。
- `mcp_base.py` に `run_streamable_http` 共通ユーティリティを実装し、背景タスク (Heartbeat等) の統合管理を実現しました。
- `jules`, `hermeneus` 内の重複実装を排除し、不要な SSH リモートラッパースクリプトを全削除。
- `mcp_config.json` を URL ベースの接続に統一、すべて正常稼働を確認しました。
