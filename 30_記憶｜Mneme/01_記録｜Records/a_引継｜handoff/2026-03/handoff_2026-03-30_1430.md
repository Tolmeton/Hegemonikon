### 設計判断 (DECISION)

1. **名前 = pinakas (πίνακας)** — HGK のギリシャ語体系と調和。Creator の直感で `/board` より `/pina` を選択
2. **3 ボード MECE** — Seed (ふわっとしたアイデア) / Task (具体的作業) / Question (調べること)。迷ったら Seed に入れる
3. **ファイルプロトコル first → MCP second** — YAML を Claude Code の Read/Edit で直接操作 → 検証後に MCP バックエンド構築
4. **自動 post = D+C ハイブリッド** — ヒューリスティクスによる自動分類は棄却 (ノイズ過多)。Claude の明示的マーキング (`📋T:`) + Stop リマインダー
