## 📚 Knowledge Item

### KI: generateChat API の Claude 互換性

**事実**: `cloudcode-pa.googleapis.com/generateChat` API は `model_config_id` パラメータで Claude モデル (例: `claude-sonnet-4-5`) を指定可能。Gemini 専用 API ではなく、マルチモデル対応 API である。

**操作**: `model_config_id` を API ペイロードに含めるだけでモデルが切り替わる。`thinkingConfig` 等の Gemini 固有パラメータは Claude 使用時に無視される。

**モデル名解決チェーン**: `displayName` > `id` > `_MODEL_DISPLAY_NAMES[request_model]` > `request_model` > `cortex-chat (cid=...)`

[確信: 95%] (SOURCE: 実 API テスト G8 で Claude Sonnet 4.5 の正常応答確認済)
