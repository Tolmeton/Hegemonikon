## 法則化

### 法則: 「Claude Desktop の mcpServers はローカル stdio のみ」

> Claude Desktop の `claude_desktop_config.json` の `mcpServers` はローカル stdio プロセスのみサポート。
> リモート MCP (streamable-http) は **claude.ai Web UI のコネクタ設定** から接続する。
> mcpServers にリモート URL を書くとクラッシュループが発生する。

- **発動条件**: Claude Desktop に MCP サーバーを追加しようとしたとき
- **検証**: ログに "Starting app" が連続で出現 = クラッシュループ
