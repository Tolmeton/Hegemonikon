## 分離方針

```
モノレポ内の PJ  ─── 成熟 (v1.0+) ──→  独立リポジトリ
                                         ↓
                                    50_外部 に submodule 登録
                                         ↓
                                    本ポータルにリンク追記
```

**分離条件**:
1. 独立した CI/CD が意味を持つ規模
2. HGK 以外のユーザーが利用可能
3. Creator の明示的判断
