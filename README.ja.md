## Hegemonikon とは？

Hegemonikon (ἡγεμονικόν) — ストア派哲学で「魂の統率機能」を意味する語。

AI エージェントのための普遍的な認知ハーネス。知覚・推論・行為のすべてを単一の公理体系で統治する。特定のモデル・IDE・ツールチェーンに依存しない。フレームワークは移り変わる。公理は残る。

**公理は1つ。** 自由エネルギー原理 (FEP) — あらゆる自己組織化系は変分自由エネルギーを最小化する。

**座標は8つ。** マルコフ境界の位相構造 (求心性/遠心性 → 4象限) と6つの修飾座標 (価値・機能・精度・尺度・価値方向・時間)。

**認知操作は48。** 4象限 × 6座標 × 2極 = エージェントが実行しうる認知動詞の完全集合。圏論における関手として形式化されている。


## 理論的基盤

Hegemonikon は二本の柱の上に立つ。

- **自由エネルギー原理 (FEP)** — Karl Friston の変分フレームワーク。この体系のあらゆる構造は VFE 最小化の具体的インスタンスである: 知覚は精度加重、行為は能動推論、学習はモデル更新。

- **圏論** — 構造の文法。認知操作は関手。族内関係は随伴 (F⊣G)。品質 (Kalon) は不動点 Fix(G∘F)。状態間の距離は [0,1]-豊穣圏の Hom 値に住む。

これは比喩ではない。公理階層は形式的に導出され、48の認知操作は与えられた座標の下での完全な列挙である。
